## Summary

BSE has scheduled a mock trading session for the Equity Derivatives segment on Saturday, November 22, 2025. This session will allow trading members and vendors to test their trading applications, including the newly introduced pre-open session functionality. The mock trading will simulate various market conditions including risk-reduction mode and trading halts. All trades executed during this session will not create any actual rights, liabilities, or settlement obligations.

## Key Points

- Mock trading session scheduled for Saturday, November 22, 2025 for Equity Derivatives segment
- No new BOLT TWS Release will be deployed
- Pre-open session testing will be included as per circular 20251110-40 dated November 10, 2025
- Special test contract master file will be provided for pre-open session testing
- Trading members using third-party platforms or in-house ETI API systems can test various functionalities
- All trades during mock session will not attract any margin, pay-in, pay-out obligations or create legal rights/liabilities
- Feedback must be submitted by 5:00 PM on the same day

## Regulatory Changes

No regulatory changes. This is a testing session to facilitate the implementation of the pre-open session in Equity Derivatives segment as announced in earlier circular 20251110-40.

## Compliance Requirements

- Trading members should download the test contract master file (Mock_22-11-2025.zip) from https://www.bseindia.com/members/index.aspx on Friday, November 21, 2025 at EOD
- Members must use the test contract master file during the mock session
- After the mock session, members must replace the test file with the appropriate contract master file for live trading on Monday, November 24, 2025
- Members are requested to actively participate in the mock trading session
- Feedback should be submitted by 5:00 PM on November 22, 2025 to BSE Tech Support or Help Desk

## Important Dates

- **November 21, 2025 (EOD)**: Test contract master file (Mock_22-11-2025.zip) available on BSE website
- **November 22, 2025**: Mock trading session
  - 11:15 AM - 12:00 PM: Log-in
  - 12:00 PM - 01:27/28 PM: Pre-Open Session Order Entry period (random stoppage in last 1 minute)
  - 01:28 PM - 01:32 PM: Order Matching period
  - 01:35 PM - 04:00 PM: Continuous Trading
  - Until 04:15 PM: Trade Modification
  - By 5:00 PM: Feedback submission deadline
- **November 24, 2025**: Resume live trading with appropriate contract master file

## Impact Assessment

**Market Impact**: None - this is a testing session with no actual settlement or obligations.

**Operational Impact**: Low to Medium for trading members and vendors who need to:
- Allocate resources for mock trading participation on Saturday
- Download and configure test contract master files
- Test their systems for pre-open session functionality
- Ensure file replacement before Monday live trading
- Provide feedback to BSE

**Testing Scope**: The mock session enables testing of critical functionalities including exceptional market conditions (risk-reduction mode, trading halts), pre-open session mechanics, and general system readiness. This is particularly important for members preparing for the new pre-open session feature in Equity Derivatives.
