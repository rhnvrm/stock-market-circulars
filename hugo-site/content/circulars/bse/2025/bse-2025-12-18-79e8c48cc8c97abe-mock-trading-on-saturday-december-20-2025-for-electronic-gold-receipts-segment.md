## Summary

BSE has announced a mock trading session scheduled for Saturday, December 20, 2025, for the Electronic Gold Receipts (EGR) segment. The circular provides technical connection parameters for accessing the Disaster Recovery (DR) site of the BOLT Plus trading system during the mock session. Trading members can connect using BOLT TWS, third-party trading applications, or in-house developed systems through ETI APIs.

## Key Points

- Mock trading session scheduled for December 20, 2025 (Saturday)
- Applies to Electronic Gold Receipts (EGR) segment
- DR site connection parameters provided for BOLT Plus trading system
- Multiple connectivity options available: BOLT TWS, third-party applications via ETI APIs, and web-based systems
- Primary DR IP: 10.255.241.26, Port: 14910 (for ETI connections)
- Secondary DR IP: 10.255.241.27, Port: 14910 (for ETI connections)
- Web-based systems accessible through provided URLs for RTRMS, CLASS Collateral, Extranet, and other platforms

## Regulatory Changes

No regulatory changes announced. This is a technical circular for mock trading session connectivity.

## Compliance Requirements

- Trading members participating in the mock session must update their technical connection parameters
- BOLT TWS users need to change configuration settings at login using scrip profile icon or Shift+F12 shortcut
- Members using third-party trading applications must update DR connection parameters in their respective systems
- Members using Online Trade File application must update remote server IP and port settings in setting.ini file

## Important Dates

- **December 20, 2025 (Saturday)**: Mock trading session for Electronic Gold Receipts segment

## Impact Assessment

This is a routine mock trading exercise with minimal impact on market participants. The session is designed for system testing and disaster recovery preparedness. Only members participating in the EGR segment need to take action by updating their connection parameters. There is no impact on live trading operations or regular market activities. The mock session helps ensure business continuity and system reliability for the Electronic Gold Receipts segment.
