## Summary

BSE has released the monthly penalty report (file PM1025.Clg) for client code modifications of non-institutional trades conducted during October 2025. The report has been made available in the Equity Transaction Folder on the BSE Extranet. This release is in accordance with Exchange Notice Nos. 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011).

## Key Points

- Penalty report covers client code modifications for non-institutional trades across all segments during October 2025
- Report file name: PM1025.Clg
- Available for download in the Equity Transaction Folder on BSE Extranet
- Members are advised to save the file for future reference
- Contact details provided for queries: 022-2272 8435 / 022-2272 5785 or bse.reports@bseindia.com

## Regulatory Changes

No new regulatory changes. This circular implements existing penalty framework established under Exchange Notice Nos. 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011).

## Compliance Requirements

- Trading members must download and review the penalty report from the Extranet
- Members should save the report for their records and future reference
- Any penalties levied must be addressed according to existing exchange regulations
- Members with queries should contact BSE Trading Operations at the provided contact details

## Important Dates

- **Notice Date**: November 11, 2025
- **Reporting Period**: October 2025
- **File Download**: Available immediately in today's Equity Transaction Folder

## Impact Assessment

**Operational Impact**: Minimal. This is a routine monthly compliance report affecting only those trading members who modified client codes for non-institutional trades during October 2025.

**Market Impact**: None. This is an administrative compliance matter with no direct market implications.

**Member Impact**: Low. Only affects members who made client code modifications during the specified period and may face penalties as per existing regulations. Most members will not be affected unless they conducted such modifications.
