## Summary

BSE has made available the penalty report for client code modifications of non-institutional trades executed during November 2025. The report file named PM1125.Clg has been uploaded to the Equity Transaction Folder in the BSE Extranet. This penalty framework is based on Exchange Notice Nos. 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011).

## Key Points

- Penalty report covers client code modifications for non-institutional trades during November 2025
- Report filename: PM1125.Clg
- Available in Equity Transaction Folder on BSE Extranet
- Members must save the file for future reference
- Applicable to all segments (Equity, Derivatives, Currency, etc.)

## Regulatory Changes

No new regulatory changes. This circular implements existing penalty framework established in 2011 notices.

## Compliance Requirements

- Trading members must download and save the penalty report PM1125.Clg from the Equity Transaction Folder
- Members should review penalties assessed for their client code modifications during November 2025
- Contact BSE Trading Operations for queries via:
  - Phone: 022-2272 8435 or 022-2272 5785
  - Email: bse.reports@bseindia.com

## Important Dates

- **Report Period**: November 2025
- **Publication Date**: December 11, 2025

## Impact Assessment

**Market Impact**: Minimal. This is a routine administrative process.

**Operational Impact**: Low. Affects only trading members who modified client codes for non-institutional trades during November 2025. Members need to download, review, and retain the penalty report for compliance and reconciliation purposes.

**Trading Members**: Should review penalties assessed and ensure proper record-keeping for audit and compliance verification.
