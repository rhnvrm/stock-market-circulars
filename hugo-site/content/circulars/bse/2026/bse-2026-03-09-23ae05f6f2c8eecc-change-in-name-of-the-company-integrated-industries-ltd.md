## Summary

BSE has notified trading members that Integrated Industries Ltd (Scrip Code 531889) has changed its corporate name to Nurture Well Industries Limited. Effective March 13, 2026, the company's SCRIP ID and abbreviated name on the BOLT Plus System will be updated to NWIL.

## Key Points

- Scrip Code 531889 remains unchanged
- Old name: Integrated Industries Ltd
- New name: Nurture Well Industries Limited
- New SCRIP ID on BOLT Plus System: NWIL
- New abbreviated name on BOLT Plus System: NWIL
- Change effective date: March 13, 2026

## Regulatory Changes

No regulatory changes. This is a corporate name change communicated by the company to the Exchange. BSE is updating the SCRIP ID and abbreviated name on its BOLT Plus trading system to reflect the new name.

## Compliance Requirements

Trading members should update their records and systems to reflect the new company name (Nurture Well Industries Limited) and new SCRIP ID/abbreviated name (NWIL) on the BOLT Plus System effective March 13, 2026.

## Important Dates

- **March 09, 2026**: Notice issued by BSE
- **March 13, 2026**: Effective date for name change on BOLT Plus System

## Impact Assessment

Minimal market impact. The name change is purely administrative. The scrip code (531889) remains unchanged, so existing orders, holdings, and positions are unaffected. Trading members need to note the updated SCRIP ID (NWIL) for BOLT Plus System transactions from March 13, 2026 onwards.
