## Summary

BSE has announced a mock trading session for the Electronic Gold Receipts (EGR) segment scheduled for Saturday, January 17, 2026. The circular provides technical connection parameters for trading members to connect to the Disaster Recovery (DR) site of the BOLT Plus trading system using either BOLT TWS or third-party trading applications through ETI APIs. The document includes detailed connectivity instructions for both Commodity Derivatives segment trading and web-based Exchange systems.

## Key Points

- Mock trading session scheduled for Electronic Gold Receipts segment on Saturday, January 17, 2026
- Trading members can connect via BOLT TWS or third-party trading applications through ETI APIs
- DR site connection parameters provided for BOLT Plus trading system
- Primary DR connectivity: IP 10.255.241.26, Port 14910
- Secondary DR connectivity: IP 10.255.241.27, Port 14910
- Configuration settings accessible via scrip profile icon or Shift+F12 shortcut keys on BOLT TWS
- Web-based systems (RTRMS, CLASS collateral, Extranet, EBOSS, LEIPS) accessible through provided URLs
- BOLTPLUS Connectivity Manual Version 1.16 included as reference documentation

## Regulatory Changes

No regulatory changes. This is a routine technical mock trading session for system testing and validation purposes.

## Compliance Requirements

- Trading members must update technical connection parameters in their configuration settings to connect to the DR site
- BOLT TWS users need to modify settings in the configuration settings window at login
- Third-party application users must change connection parameters to specified Primary and Secondary DR IP addresses and ports
- Members using online trade file applications must update remote server IP and port settings in the setting.ini file
- Existing user IDs and passwords remain valid for web-based Exchange systems

## Important Dates

- **January 17, 2026 (Saturday)**: Mock trading session for Electronic Gold Receipts segment

## Impact Assessment

**Market Impact**: None. This is a mock trading session conducted outside regular market hours for testing purposes only.

**Operational Impact**: Minimal. Trading members who participate in the EGR segment may need to temporarily adjust their technical connectivity parameters to connect to the DR site for the mock session. This is a voluntary testing exercise to ensure business continuity preparedness and system reliability.

**Technical Impact**: Trading members will need to configure their systems with DR site connection parameters. The session provides an opportunity to test disaster recovery infrastructure and validate connectivity through backup systems without disrupting live trading operations.
