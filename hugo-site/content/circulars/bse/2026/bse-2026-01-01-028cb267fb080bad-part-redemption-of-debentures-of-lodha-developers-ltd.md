## Summary

BSE has announced the actual reduced face values for two series of debentures issued by Lodha Developers Ltd following part redemption. This notice updates the earlier communication dated December 11, 2025, and provides final reduced face values that will be effective from January 2, 2026.

## Key Points

- Two debenture series of Lodha Developers Ltd have undergone part redemption
- Scrip Code 976923 (LDL-Reset Rate-21-01-28-PVT, ISIN: INE670K07307): Reduced face value is Rs. 89,512 per debenture
- Scrip Code 977293 (LDL-TBILLS-13-11-28-PVT, ISIN: INE670K07349): Reduced face value is Rs. 98,454 per debenture
- This updates the earlier notice dated December 11, 2025 (Notice No. 20251211-76)
- Trading with new face values commences from January 2, 2026
- DR reference: DR-790/2025-2026

## Regulatory Changes

No regulatory framework changes. This is a corporate action implementation update.

## Compliance Requirements

- Trading members must note the updated reduced face values for both debenture series
- All trades from January 2, 2026 onwards must reflect the new face values
- Trading members should update their systems to reflect the revised face values

## Important Dates

- **Notice Date**: January 1, 2026
- **Effective Date for Trading**: January 2, 2026
- **Previous Notice**: December 11, 2025 (Notice No. 20251211-76)

## Impact Assessment

**Market Impact**: Medium - affects debenture holders and traders of these specific securities. The reduced face values will impact trading calculations, yields, and settlement amounts.

**Operational Impact**: Trading members need to update their systems with the new face values before commencement of trading on January 2, 2026. This affects pricing, settlement, and position calculations for these debentures.
