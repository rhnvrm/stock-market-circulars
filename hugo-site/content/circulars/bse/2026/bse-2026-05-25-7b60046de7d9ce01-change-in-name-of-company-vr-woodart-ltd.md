## Summary

BSE has notified trading members that VR Woodart Ltd (Scrip Code: 523888) has informed the Exchange of a change in its company name to MEGAMONT LIMITED. The SCRIP ID and abbreviated name on the BOLT Plus System will be updated accordingly, effective June 01, 2026.

## Key Points

- Company name changes from **VR Woodart Ltd** to **MEGAMONT LIMITED**
- Scrip Code remains unchanged: **523888**
- New SCRIP ID on BOLT Plus System: **MEGAMONT**
- New abbreviated name on BOLT Plus System: **MEGAMONT**
- Notice issued by Listing Compliance and Operations department

## Regulatory Changes

The Exchange will update the SCRIP ID and abbreviated name for Scrip Code 523888 on the BOLT Plus System to reflect the new corporate name MEGAMONT LIMITED.

## Compliance Requirements

Trading members should update their systems and records to reflect the new company name (MEGAMONT LIMITED) and the revised SCRIP ID (MEGAMONT) effective June 01, 2026. No additional action is required from investors, as the scrip code and ISIN remain unchanged.

## Important Dates

- **Notice Date:** May 25, 2026
- **Effective Date of Name Change on BOLT Plus System:** June 01, 2026

## Impact Assessment

This is a routine administrative update with minimal market impact. The scrip code (523888) remains the same, ensuring continuity for existing shareholders and traders. Trading members using the BOLT Plus System should note the updated SCRIP ID and abbreviated name (MEGAMONT) for order placement and reference purposes from June 01, 2026 onwards.
