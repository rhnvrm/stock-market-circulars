## Summary

BSE has informed trading members that Checkpoint Trends Ltd (Scrip Code 531099) has changed its corporate name to Centuple Global Limited. The Scrip ID and Abbreviated Name on the BOLT Plus System will be updated accordingly effective May 20, 2026.

## Key Points

- Company scrip code 531099 remains unchanged
- Existing name: Checkpoint Trends Ltd
- New name: Centuple Global Limited
- New Scrip ID on BOLT Plus System: CENTUPLE
- New Abbreviated Name on BOLT Plus System: CENTUPLE
- Change is effective May 20, 2026
- Certificate of Incorporation attached as supporting document

## Regulatory Changes

No regulatory changes. This is an administrative update reflecting the company's legally approved name change. The BOLT Plus System identifiers (Scrip ID and Abbreviated Name) are being updated to reflect the new corporate name.

## Compliance Requirements

Trading members should update their records to reflect the new company name (Centuple Global Limited) and the new BOLT Plus System identifiers (Scrip ID: CENTUPLE, Abbreviated Name: CENTUPLE) effective May 20, 2026.

## Important Dates

- Circular Date: May 14, 2026
- Effective Date of Name Change on BOLT Plus System: May 20, 2026

## Impact Assessment

Minimal market impact. This is a routine name change affecting only the display identifiers on the BSE BOLT Plus trading system. The scrip code (531099) remains unchanged, ensuring continuity for existing orders and positions. Trading members need to note the new Scrip ID and Abbreviated Name for future transactions.
