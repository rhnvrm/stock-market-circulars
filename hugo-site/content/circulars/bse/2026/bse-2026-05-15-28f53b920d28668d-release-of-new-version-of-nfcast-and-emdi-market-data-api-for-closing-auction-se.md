## Summary

BSE has released updated versions of the NFCAST market data API (ver. 6.1) and EMDI (Enhanced Market Data Interface, ver. 1.4.5) as part of the rollout of Closing Auction Session (CAS) related market data changes. This notice is a continuation of the earlier circular 20260330-39 dated March 30, 2026 regarding ETI API and NFCAST enhancements for CAS.

## Key Points

- NFCAST market data API ver. 6.1 is now available, incorporating additional enhancements for Closing Auction Session changes.
- EMDI (Enhanced Market Data Interface) ver. 1.4.5 is now available with suitable updates.
- Updated documents are available on the BSE website at http://www.bseindia.com/nta.aspx.
- This is a continuation of circular 20260330-39 dated March 30, 2026.
- BSE will separately notify members when CAS-related changes are available in the test (simulation) environment.

## Regulatory Changes

No new regulatory mandates are introduced. This circular implements technical enhancements to market data APIs to support the previously announced Closing Auction Session changes.

## Compliance Requirements

- Trading members must initiate necessary changes in their respective applications to support the new API versions.
- Third-party front-end trading application vendors must update their applications accordingly.
- All affected parties must test their changes in the BSE test/simulation environment before the live rollout.
- For technical queries, contact: bse.tech@bseindia.com or call 022-2272 8053.

## Important Dates

- Notice Date: May 15, 2026
- Live rollout date: To be notified separately by BSE once CAS changes are available in the test environment.

## Impact Assessment

This update primarily impacts trading members and third-party front-end trading application vendors that interface with BSE's market data feeds via NFCAST or EMDI APIs. They are required to review, implement, and test the updated API versions in a simulated environment ahead of the go-live date. Failure to update in time may result in disruptions to real-time market data consumption during the Closing Auction Session. General investors and non-technical market participants are not directly impacted.
