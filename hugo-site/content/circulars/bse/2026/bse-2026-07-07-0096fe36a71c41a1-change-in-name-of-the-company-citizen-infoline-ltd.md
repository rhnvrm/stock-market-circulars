## Summary

BSE has notified trading members that Citizen Infoline Ltd (Scrip Code: 538786) has changed its name to Citizen Solar Limited. The change takes effect on the BOLT Plus System from July 13, 2026. The scrip ID and abbreviated name on the BOLT Plus System will continue to be "CITIZEN" for both the existing and new name.

## Key Points

- Company: Citizen Infoline Ltd (Scrip Code: 538786)
- New Name: Citizen Solar Limited
- Scrip ID on BOLT Plus System remains: CITIZEN
- Abbreviated name on BOLT Plus System remains: CITIZEN
- Change effective from July 13, 2026
- Notice issued by Marian Dsouza, Assistant Vice President - Listing Compliance & Operations

## Regulatory Changes

No change in regulatory framework; this is a corporate name change update reflected in BSE's trading systems. The company's scrip code (538786) remains unchanged.

## Compliance Requirements

Trading members should update their internal records to reflect the company's new legal name, Citizen Solar Limited, while continuing to reference the same scrip code and BOLT Plus System identifiers (CITIZEN).

## Important Dates

- Notice Date: July 07, 2026
- Effective Date of Name Change: July 13, 2026

## Impact Assessment

This is a low-impact administrative update. Trading continues uninterrupted under the same scrip code and abbreviated name (CITIZEN); only the company's legal name changes from Citizen Infoline Ltd to Citizen Solar Limited. No action required from investors regarding existing holdings.
