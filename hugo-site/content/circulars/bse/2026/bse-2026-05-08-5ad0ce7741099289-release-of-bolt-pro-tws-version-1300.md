## Summary

BSE has announced the release of BOLT Pro TWS Version 13.00, which is a **compulsory upgrade** for all trading members effective Monday, May 11, 2026. The upgrade applies to Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The new version introduces an enhancement in the display of the Indicative Close Value of Index.

## Key Points

- BOLT Pro TWS ver. 13.00 goes live on **Monday, May 11, 2026**
- This is a **compulsory** version release — all trading members must upgrade
- Applicable segments: Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR
- New feature: Enhancement in Display of Indicative Close Value of Index
- Version available for download from **May 8, 2026 at 7:00 PM onwards**
- Mock trading session available on **Saturday, May 9, 2026**
- Leased Line users can upgrade via the TWS Version tab in the Auto-Download Interface

## Regulatory Changes

No regulatory rule changes; this is a technical/software upgrade mandated by BSE for all trading members using BOLT Pro TWS across the specified segments.

## Compliance Requirements

- All trading members using BOLT Pro TWS must upgrade to ver. 13.00 before May 11, 2026
- Leased Line users: Select ver. 13.00 in the TWS Version tab of the Auto-Download Interface on startup
- Website download steps:
  1. Browse to http://www.bseindia.com/help/
  2. Click on the "Update.zip" link
  3. Download and unzip 'Software Update.zip'
  4. Follow instructions in 'How to use this files.doc' to copy files to the BOLT Pro TWS installation directory
- For queries, contact BSE Helpdesk: 022-45720400/600 & 022-69158500 or bsehelp@bseindia.com

## Important Dates

| Date | Event |
|------|-------|
| May 8, 2026 (7:00 PM onwards) | BOLT Pro TWS ver. 13.00 available for download |
| May 9, 2026 | Mock trading session to test new version |
| May 11, 2026 | Compulsory go-live date for BOLT Pro TWS ver. 13.00 |

## Impact Assessment

This is a high-impact operational notice for all BSE trading members. Since the upgrade is **compulsory**, failure to upgrade before May 11, 2026 may result in inability to trade on BSE platforms across the five affected segments. The functional enhancement — improved display of Indicative Close Value of Index — is relatively minor but the mandatory nature of the upgrade makes timely action critical for uninterrupted trading operations.
