## Summary

BSE has informed trading members that ten companies have fixed record dates for payment of interest and/or part redemption of their Non-Convertible Debentures (NCDs). Trading in these debentures will occur at reduced face value effective from the respective ex-dates, reflecting the partial redemption amounts.

## Key Points

- Ten companies' NCDs are affected by part redemption/interest payment record dates.
- Manba Finance Limited (Scrip Code 975966, ISIN INE939X07150): Record Date 14/07/2026, reduced face value Rs. 50,000/- per debenture.
- Lendingkart Finance Limited (Scrip Code 975172, ISIN INE090W07618): Record Date 15/07/2026, reduced face value Rs. 8,333.33 per debenture.
- Keertana Finserv Limited (Scrip Code 976653, ISIN INE0NES07170): Record Date 15/07/2026, reduced face value Rs. 28,571.45 per debenture.
- Greaves Finance Limited (Scrip Code 977441, ISIN INE1QWF07022): Record Date 15/07/2026, reduced face value Rs. 37,499.95 per debenture.
- Electronica Finance Limited (Scrip Code 975117, ISIN INE612U07118): Record Date 16/07/2026, reduced face value Rs. 12,500/- per debenture.
- Auxilo Finserve Private Limited (Scrip Code 975875, ISIN INE605Y07171): Record Date 16/07/2026, reduced face value Rs. 33,333.32 per debenture.
- Akara Capital Advisors Private Limited (Scrip Code 976306, ISIN INE08XP07274): Record Date 16/07/2026, reduced face value Rs. 25,000/- per debenture.
- Varthana Finance Private Limited (Scrip Code 976661, ISIN INE125T07261): Record Date 16/07/2026, reduced face value Rs. 19,467/- per debenture.
- Spandana Sphoorty Financial Limited (Scrip Code 977245, ISIN INE572J07760): Record Date 16/07/2026, reduced face value Rs. 85,000/- per debenture.
- Keertana Finserv Limited (Scrip Code 976943, ISIN INE0NES07246): Record Date 19/07/2026 (Ex Date 17/07/2026), reduced face value Rs. 400/- per debenture.

## Regulatory Changes

No new regulatory changes; this is a routine operational notice implementing existing debenture terms regarding interest payment and part redemption schedules.

## Compliance Requirements

Trading members must update their systems to reflect the reduced face values of the listed debentures effective from the respective ex-dates/settlement dates, and ensure trades are settled based on the adjusted face value post record date.

## Important Dates

- 14/07/2026: Record Date and Ex Date for Manba Finance Limited NCD (975966).
- 15/07/2026: Record Date and Ex Date for Lendingkart Finance Limited (975172), Keertana Finserv Limited (976653), and Greaves Finance Limited (977441) NCDs.
- 16/07/2026: Record Date and Ex Date for Electronica Finance Limited (975117), Auxilo Finserve Private Limited (975875), Akara Capital Advisors Private Limited (976306), Varthana Finance Private Limited (976661), and Spandana Sphoorty Financial Limited (977245) NCDs.
- 17/07/2026: Ex Date for Keertana Finserv Limited NCD (976943).
- 19/07/2026: Record Date for Keertana Finserv Limited NCD (976943).

## Impact Assessment

This notice has limited market-wide impact and is relevant primarily to debt market participants and trading members dealing in the specific NCDs listed. The reduced face values reflect scheduled part redemptions and do not indicate any credit event or distress; accurate implementation is required to ensure correct settlement pricing for these debt instruments.
