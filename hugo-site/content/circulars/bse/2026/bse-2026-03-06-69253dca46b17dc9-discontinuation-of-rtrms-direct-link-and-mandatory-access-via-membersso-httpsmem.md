## Summary

BSE is discontinuing the RTRMS direct link and mandating that all members (brokers, banks, and mutual funds) access BSE applications exclusively through the new Member Single Sign-On (SSO) portal powered by Cymmetri, available at https://membersso.bseindia.com/. This circular serves as the official user manual for onboarding and operating within the new portal.

## Key Points

- The new MemberSSO portal (Cymmetri) replaces the old Member SSO with enhanced features including a new interface, self-service password reset, multiple authentication modes, and self-unlock capability.
- Access URL: https://membersso.bseindia.com/
- Three modes of multi-factor authentication are supported: Cymmetri Authenticator app, SMS Authenticator, and Secret Questions.
- Each member has one Member ID but can create multiple sub-users from the main/admin account.
- After 3 incorrect password attempts, the user account is locked automatically.
- Sub-users can request access to specific applications with defined access periods (start/end date or lifetime access).
- Application access requests require approval from the Main Member before being granted.

## Regulatory Changes

- The RTRMS direct link is being discontinued; members must transition to MemberSSO for all BSE application access.
- All member entities (brokers, banks, mutual funds) are required to onboard to the Cymmetri-based SSO portal.

## Compliance Requirements

- All members must register and log in via https://membersso.bseindia.com/ to access BSE applications.
- Members must set up at least one mode of multi-factor authentication: Cymmetri Authenticator (requires downloading "Cymmetri Verify" from Play Store/App Store), SMS OTP, or Secret Questions.
- When creating sub-users, the UserType field must be set to "Sub_User" exactly — incorrect UserType will prevent sub-user creation.
- Sub-users must request application access through the portal and await Main Member approval.

## Important Dates

- Effective from the date of this circular (2026-03-06): RTRMS direct link is discontinued and MemberSSO access is mandatory.

## Impact Assessment

- **Operational Impact (High):** All brokers, banks, and mutual funds must migrate to the new portal to retain access to BSE member applications. Failure to onboard will result in loss of access.
- **Security Impact (Positive):** The new system introduces stronger multi-factor authentication, self-service account management, and granular sub-user access controls, improving overall security posture.
- **Administrative Impact:** Firms must reconfigure user access, create sub-users under the new system, and train staff on the new authentication workflows.
- No direct market trading or listing impact, but operational disruption is possible during the transition period if members delay onboarding.
