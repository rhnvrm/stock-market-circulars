## Summary

BSE announces the release of BOLT Pro TWS version 12.50, which will go live on Monday, January 12, 2026. This is an optional version release applicable to Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The update introduces technical enhancements to the trading platform.

## Key Points

- BOLT Pro TWS version 12.50 is an OPTIONAL upgrade
- Applicable to all major segments: Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives & EGR
- Functionality: Technical Enhancement
- Available for download from January 9, 2026 at 7:00 PM onwards
- Testing available during mock trading session on Saturday, January 10, 2026
- Leased Line users can upgrade via Auto-Download Interface by selecting ver. 12.50
- Alternative download available from BSE website at http://www.bseindia.com/help/

## Regulatory Changes

No regulatory changes. This is a technical platform update.

## Compliance Requirements

No mandatory compliance requirements. The upgrade is optional for trading members. Members choosing to upgrade should:

- Download the new version from either the Auto-Download Interface or BSE website
- Follow the installation instructions provided in 'How to use this files.doc'
- Test the system during the mock trading session on January 10, 2026
- Contact BSE Helpdesk for any technical support (022-45720400/600, 022-69158500, or bsehelp@bseindia.com)

## Important Dates

- **January 8, 2026**: Circular announcement date
- **January 9, 2026 (7:00 PM onwards)**: Download becomes available
- **January 10, 2026**: Mock trading session for testing the new version
- **January 12, 2026**: Official release and go-live date

## Impact Assessment

Minimal market impact as this is an optional technical upgrade. Trading members can continue using existing versions without disruption. The technical enhancements are intended to improve platform performance and stability. No impact on trading operations, market structure, or compliance obligations. Members who choose to upgrade should allocate time for download, installation, and testing during the mock session.
