## Summary

BSE is rolling out an enhanced version of the Online Trade Download (OTD) application for the Equity Derivatives segment to all trading members. This update follows the pilot launch announced in circular 20260109-17 dated January 09, 2026. The new version includes technical enhancements for performance optimization and introduces Internet connectivity as an alternative to leased line connections. The enhanced OTD application will be available for live trading from Monday, February 09, 2026.

## Key Points

- New OTD version with technical enhancements for performance optimization being rolled out to all trading members
- Internet connectivity option now available in addition to existing leased line connectivity
- Members must contact BSE helpdesk to download and enable the new version
- Pilot users currently using OTD via leased line must download new version to switch to Internet connectivity
- Continuation of circular no. 20260109-17 dated January 09, 2026

## Regulatory Changes

No regulatory changes introduced. This is a technical system enhancement to existing OTD facility.

## Compliance Requirements

- Trading members must contact BSE helpdesk team to download the new OTD application version
- Members need to obtain connectivity details from BSE helpdesk
- Members currently using pilot version via leased line who wish to switch to Internet connectivity must download the new version
- Contact helpdesk at 022-45720400/600 & 022-69158500 or email bsehelp@bseindia.com for queries and support

## Important Dates

- **February 06, 2026**: Circular issue date
- **February 09, 2026**: New OTD version available for live trading in Equity Derivatives segment

## Impact Assessment

**Operational Impact**: Medium - All trading members in the Equity Derivatives segment are affected and need to download the new version to access enhanced features and Internet connectivity option.

**Market Impact**: Low - This is a technical infrastructure upgrade that improves system performance and connectivity options but does not change trading rules or market structure.

**Technical Impact**: The performance optimization enhancements should improve system efficiency for trading members. The addition of Internet connectivity provides more flexibility in connection methods beyond leased lines.
