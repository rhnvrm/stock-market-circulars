## Summary

BSE has issued an update to its earlier circular (No. 20260624-30 dated June 24, 2026) regarding the release of new versions of the ETI API and NFCAST Market Data API to support Closing Auction Session related changes. The go-live date for the new MDI/EMDI API (Version 1.4.5) in the Equity Segment has been revised/postponed to July 20, 2026. There is no change to the go-live schedule for NFCAST Version 6.2, which remains as previously communicated.

## Key Points

- This notice is a continuation/update to Exchange circular No. 20260624-30 dated June 24, 2026.
- Go-live date for MDI/EMDI API Version 1.4.5 revised from July 13, 2026 to July 20, 2026.
- Mock trading for MDI/EMDI API Version 1.4.5 rescheduled from July 11, 2026 to July 18, 2026.
- NFCAST API upgrade from Version 5.0 to Version 6.2: go-live date remains unchanged at August 3, 2026, with mock trading on August 1, 2026.
- EOBI remains at Version 1.4 with no changes.
- Applies to the Trading Development / I.T. Development departments and members using Equity Segment market data feeds.

## Regulatory Changes

No regulatory or policy change; this is a technical release schedule update for Exchange market data infrastructure (MDI/EMDI and NFCAST APIs) supporting the Closing Auction Session enhancements.

## Compliance Requirements

Trading members and vendors consuming BSE market data (MDI/EMDI, NFCAST, EOBI feeds) should update their systems and testing schedules to align with the revised go-live and mock trading dates. Queries can be directed to bse.tech@bseindia.com or 022-2272 8053.

## Important Dates

- **July 18, 2026**: Mock trading for MDI/EMDI API Version 1.4.5 (revised from July 11, 2026).
- **July 20, 2026**: Go-live for MDI/EMDI API Version 1.4.5 (revised from July 13, 2026).
- **August 1, 2026**: Mock trading for NFCAST API Version 6.2 (unchanged).
- **August 3, 2026**: Go-live for NFCAST API Version 6.2 (unchanged).

## Impact Assessment

This is a low-impact, technical/operational notice primarily relevant to trading members' IT teams and market data vendors who need to adjust integration and testing timelines for the revised MDI/EMDI API rollout. It does not affect trading in specific securities, investor actions, or corporate actions.
