## Summary

BSE has informed trading members that a new version of BOLT Pro TWS (ver. 13.25) will be released and made live with effect from Monday, July 20, 2026. This version is a compulsory release across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments, introducing technical enhancements.

## Key Points

- BOLT Pro TWS ver. 13.25 is a mandatory (compulsory) version upgrade for all trading members.
- Applicable across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments.
- The update includes technical enhancements (no functional/feature list specified beyond this).
- Available for download from Friday, July 17, 2026, 7:00 pm onwards via the Auto-Download Interface for Leased Line Connection users.
- Can also be downloaded manually from the BSE website (http://www.bseindia.com/help/) by downloading and unzipping 'Update.zip' and following installation instructions in 'How to use this files.doc'.
- Members can test the new version during the mock trading session on Saturday, July 18, 2026.
- Queries can be directed to BSE Helpdesk at 022-45720400/600, 022-69158500, or bsehelp@bseindia.com.

## Regulatory Changes

None; this is a routine software version update notice, not a regulatory change.

## Compliance Requirements

All trading members using BOLT Pro TWS must upgrade to version 13.25 before it goes live on July 20, 2026, as it is a compulsory release. Members should test the new version during the mock session on July 18, 2026, to ensure smooth transition.

## Important Dates

- July 17, 2026, 7:00 pm: BOLT Pro TWS ver. 13.25 available for download.
- July 18, 2026: Mock trading session to test new version.
- July 20, 2026 (Monday): New version becomes live and compulsory.

## Impact Assessment

This is an operational/technical notice with low market impact. It affects trading members' back-office/terminal software compliance but does not directly affect stock prices, listings, or investor actions. Failure to upgrade could disrupt a member's ability to trade from July 20, 2026 onwards, making timely compliance operationally important for trading members specifically.
