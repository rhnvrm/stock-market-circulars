## Summary

BSE has informed trading members that the penalty report for client code modifications of non-institutional trades executed during June 2026 has been uploaded to the Exchange's Extranet, under the EQUITY TRANSACTION FOLDER, with the file name PM0626.Clg no. This report is generated as per earlier Exchange notices governing client code modification penalties.

## Key Points

- Penalty report pertains to client code modifications of non-institutional trades for June 2026.
- File uploaded with name PM0626.Clg no. in the EQUITY TRANSACTION FOLDER on Extranet.
- Applicable across All Segments.
- Issued under reference to Exchange Notice Nos. 20110729-24 (dated July 29, 2011) and 20110826-4 (dated August 26, 2011).
- Trading members are advised to save the file for future reference.

## Regulatory Changes

No new regulatory changes; this notice implements the existing framework established by Exchange Notice Nos. 20110729-24 and 20110826-4 governing penalties for client code modifications of non-institutional trades.

## Compliance Requirements

Trading members must download and retain the PM0626.Clg no. file from their Extranet EQUITY TRANSACTION FOLDER for record-keeping and compliance verification purposes. Any queries can be directed to BSE's Trading Operations department via the contact details or email provided.

## Important Dates

- Notice Date: July 10, 2026
- Reporting Period Covered: June 2026 client code modifications

## Impact Assessment

This is a routine, low-impact operational notice affecting only BSE trading members' back-office compliance processes. It does not affect stock prices, trading activity, or investors directly.
