## Summary

BSE announces the release of BOLT Pro TWS version 12.50, scheduled to go live on Monday, January 12, 2026. This is an optional version release applicable to Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The upgrade includes technical enhancements to the trading platform.

## Key Points

- BOLT Pro TWS version 12.50 is an optional upgrade, not mandatory
- Applies to all major trading segments: Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR
- Primary feature: Technical Enhancement
- Available for download from January 9, 2026 at 7:00 PM onwards
- Leased Line Connection users can upgrade via Auto-Download Interface by selecting version 12.50
- Alternative download available from BSE website at http://www.bseindia.com/help/
- Mock trading session for testing scheduled for Saturday, January 10, 2026

## Regulatory Changes

No regulatory changes introduced. This is a technical system upgrade.

## Compliance Requirements

No mandatory compliance requirements. Trading members may choose to upgrade at their discretion. Members opting to upgrade should:

- Download the update from either the Auto-Download Interface or BSE website
- Follow the installation instructions provided in the 'How to use this files.doc' document
- Test the new version during the mock trading session on January 10, 2026
- Contact BSE Helpdesk for any queries at 022-45720400/600, 022-69158500, or bsehelp@bseindia.com

## Important Dates

- **January 8, 2026**: Circular announcement date
- **January 9, 2026, 7:00 PM onwards**: Version 12.50 available for download
- **January 10, 2026**: Mock trading session for testing new version
- **January 12, 2026**: Version 12.50 goes live in production

## Impact Assessment

**Market Impact**: Minimal. This is an optional software upgrade with no impact on trading rules, regulations, or market operations.

**Operational Impact**: Low. Trading members can continue using existing versions. Those choosing to upgrade have a testing window before the live release. The upgrade process is straightforward with support available from BSE Helpdesk.

**Member Action Required**: Optional. Trading members may upgrade at their convenience using either the Auto-Download Interface or manual download from the BSE website.
