## Summary

BSE has announced the release of new versions of Market Data APIs for the Currency Derivatives segment. Trading members and front-end trading application vendors are required to complete testing in the test environment ahead of the live rollout on September 7, 2026.

## Key Points

- NFCAST API will be upgraded from Version 5.0 to Version 6.2
- MDI/EMDI API will be upgraded from Version 1.4.4 to Version 1.4.5
- EOBI (Version 1.4) remains unchanged
- New API documentation is available on the BSE website at http://www.bseindia.com/nta.aspx
- Trading members and vendors must complete testing in the test environment before go-live

## Regulatory Changes

No regulatory rule changes. This is a technical infrastructure upgrade to Market Data APIs in the Currency Derivatives segment.

## Compliance Requirements

- Trading members must update their systems to support the new API versions before September 7, 2026
- Front-end trading application vendors are required to complete necessary testing in the test environment
- Members should refer to updated API documentation available on the BSE website

## Important Dates

| Milestone | Date |
|---|---|
| Test Environment Availability | August 10, 2026 |
| Mock Trading | September 5, 2026 |
| Proposed Go-Live (Live Environment) | September 7, 2026 |

## Impact Assessment

This upgrade impacts trading members and front-end trading application vendors operating in the Currency Derivatives segment. The NFCAST API version jump (5.0 to 6.2) is significant and may require non-trivial integration changes, while the MDI/EMDI API update (1.4.4 to 1.4.5) is a minor version bump. EOBI users are unaffected. Sufficient lead time (approximately 6 weeks for testing, 10 weeks to go-live) has been provided. For queries, members may contact bse.tech@bseindia.com or 022-2272 8053.
