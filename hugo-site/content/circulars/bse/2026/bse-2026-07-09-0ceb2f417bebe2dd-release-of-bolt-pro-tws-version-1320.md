## Summary

BSE has announced the release of a new version of BOLT Pro TWS (ver. 13.20), which will go live from Monday, July 13, 2026. This is a compulsory version release applicable across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The update primarily incorporates technical enhancements related to changes in Trading System APIs in the Equity segment as previously communicated via earlier circulars.

## Key Points

- New BOLT Pro TWS version 13.20 will be compulsory across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments.
- The version introduces technical enhancements handling changes related to Trading System APIs in the Equity segment.
- These API changes were published via circular numbers 20260515-36 (May 15, 2026), 20260529-13 (May 29, 2026), and 20260624-30 (June 24, 2026).
- Leased Line Connection users can upgrade via the TWS Version tab of the Auto-Download Interface.
- The new version can also be downloaded from the BSE website by following provided steps.

## Regulatory Changes

No new regulatory changes are introduced; this circular implements technical/API handling changes previously notified in earlier circulars (20260515-36, 20260529-13, 20260624-30).

## Compliance Requirements

- Trading members must upgrade to BOLT Pro TWS ver. 13.20 as it is a compulsory release.
- Members using Leased Line Connections should select ver. 13.20 in the TWS Version tab of the Auto-Download Interface.
- Members may alternatively download the update from the BSE website following the specified steps (download Update.zip, unzip, and follow instructions in 'How to use this files.doc').

## Important Dates

- **Friday, July 10, 2026, 7:00 PM onwards**: New version available for download.
- **Saturday, July 11, 2026**: Mock trading session to test the new version.
- **Monday, July 13, 2026**: BOLT Pro TWS ver. 13.20 goes live and becomes compulsory.

## Impact Assessment

This is a routine, low-impact operational update affecting trading terminal software used by BSE trading members. It does not affect listed securities, stock prices, or market operations beyond ensuring trading members' systems remain compatible with Exchange API changes. Trading members should ensure timely upgrade to avoid disruption to their trading operations after July 13, 2026.
