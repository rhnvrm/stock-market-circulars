## Summary

BSE has announced the release of new versions of Market Data APIs for the Commodity Derivatives segment. The NFCAST API will be upgraded from Version 5.0 to Version 6.2, and the MDI/EMDI API will be upgraded from Version 1.4.4 to Version 1.4.5. Testing in the test environment is proposed from August 10, 2026, with mock trading on September 5, 2026, and go-live on September 7, 2026.

## Key Points

- NFCAST API upgrade: Version 5.0 → Version 6.2
- MDI/EMDI API upgrade: Version 1.4.4 → Version 1.4.5
- EOBI Version 1.4 remains unchanged
- New API versions are available on the BSE website: http://www.bseindia.com/nta.aspx
- Trading members and front-end trading application vendors must complete testing in the test environment before go-live

## Regulatory Changes

No regulatory rule changes; this is a technical upgrade to Market Data API versions in the Commodity Derivatives segment.

## Compliance Requirements

- Trading members must update their systems to support the new API versions before go-live on September 7, 2026.
- Front-end trading application vendors must complete necessary testing in the test environment.
- Members should reach out to BSE for queries via email at bse.tech@bseindia.com or call 022-2272 8053.

## Important Dates

- **August 10, 2026**: New API versions available in the test environment
- **September 5, 2026**: Mock trading session
- **September 7, 2026**: Proposed go-live date in the live environment

## Impact Assessment

This upgrade affects trading members and front-end trading application vendors operating in the Commodity Derivatives segment. Members must integrate and test the new NFCAST API v6.2 and MDI/EMDI API v1.4.5 versions prior to the September 7, 2026 go-live. Failure to update in time could result in disruptions to market data feeds. The EOBI version remains unchanged, limiting the scope of required changes.
