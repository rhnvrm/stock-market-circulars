## Summary

BSE has notified trading members that Nirbhay Colours India Ltd (Scrip Code: 526349) has officially changed its name to **Craftroot Retail Limited**. The Scrip ID and abbreviated name on the BOLT Plus System will be updated accordingly effective June 05, 2026.

## Key Points

- Existing company name: Nirbhay Colours India Ltd
- New company name: Craftroot Retail Limited
- Scrip Code remains unchanged: **526349**
- New Scrip ID on BOLT Plus System: **CRAFTROOT**
- New Abbreviated Name on BOLT Plus System: **CRAFTROOT**
- Change effective from: **June 05, 2026**

## Regulatory Changes

The company's identifiers on the BSE BOLT Plus trading system are being updated to reflect the new corporate name. The scrip code (526349) remains the same; only the Scrip ID and abbreviated name change.

## Compliance Requirements

Trading members should update their systems and references to use the new Scrip ID **CRAFTROOT** and abbreviated name **CRAFTROOT** for Scrip Code 526349 from June 05, 2026 onwards.

## Important Dates

- Notice Date: June 01, 2026
- Effective Date of Name/Symbol Change: **June 05, 2026**

## Impact Assessment

This is a routine administrative update with minimal market impact. The scrip code is unchanged, ensuring continuity in settlement and position records. Trading members need to update their reference data for the new Scrip ID and abbreviated name before the effective date to avoid operational issues on BOLT Plus System.
