## Summary

BSE has uploaded the penalty report for client code modifications of non-institutional trades for the month of February 2026. The report has been made available in the EQUITY TRANSACTION FOLDER on Extranet under the filename PM0226.Clg. This is a routine monthly compliance activity in accordance with Exchange Notice Nos. 20110729-24 and 20110826-4.

## Key Points

- Penalty report for client code modifications (non-institutional trades, all segments) for February 2026 is now available
- File name: PM0226.Clg
- Location: EQUITY TRANSACTION FOLDER in Extranet
- Members are advised to save the file for future reference
- Issued under the framework of Exchange Notice No. 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011)

## Regulatory Changes

No new regulatory changes. This notice is issued in continuance of the existing penalty framework for client code modifications established via Exchange Notices dated July 29, 2011 and August 26, 2011.

## Compliance Requirements

- Trading members must download and save the penalty report file PM0226.Clg from Extranet
- Members should review the report to check for any penalties applicable to their firm for client code modifications made during February 2026
- File must be retained for future reference and audit purposes

## Important Dates

- Report covers: February 2026 (client code modification activity)
- Notice date: March 10, 2026
- Download available: From March 10, 2026 onwards via Extranet

## Impact Assessment

This is a routine monthly administrative notice with minimal market impact. It affects only trading members who carried out client code modifications for non-institutional trades during February 2026. Members with penalties listed in the report should account for the applicable charges. For queries, members may contact BSE Trading Operations at 022-2272 8435 or 022-2272 5785, or email bse.reports@bseindia.com.
