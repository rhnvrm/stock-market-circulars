## Summary

BSE has informed trading members that LTIMindtree Limited (Scrip Code: 540005) will have its Scrip ID in the BOLT Plus trading system changed from **LTIM** to **LTM**, effective February 27, 2026. This change has been made at the request of the company.

## Key Points

- Scrip Code 540005 (LTIMindtree Limited) retains the same numeric code; only the alphanumeric Scrip ID changes.
- Old Scrip ID: **LTIM**
- New Scrip ID: **LTM**
- Change is effective from **February 27, 2026**.
- All trading members using BOLT Plus must update their systems/references accordingly.

## Regulatory Changes

No regulatory rule changes are involved. This is an administrative update to the BOLT Plus system identifier for LTIMindtree Limited, processed at the company's request.

## Compliance Requirements

- Trading members are required to take note of the new Scrip ID (LTM) effective February 27, 2026.
- Members should update any internal systems, order templates, or automated trading configurations that reference the old Scrip ID (LTIM).
- For clarifications, members may contact:
  - Mr. Tushar Shetty: +91 22 2272 8634 | Listing.crd@bseindia.com
  - Ms. Asmita Chavan: +91 22 2272 8266 | Listing.crd@bseindia.com

## Important Dates

- **February 23, 2026**: Notice issued by BSE.
- **February 27, 2026**: New Scrip ID (LTM) becomes effective in BOLT Plus system.

## Impact Assessment

The impact is minimal and operational in nature. The scrip code (540005) and the listed securities remain unchanged. Trading members and algorithmic systems referencing the old ticker symbol LTIM in the BOLT Plus system must update their configurations before February 27, 2026 to avoid order entry errors. Retail investors are largely unaffected as the company's stock exchange code and market identity remain the same.
