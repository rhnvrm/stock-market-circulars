## Summary

BSE has issued a notice detailing the current versions of trading system APIs for the Electronic Gold Receipts (EGR) segment and the planned upgrades for both ETI and Market Data APIs. Trading members and front-end application vendors must complete testing in the test environment ahead of the scheduled go-live dates.

## Key Points

- Current ETI API version for EGR segment is **Version 1.6.13**
- Current Market Data API versions: NFCAST v5.0, MDI/EMDI v1.4.4, EOBI v1.4
- ETI API v1.6.14 is planned for go-live on **Monday, July 13, 2026** (Mock trading on Saturday, July 11, 2026)
- NFCAST API v6.2 and MDI/EMDI v1.4.5 are planned for go-live on **September 7, 2026** (Mock trading on September 5, 2026)
- EOBI API has no change planned
- This notice continues from Master Circular 20260430-23 dated April 26, 2026 for EGR segment

## Regulatory Changes

No structural regulatory changes. This is a technical upgrade notice for existing APIs used to connect and trade in the EGR segment, in continuation of circulars 20260529-13 and 20260624-30 regarding enhancements in Trading System APIs across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments.

## Compliance Requirements

- Trading members and front-end trading application vendors must complete necessary testing in the test environment before go-live dates
- Members must integrate and validate ETI API v1.6.14 before July 13, 2026
- Members must integrate and validate NFCAST v6.2 and MDI/EMDI v1.4.5 before September 7, 2026
- Contact BSE for queries: **bse.tech@bseindia.com** or **022-2272 8053**

## Impact Assessment

This circular primarily impacts trading members and front-end application vendors who connect to BSE's EGR segment. Firms must update their API integrations in two phases: ETI API by mid-July 2026 and Market Data APIs by early September 2026. Failure to complete testing and upgrade may result in connectivity or trading disruptions in the EGR (Electronic Gold Receipts) segment. The EOBI interface requires no changes. Overall market impact is limited to technical operations for EGR segment participants.
