## Summary

BSE Notice No. 20260422-39 informs all trading members that LKP Finance Ltd (Scrip Code 507912) has notified the Exchange of a change in its company name to **GYFTR Limited**. The SCRIP ID and abbreviated name on the BOLT Plus System will be updated accordingly, effective April 28, 2026.

## Key Points

- Company scrip code **507912** remains unchanged
- Old name: **LKP Finance Ltd**
- New name: **GYFTR Limited**
- New SCRIP ID on BOLT Plus System: **GYFTR**
- New abbreviated name on BOLT Plus System: **GYFTR**
- Notice issued by Listing Compliance & Operations department

## Regulatory Changes

The Exchange will update the SCRIP ID and abbreviated name for Scrip Code 507912 in the BOLT Plus trading system to reflect the new company name GYFTR Limited. A Certificate of Incorporation supporting the name change has been attached to the notice.

## Compliance Requirements

Trading members must update their internal records and systems to recognize the new SCRIP ID **GYFTR** and abbreviated name **GYFTR** for Scrip Code 507912, replacing the previous LKP Finance Ltd references.

## Important Dates

- **Notice Date:** April 22, 2026
- **Effective Date:** April 28, 2026 — SCRIP ID and abbreviated name change takes effect on BOLT Plus System

## Impact Assessment

This is a routine administrative change with medium operational impact for trading members. Orders, positions, and references using the old SCRIP ID or abbreviated name will need to be updated to **GYFTR** from April 28, 2026. No changes to the scrip code (507912) or the company's equity listing segment. Investors holding shares of LKP Finance Ltd will continue to hold the same shares, now under the GYFTR Limited name.
