## Summary

BSE has issued a revised notice confirming the exact reduced face value for Adani Airport Holdings Limited (AAHL) debentures following part redemption. This notice supersedes the earlier communication dated January 19, 2025 (Notice No. 20260119-36). The debentures with scrip code 976065 (ISIN: INE0GCN07047) now have a reduced face value of Rs. 71,355 per debenture, effective from January 7, 2026.

## Key Points

- Revised notice following earlier communication dated January 19, 2025
- Debenture: ADANI AIRPORT HOLDINGS LIMITED 9.35% (Maturity: September 8, 2028)
- Scrip Code: 976065
- ISIN: INE0GCN07047
- Exact Reduced Face Value: Rs. 71,355 per debenture
- Trading with reduced face value commenced from January 7, 2026
- Reference: DR-793/2025-2026

## Regulatory Changes

No new regulatory changes. This is a revised corporate action notification providing exact face value details following part redemption of privately placed debentures.

## Compliance Requirements

- Trading members must note the revised reduced face value of Rs. 71,355 per debenture
- All trades in these debentures must reflect the reduced face value
- Trading members should update their systems to reflect the correct face value for scrip code 976065

## Important Dates

- **January 19, 2025**: Original notice issued (Notice No. 20260119-36)
- **January 7, 2026**: Effective date for trading with reduced face value
- **February 5, 2026**: Revised notice issued with exact reduced face value

## Impact Assessment

**Market Impact**: Medium - affects holders and traders of AAHL debentures in the debt market segment. The clarification of exact reduced face value ensures proper pricing and valuation.

**Operational Impact**: Trading members must ensure their systems reflect the correct reduced face value of Rs. 71,355 per debenture for accurate trade execution and settlement. This revision provides certainty for valuation and accounting purposes for debt market participants holding these securities.
