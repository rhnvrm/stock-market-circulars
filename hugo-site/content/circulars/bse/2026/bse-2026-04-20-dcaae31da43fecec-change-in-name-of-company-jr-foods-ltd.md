## Summary

BSE has notified trading members that Datacipher Limited (Scrip Code 530915) has officially changed its company name to JR Foods Ltd. The Scrip ID and Abbreviated Name on the BOLT Plus System will be updated accordingly, effective April 24, 2026.

## Key Points

- Company scrip code 530915 retains the same code but changes name from Datacipher Limited to JR Foods Ltd
- Scrip ID on BOLT Plus System remains: DATA
- Abbreviated name on BOLT Plus System remains: DATA
- Change is effective from April 24, 2026
- Segment: Debt
- Certificate of Incorporation for JR Foods is attached as supporting document

## Regulatory Changes

No regulatory changes. This is an administrative update reflecting the company's legal name change as registered with the relevant authority. The Exchange is updating its systems to reflect the new name per the company's intimation.

## Compliance Requirements

- Trading members should update their records to reflect the new company name: JR Foods Ltd
- References to Datacipher Limited under scrip code 530915 should be updated to JR Foods Ltd
- No action required by investors beyond noting the name change

## Important Dates

- **Notice Date:** April 20, 2026
- **Effective Date:** April 24, 2026 — Scrip ID and Abbreviated Name on BOLT Plus System will reflect the new name

## Impact Assessment

Minimal market impact. This is a routine name change notification with no effect on the company's financials, trading terms, or investor rights. Trading members and market participants need only update internal records to associate scrip code 530915 with the new name JR Foods Ltd instead of Datacipher Limited.
