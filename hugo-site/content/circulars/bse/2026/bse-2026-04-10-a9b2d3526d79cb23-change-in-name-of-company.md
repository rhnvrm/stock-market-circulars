## Summary

BSE has notified trading members that Banganga Paper Industries Ltd (Scrip Code: 512025) has officially changed its corporate name to **Asgard Alcobev Limited**. The SCRIP ID and abbreviated name on the BOLT Plus System will be updated from the old identifiers to `ASGARD` effective April 17, 2026.

## Key Points

- Company scrip code **512025** is undergoing a name change from Banganga Paper Industries Ltd to Asgard Alcobev Limited.
- The SCRIP ID on BOLT Plus System changes to **ASGARD**.
- The abbreviated name on BOLT Plus System changes to **ASGARD**.
- Change is effective **April 17, 2026**.
- Issued by Marian D'Souza, Assistant Vice President – Listing Compliance & Operations.

## Regulatory Changes

The Exchange has formally updated the SCRIP ID and abbreviated name for scrip code 512025 on the BOLT Plus System to reflect the new corporate name. No change in scrip code or ISIN is indicated.

## Compliance Requirements

- Trading members must update their internal records, order management systems, and client-facing tools to reflect the new name **Asgard Alcobev Limited** and the new SCRIP ID/abbreviated name **ASGARD** on BOLT Plus System.
- Any references to "Banganga Paper Industries Ltd" or the old SCRIP ID should be replaced before the effective date to avoid order entry errors.

## Important Dates

- **Notice Date:** April 10, 2026
- **Effective Date of Name Change on BOLT Plus System:** April 17, 2026

## Impact Assessment

This is a routine administrative update resulting from a corporate name change. The impact is limited to trading members and market participants who reference the scrip by name or abbreviated ID. Investors holding shares in scrip code 512025 are not affected in terms of holdings, but should note the new trading identity. Systems and watchlists using the old name or SCRIP ID will need to be updated before April 17, 2026 to ensure seamless trading.
