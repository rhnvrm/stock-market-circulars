## Summary

BSE has issued mandatory system configuration requirements for all trading members using the BOLT PRO TWS Application. The exchange requires users to review and update their systems to meet recommended specifications to ensure optimal performance, stability, and enhanced user experience. The circular emphasizes strict adherence to Windows 11 OS, warning that other Windows versions may cause runtime errors.

## Key Points

- System must run on Windows 11 OS exclusively to avoid runtime errors
- Minimum processor requirement: Intel i5 or equivalent
- RAM: 8GB minimum (16GB recommended)
- Network: 1 Ethernet port required
- Storage: 100GB free disk space minimum
- BOLT PRO TWS must not be used on other Windows OS versions
- Trading members can contact BSE Helpdesk for queries and clarification

## Regulatory Changes

No regulatory changes. This is a technical infrastructure update mandating specific system configurations for trading platform users.

## Compliance Requirements

**Applicable to:** All trading members using BOLT PRO TWS Application

**Actions Required:**
- Review current system configurations
- Update systems to meet minimum specifications:
  - **Processor:** Intel i5 or equivalent
  - **RAM:** 8GB minimum (16GB recommended)
  - **Network:** 1 Ethernet port
  - **Storage:** 100GB free disk space
  - **OS:** Windows 11 (mandatory)
- Ensure BOLT PRO TWS is not run on unsupported Windows OS versions
- Contact BSE Helpdesk for technical support if needed

**Support Contact:**
- BSE Helpdesk: 022-45720400/600 & 022-69158500
- Email: bsehelp@bseindia.com

## Important Dates

- **Circular Date:** January 23, 2026
- **Effective Date:** Immediate implementation recommended
- No specific compliance deadline mentioned, but updates should be done urgently to avoid runtime errors

## Impact Assessment

**Operational Impact:** Medium - Trading members must ensure their infrastructure meets specifications to maintain uninterrupted trading operations. Systems not meeting requirements may experience runtime errors, affecting trading capabilities.

**Financial Impact:** Low to Medium - Members may need to invest in hardware upgrades (RAM, processor) or Windows 11 licenses if current systems don't meet specifications.

**Technical Impact:** High - Strict OS requirement (Windows 11 only) may require significant system upgrades for members running older Windows versions.

**Risk:** Systems not meeting specifications may experience instability, performance issues, or runtime errors, potentially disrupting trading activities and causing operational losses.
