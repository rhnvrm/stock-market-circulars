## Summary

BSE Limited has informed trading members that further new securities issued under Employee Stock Option Plans/Schemes (ESOP/ESOS) by eight listed companies have been listed and admitted for trading with effect from Tuesday, July 21, 2026.

## Key Points

- Eight companies have new ESOP/ESOS-issued shares listed on BSE
- Delhivery Limited (ISIN INE148O01028, Scrip Code 543529): 206,502 shares, face value ₹1
- Fedbank Financial Services Limited (ISIN INE007N01010, Scrip Code 544027): 54,650 shares, face value ₹10
- Godrej Consumer Products Limited (ISIN INE102D01028, Scrip Code 532424): 60,372 shares, face value ₹1
- IIFL Finance Limited (ISIN INE530B01024, Scrip Code 532636): 56,055 shares, face value ₹2
- Lodha Developers Limited (ISIN INE670K01029, Scrip Code 543287): 22,798 shares, face value ₹10
- Mastek Limited (ISIN INE759A01021, Scrip Code 523704): 1,625 shares, face value ₹5
- Shree Digvijay Cement Co. Limited (ISIN INE232A01011, Scrip Code 502180): 45,000 shares, face value ₹10
- Wipro Limited (ISIN INE075A01022, Scrip Code 507685): 254,943 shares, face value ₹2
- None of the newly listed shares are under lock-in

## Regulatory Changes

No regulatory changes; this is a routine listing notification for shares allotted under existing employee stock option schemes.

## Compliance Requirements

No specific action required from trading members or investors; the notice is informational, confirming admission of the new shares for trading.

## Important Dates

- Notice Date: July 20, 2026
- Effective Date of Listing/Trading: Tuesday, July 21, 2026

## Impact Assessment

The impact is minimal, as the listed quantities represent a small incremental addition to each company's existing equity base arising from routine employee stock option exercises. This does not signal any change in corporate strategy, ownership structure, or financial performance for the affected companies.
