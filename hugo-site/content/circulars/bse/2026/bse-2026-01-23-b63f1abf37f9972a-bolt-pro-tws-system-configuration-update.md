## Summary

BSE has issued updated system configuration specifications for the BOLT PRO TWS Application. All trading members and users are required to review and update their systems to meet the recommended specifications to ensure optimal performance, stability and enhanced user experience. The circular emphasizes that BOLT PRO TWS must be used strictly on the recommended Windows 11 operating system, as other Windows OS versions may cause runtime errors.

## Key Points

- Minimum system requirements updated for BOLT PRO TWS Application
- Intel I5 or equivalent processor required
- 8 GB RAM minimum, 16 GB recommended
- 100 GB free disk space required
- Windows 11 is the mandatory operating system
- Using non-recommended Windows OS versions may lead to runtime errors
- Helpdesk support available at 022-45720400/600, 022-69158500 and bsehelp@bseindia.com

## Regulatory Changes

No regulatory changes. This is a technical infrastructure update.

## Compliance Requirements

Trading members must:
- Review current system configurations against recommended specifications
- Upgrade systems to meet minimum requirements (Intel I5, 8GB RAM, 100GB disk space)
- Ensure Windows 11 operating system is installed
- Upgrade to 16GB RAM for optimal performance (recommended)
- Contact BSE Helpdesk for queries or clarifications

## Important Dates

- Notice Date: January 23, 2026
- Implementation: Immediate review and update requested

## Impact Assessment

This update affects all trading members using BOLT PRO TWS Application. Members operating on non-compliant systems risk experiencing runtime errors and degraded performance. The mandatory Windows 11 requirement may necessitate hardware upgrades for older systems. Trading operations could be disrupted if systems do not meet specifications. The RAM upgrade from 8GB minimum to 16GB recommended suggests increased application resource requirements.
