## Summary

BSE is releasing BOLT Pro TWS version 12.50 on Monday, January 12, 2026. This is an optional version upgrade for trading members across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The new version introduces technical enhancements and will be available for download from January 9, 2026 at 7:00 PM, with testing available during the mock trading session on January 10, 2026.

## Key Points

- BOLT Pro TWS version 12.50 is an optional upgrade, not mandatory
- Applicable to Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives & EGR segments
- Primary enhancement: Technical improvements
- Leased Line Connection users can upgrade via Auto-Download Interface by selecting version 12.50
- Alternative download method available through BSE website at http://www.bseindia.com/help/
- Manual installation requires downloading Update.zip, extracting files, and following instructions in 'How to use this files.doc'

## Regulatory Changes

No regulatory changes introduced. This is a technical software update only.

## Compliance Requirements

No compliance requirements. The upgrade is optional for all trading members. Members may continue using previous versions or upgrade at their convenience.

## Important Dates

- **January 8, 2026**: Circular issued
- **January 9, 2026, 7:00 PM onwards**: Version 12.50 available for download
- **January 10, 2026**: Testing available during mock trading session
- **January 12, 2026**: Version 12.50 goes live

## Impact Assessment

**Market Impact**: None. This is an optional software upgrade with no impact on trading operations or market structure.

**Operational Impact**: Low. Trading members who choose to upgrade will benefit from technical enhancements. The optional nature means members can upgrade based on their own readiness and schedule. Members using Leased Line connections have automated upgrade options, while others can manually download from the BSE website.

**Support**: BSE Helpdesk available at 022-45720400/600, 022-69158500, and bsehelp@bseindia.com for queries and assistance.
