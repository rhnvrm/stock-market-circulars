## Summary

BSE has notified trading members that the debt securities of Kumar Sinew Developers Private Limited (Scrip Code 953686) will be available for trading with revised terms effective Thursday, June 18, 2026. This notice is a continuation of Exchange Notice No. 20231206-9 dated December 06, 2023.

## Key Points

- Scrip Code 953686 will trade with revised terms from June 18, 2026
- The ISIN changes from INE994K07079 to INE994K07087
- Maturity date is extended from 31/12/2023 to 31/12/2028 (5-year extension)
- Scrip ID and Abbreviated Name change from 148KSDPL23 to 148KSDPL28
- Scrip Long Name changes from KSDPL-14.80%-31-12-23-PVT to KSDPL-14.80%-31-12-28-PVT
- Coupon rate remains unchanged at 14.80%

## Regulatory Changes

The following parameters for Scrip Code 953686 are being revised:

| Parameter | Existing | Revised |
|---|---|---|
| ISIN | INE994K07079 | INE994K07087 |
| Maturity Date | 31/12/2023 | 31/12/2028 |
| Scrip ID | 148KSDPL23 | 148KSDPL28 |
| Abbreviated Name | 148KSDPL23 | 148KSDPL28 |
| Scrip Long Name | KSDPL-14.80%-31-12-23-PVT | KSDPL-14.80%-31-12-28-PVT |

## Compliance Requirements

Trading members are required to:
- Note the revised ISIN (INE994K07087) for Scrip Code 953686 effective June 18, 2026
- Update internal systems to reflect the new ISIN, maturity date, and scrip identifiers
- Contact BSE DCS-Listing department for any clarifications

## Important Dates

- **Effective Date:** Thursday, June 18, 2026 — Scrip Code 953686 trades with revised terms
- **Revised Maturity Date:** December 31, 2028

## Impact Assessment

This update directly affects holders of Kumar Sinew Developers Private Limited debt securities under Scrip Code 953686. The maturity date extension from December 2023 to December 2028 represents a significant restructuring of the instrument terms. Trading members and investors holding these securities must update records to reflect the new ISIN (INE994K07087) to ensure accurate settlement and portfolio accounting. Broader market impact is minimal given the private limited company nature and single scrip scope.
