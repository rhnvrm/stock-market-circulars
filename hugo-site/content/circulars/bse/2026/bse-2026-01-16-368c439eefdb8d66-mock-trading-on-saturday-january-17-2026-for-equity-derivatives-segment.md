## Summary

BSE has scheduled a mock trading session for the Equity Derivatives segment on Saturday, January 17, 2026. Trading members can connect to the DR (Disaster Recovery) site of the BOLT Plus trading system using either BOLT TWS, third-party trading applications from empanelled vendors, or in-house developed systems through ETI APIs. The circular provides detailed technical connection parameters and web-based system access information.

## Key Points

- Mock trading session scheduled for Saturday, January 17, 2026
- Applies to Equity Derivatives segment only
- Trading members must use DR site connection parameters
- Multiple connectivity options available: BOLT TWS, 3rd party applications via ETI APIs
- Configuration settings must be changed at login time
- DR site ETI API connection parameters: Primary DR IP 10.255.241.16 Port 15910, Secondary DR IP 10.255.241.17 Port 15910
- Web-based systems (RTRMS, CLASS collateral, Extranet, EBOSS, LEIPS) accessible through specified URLs
- No new version release during this mock trading session

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Trading members must configure their trading systems to connect to DR site parameters
- BOLT TWS users must change technical connection parameters in configuration settings window using scrip profile icon or Shift+F12 shortcut
- Members using 3rd party applications must update ETI API connection parameters to DR site IPs and ports
- Members must use existing user IDs and passwords for web-based system access

## Important Dates

- **January 17, 2026 (Saturday)**: Mock trading session for Equity Derivatives segment

## Impact Assessment

This is a routine mock trading exercise with minimal impact on market operations. The session is designed for system testing and DR site validation purposes. Trading members need to temporarily reconfigure their systems to connect to the DR site, but this does not affect live trading operations. No impact on investors or market participants beyond trading members' technical teams.
