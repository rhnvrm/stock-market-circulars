## Summary

BSE has notified Trading Members of Record Dates fixed by certain companies for payment of interest and/or part redemption of their Non-Convertible Debentures (NCDs). Consequent to the part redemption, these debentures will trade with a reduced face value from the record/ex-date onward.

## Key Points

- Kolte-Patil Developers Ltd has four NCD series undergoing part redemption:
  - Scrip Code 974771 (KPDL-ZC-17-4-33-PVT, ISIN INE094I07049) - reduced face value Rs. 46,884/- per debenture
  - Scrip Code 976030 (KPDL-ZC-20-9-34-PVT, ISIN INE094I07072) - reduced face value Rs. 39,315/- per debenture
  - Scrip Code 977231 (KPDL-15-10-35-PVT, ISIN INE094I07080) - reduced face value Rs. 91,204/- per debenture
  - Scrip Code 977351 (KPDL-ZC-30-10-35-PVT, ISIN INE094I07098) - reduced face value Rs. 89,907/- per debenture
- JM Financial Products Limited (Scrip Code 975200, JMFPL-8.92%-16-11-26-PVT, ISIN INE523H07CB9) - reduced face value Rs. 33,333.34 per debenture
- Satin Housing Finance Limited (Scrip Code 976409, SHFL-11%-14-2-27-PVT, ISIN INE02YC07034) - reduced face value Rs. 66,667/- per debenture

## Regulatory Changes

No regulatory changes; this is a routine operational notice regarding scheduled interest payment and part redemption of debentures already issued.

## Compliance Requirements

Trading Members must update their systems to reflect the reduced face value of the affected debentures from the specified record/ex-date and settle trades accordingly under the relevant settlement numbers (673/2026-2027 for Kolte-Patil series, 684/2026-2027 for JM Financial and Satin Housing series).

## Important Dates

- 16 Jul 2026: Record Date and Ex Date for all four Kolte-Patil Developers Ltd NCD series
- 31 Jul 2026: Record Date and Ex Date for JM Financial Products Limited and Satin Housing Finance Limited NCDs

## Impact Assessment

This notice has limited market impact, affecting only holders and traders of the specific NCD instruments listed. It ensures accurate settlement pricing by adjusting face values to reflect part redemption. No impact on equity markets or the underlying companies' equity shares.
