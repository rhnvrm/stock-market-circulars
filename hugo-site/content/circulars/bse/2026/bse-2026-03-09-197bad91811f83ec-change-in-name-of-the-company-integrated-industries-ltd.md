## Summary

BSE has informed trading members that Integrated Industries Ltd (Scrip Code 531889) has officially changed its name to **Nurture Well Industries Limited**. The SCRIP ID and abbreviated name on the BOLT Plus system will be updated accordingly, effective March 13, 2026.

## Key Points

- Scrip Code 531889 previously traded under the name Integrated Industries Ltd
- New name: **Nurture Well Industries Limited**
- New SCRIP ID on BOLT Plus System: **NWIL**
- New Abbreviated Name on BOLT Plus System: **NWIL**
- Change is effective from **March 13, 2026**

## Regulatory Changes

No regulatory changes. This is a straightforward administrative update to reflect the company's legal name change on BSE's trading platform (BOLT Plus System).

## Compliance Requirements

- Trading members should update their records to reflect the new name and scrip ID (NWIL) for Scrip Code 531889
- No additional compliance action required from trading members beyond awareness of the updated identifier

## Important Dates

- **March 09, 2026**: Circular issued by BSE
- **March 13, 2026**: Effective date for name and SCRIP ID change on BOLT Plus System

## Impact Assessment

Minimal market impact. This is a purely administrative change affecting the display name and scrip identifier on BSE's BOLT Plus trading system. The scrip code (531889) remains unchanged. Trading members and investors holding positions in this stock should note the new ticker symbol NWIL to avoid confusion in order entry and portfolio tracking.
