## Summary

BSE announces the release of BOLT Pro TWS version 12.50, which will be available as an optional upgrade from Monday, January 12, 2026. The new version includes technical enhancements and will be applicable across Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The software will be available for download from January 9, 2026 (7:00 PM onwards) and can be tested during the mock trading session on January 10, 2026.

## Key Points

- BOLT Pro TWS version 12.50 is an OPTIONAL upgrade (not mandatory)
- Includes technical enhancements to the trading platform
- Available for all major segments: Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR
- Leased Line Connection users can upgrade via Auto-Download Interface by selecting version 12.50
- Alternative download available from BSE website at http://www.bseindia.com/help/
- BSE Helpdesk available for support: 022-45720400/600, 022-69158500, bsehelp@bseindia.com

## Regulatory Changes

No regulatory changes. This is a technical software upgrade with enhanced functionality.

## Compliance Requirements

No mandatory compliance requirements. Trading members may choose to upgrade to the new version at their discretion.

## Important Dates

- **January 8, 2026**: Circular announcement date
- **January 9, 2026 (7:00 PM onwards)**: Software version 12.50 available for download
- **January 10, 2026**: Testing available during mock trading session
- **January 12, 2026**: Version 12.50 goes live and becomes available for production use

## Impact Assessment

**Market Impact**: Minimal - This is an optional software upgrade with no changes to trading rules or market operations.

**Operational Impact**: Low - Trading members can continue using existing versions or choose to upgrade based on their operational needs. The optional nature allows flexibility in adoption timing.

**Technical Impact**: Technical enhancements may improve system performance for members who choose to upgrade, but no disruption expected for those continuing with current versions.
