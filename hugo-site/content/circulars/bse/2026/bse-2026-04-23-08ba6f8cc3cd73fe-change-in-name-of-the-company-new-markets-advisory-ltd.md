## Summary

BSE has notified trading members that New Markets Advisory Ltd (scrip code 508867) has informed the Exchange of a change in its company name. The new name, New Markets Avenue Limited, along with updated SCRIP ID and abbreviated name on the BOLT Plus system, will take effect from April 29, 2026.

## Key Points

- Existing company name: **New Markets Advisory Ltd**
- New company name: **New Markets Avenue Limited**
- Scrip code: **508867** (unchanged)
- New SCRIP ID on BOLT Plus: **NEWMKTAVE**
- New abbreviated name on BOLT Plus: **NEWMKTAVE**
- Change effective date: **April 29, 2026**

## Regulatory Changes

The SCRIP ID and ABBREVIATED NAME for scrip code 508867 on the BOLT Plus system will be updated to reflect the new company name. No changes to the scrip code itself.

## Compliance Requirements

Trading members should update their internal records and systems to reflect the new company name and SCRIP ID (NEWMKTAVE) effective April 29, 2026. No additional compliance action is required.

## Important Dates

- **April 23, 2026**: Notice issued by BSE
- **April 29, 2026**: Effective date for name change and BOLT Plus system update

## Impact Assessment

Minimal market impact. This is a routine administrative update affecting only the display name and scrip identifier for a single equity scrip (508867) on the BOLT Plus trading system. Trading members should ensure order management systems reference the updated identifier NEWMKTAVE from April 29, 2026 onwards to avoid order entry errors.
