## Summary

BSE has notified trading members that PAE Ltd (Scrip Code 517230) is changing its name to Aurique Limited. The Scrip ID and Abbreviated Name on the BOLT Plus System will be updated accordingly, effective June 29, 2026.

## Key Points

- PAE Ltd (Scrip Code 517230) is being renamed to Aurique Limited
- New Scrip ID on BOLT Plus System: AURIQUE
- New Abbreviated Name on BOLT Plus System: AURIQUE
- Change is effective from June 29, 2026
- Notice issued by Marian D'Souza, Assistant Vice President – Listing Compliance & Operations

## Regulatory Changes

The company's identification details on BSE's BOLT Plus trading system will change as follows:

| Field | Old Value | New Value |
|---|---|---|
| Company Name | PAE Ltd | Aurique Limited |
| Scrip ID | — | AURIQUE |
| Abbreviated Name | — | AURIQUE |

The Scrip Code (517230) remains unchanged.

## Compliance Requirements

Trading members should update their internal records and systems to reflect the new company name (Aurique Limited) and the updated Scrip ID and Abbreviated Name (AURIQUE) on BOLT Plus effective June 29, 2026.

## Important Dates

- **Notice Date:** June 22, 2026
- **Effective Date of Name Change on BOLT Plus:** June 29, 2026

## Impact Assessment

This is a routine administrative update with minimal market impact. The change affects only the display name and symbol identifier on BSE's trading platform. The Scrip Code remains the same, so no disruption to trading activity or settlement is expected. Investors and members should note the new name when searching or referencing the scrip after June 29, 2026.
