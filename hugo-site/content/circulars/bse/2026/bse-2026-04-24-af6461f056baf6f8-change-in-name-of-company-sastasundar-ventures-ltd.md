## Summary

BSE has notified trading members that Sastasundar Ventures Ltd (Scrip Code: 533259) has changed its corporate name to **Health X Platform Limited**. As a result, the company's SCRIP ID and ABBREVIATED NAME on the BOLT Plus System will be updated to **HEALTHX** effective April 30, 2026.

## Key Points

- Scrip Code 533259 undergoes a name change from Sastasundar Ventures Ltd to Health X Platform Limited
- New SCRIP ID on BOLT Plus System: **HEALTHX**
- New Abbreviated Name on BOLT Plus System: **HEALTHX**
- Change is effective from **April 30, 2026**
- Certificate of Incorporation has been submitted as supporting attachment
- Notice issued by Listing Operations department under Company Related / Equity segment

## Regulatory Changes

The SCRIP ID and ABBREVIATED NAME of Scrip Code 533259 on the BOLT Plus System will be officially changed to reflect the new corporate name. No change in scrip code (533259) has been indicated.

## Compliance Requirements

- Trading members must update their records and trading systems to reference the new SCRIP ID **HEALTHX** for Scrip Code 533259 before the effective date
- Any order management or algorithmic systems referencing the old scrip identifier should be updated accordingly

## Important Dates

- **April 24, 2026**: Notice issued by BSE
- **April 30, 2026**: Effective date for SCRIP ID and ABBREVIATED NAME change to HEALTHX on BOLT Plus System

## Impact Assessment

This is a routine corporate name change with moderate operational impact. Trading members and market participants who reference the old company name or scrip ID in their systems, watchlists, or algorithms must update to the new identifier HEALTHX before April 30, 2026 to avoid trade failures or mismatch errors. The scrip code (533259) remains unchanged, limiting broader disruption.
