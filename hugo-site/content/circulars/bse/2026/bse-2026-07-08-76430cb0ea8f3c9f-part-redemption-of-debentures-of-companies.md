## Summary

BSE has informed trading members that several companies have fixed record dates for payment of interest and/or part redemption of their outstanding Non-Convertible Debentures (NCDs). Following each record date, the respective debentures will trade with a reduced face value from the specified settlement/ex-date.

## Key Points

- Light Microfinance Private Limited (ISIN INE366T07196, Scrip ID LMPL-12.95%-28-7-28-2027): Record Date 13/07/2026, reduced face value Rs. 88,875/- per debenture.
- Casablanca Industries Private Limited (ISIN INE834U07019, Scrip ID CIPL-10%-22-11-30-2027): Record Date 14/07/2026, reduced face value Rs. 65,000/- per debenture.
- Lodha Developers Limited (ISIN INE670K07307, Scrip ID LDL-Reset Rate-21-01-28-2027): Record Date 15/07/2026, reduced face value Rs. 76,833/- per debenture.
- Lodha Developers Limited (ISIN INE670K07349, Scrip ID LDL-TBILLS-13-11-28-2027): Record Date 15/07/2026, reduced face value Rs. 57,247/- per debenture.
- Varthana Finance Private Limited (ISIN INE125T07311, Scrip ID VFPL-11.7%-6-8-27-2027): Record Date 22/07/2026, reduced face value Rs. 44,445/- per debenture.
- Krazybee Services Ltd (ISIN INE07HK07809, Scrip ID KSL-10.65%-06-11-26-2027): Record Date 22/07/2026, reduced face value Rs. 50,000/- per debenture.
- Moneyboxx Finance Limited (ISIN INE296Q07084, Scrip ID MFL-12.50%-07-12-26-2027): Record Date 23/07/2026, reduced face value Rs. 40,000/- per debenture.
- Ananya Finance For Inclusive Growth Private Limited (ISIN INE774L07099, Scrip ID AFFIGPL-13.65%-7-12-26-2027): Record Date 23/07/2026, reduced face value Rs. 40,000/- per debenture.
- Krazybee Services Ltd (ISIN INE07HK07759, Scrip ID KSL-11.82%-09-11-26-2027): Record Date 24/07/2026, reduced face value Rs. 25,000/- per debenture.
- An additional entry for AAHL (partially visible, reduced face value Rs. 59,468/-) is referenced but details are cut off in the source notice.

## Regulatory Changes

No new regulation is introduced; this is a routine operational notice under BSE's Corporate Actions/Debt Listing Operations process, informing members of scheduled interest payment/part redemption events on listed NCDs.

## Compliance Requirements

Trading members must ensure that trades in the affected debentures reflect the reduced face value from the respective settlement/ex-date onward, and should update internal systems accordingly to avoid settlement mismatches.

## Important Dates

- 13/07/2026: Record Date/Settlement Ex-Date – Light Microfinance Pvt Ltd
- 14/07/2026: Record Date/Settlement Ex-Date – Casablanca Industries Pvt Ltd
- 15/07/2026: Record Date/Settlement Ex-Date – Lodha Developers Ltd (two ISINs)
- 22/07/2026: Record Date/Settlement Ex-Date – Varthana Finance Pvt Ltd and Krazybee Services Ltd
- 23/07/2026: Record Date/Settlement Ex-Date – Moneyboxx Finance Ltd and Ananya Finance For Inclusive Growth Pvt Ltd
- 24/07/2026 (Ex-Date) / 25/07/2026 (Record Date): Krazybee Services Ltd (second ISIN)

## Impact Assessment

This notice affects only debt-market participants holding the specific NCDs listed, primarily private/unlisted companies raising debt via privately placed bonds. There is no impact on listed equity trading, indices, or broader market operations. Members dealing in these specific debt instruments must adjust for reduced face values to ensure accurate settlement.
