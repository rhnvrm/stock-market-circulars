## Summary

BSE has notified trading members that BJ Duplex Boards Ltd (scrip code 531647) has officially changed its corporate name to Prabhatam Infra Venture Limited. The Scrip ID and abbreviated name on the BOLT Plus System will be updated to PRABHATAM effective June 29, 2026.

## Key Points

- Scrip code 531647 remains unchanged
- Existing name: BJ Duplex Boards Ltd
- New name: Prabhatam Infra Venture Limited
- New Scrip ID on BOLT Plus System: PRABHATAM
- New abbreviated name on BOLT Plus System: PRABHATAM
- Effective date of change: June 29, 2026

## Regulatory Changes

No regulatory policy changes. This is a company-initiated name change communicated to the exchange under listing obligations.

## Compliance Requirements

Trading members should update their systems and records to reflect the new name and Scrip ID (PRABHATAM) for scrip code 531647 prior to June 29, 2026.

## Important Dates

- Notice date: June 22, 2026
- Effective date of name/Scrip ID change: June 29, 2026

## Impact Assessment

Minimal market impact. This is a routine administrative name change. The scrip code (531647) remains the same, ensuring continuity of trading. Investors and members need only note the new company name and updated identifier on the BOLT Plus System.
