## Summary

BSE has announced a mock trading session for the Commodity Derivatives segment on Saturday, March 14, 2026. The circular provides technical connection parameters for trading members to connect to the Disaster Recovery (DR) site of the BOLT Plus trading system during the mock session.

## Key Points

- Mock trading session scheduled for Saturday, March 14, 2026 on the Commodity Derivatives segment
- Trading members must update connection parameters to connect to the DR site of BOLT Plus
- Connection can be made via BOLT TWS, third-party trading applications, or in-house systems through ETI APIs
- BOLT TWS users must change technical connection parameters in the configuration settings window (accessible via scrip profile icon or Shift+F12)
- ETI API users must update IP and port settings in their respective trading applications
- DR Primary: IP 10.255.241.26, Port 14910
- DR Secondary: IP 10.255.241.27, Port 14910
- Web-based systems (RTRMS, CLASS Collateral, Extranet, etc.) accessible via provided URLs using existing credentials

## Regulatory Changes

No regulatory changes. This is an operational notice for a scheduled mock trading exercise.

## Compliance Requirements

- Trading members participating in the mock session must update their BOLT TWS or ETI API configuration settings to use the DR site connection parameters
- Members using third-party or in-house trading applications must update IP and port settings prior to the mock session
- Members connecting to web-based systems (RTRMS, EBOSS, LEIPS, Extranet, etc.) should use the URLs specified in Annexure 2

## Important Dates

- **Mock Trading Session Date:** Saturday, March 14, 2026 — Commodity Derivatives segment

## Impact Assessment

Minimal market impact. This is a routine DR site mock trading exercise to test operational readiness and connectivity for the Commodity Derivatives segment. Trading members are required to verify their ability to connect to the DR site using updated parameters. No impact on live trading or listed securities.
