## Summary

BSE has made available the penalty report for client code modifications of non-institutional trades executed during January 2026. The report file (PM0126.Clg no.) has been uploaded to the EQUITY TRANSACTION FOLDER on the BSE Extranet for trading members to download and save for future reference.

## Key Points

- Penalty report covers client code modifications for non-institutional trades during January 2026
- Report is based on Exchange Notice Nos. 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011)
- File name: PM0126.Clg no.
- Available in EQUITY TRANSACTION FOLDER on BSE Extranet
- Applies to all segments
- Members must save the files for future reference

## Regulatory Changes

No new regulatory changes. This circular implements existing penalty framework established in 2011.

## Compliance Requirements

- Trading members must download the penalty report from the Extranet
- Members should save the downloaded files for future reference and record-keeping
- Members with queries can contact BSE at:
  - Phone: 022-2272 8435 or 022-2272 5785
  - Email: bse.reports@bseindia.com

## Important Dates

- **Report Period**: January 2026
- **Notice Date**: February 10, 2026
- **Download Availability**: February 10, 2026 onwards

## Impact Assessment

This is a routine administrative circular with limited market impact. It affects only trading members who performed client code modifications for non-institutional trades during January 2026. The penalty report allows members to review any penalties levied and maintain compliance records. No operational changes or new requirements are introduced.
