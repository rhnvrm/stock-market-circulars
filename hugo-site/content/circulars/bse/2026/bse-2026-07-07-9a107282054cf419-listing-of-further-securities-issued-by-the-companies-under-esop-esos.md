## Summary

BSE has informed trading members that further new equity shares issued by 10 listed companies under their Employee Stock Option Plan (ESOP) / Employee Stock Option Scheme (ESOS) have been listed and admitted for trading with effect from Wednesday, July 08, 2026.

## Key Points

- 10 companies have new ESOP/ESOS shares listed for trading from July 8, 2026
- Aptus Value Housing Finance India Limited: 50,000 shares (ISIN INE852O01025, Scrip Code 543335, Face Value Rs.2)
- Dixon Technologies (India) Limited: 56,210 shares (ISIN INE935N01020, Scrip Code 540699, Face Value Rs.2)
- ICICI Lombard General Insurance Company Limited: 57,538 shares (ISIN INE765G01017, Scrip Code 540716, Face Value Rs.10)
- Intense Technologies Limited: 2,000 shares (ISIN INE781A01025, Scrip Code 532326, Face Value Rs.2)
- Kotak Mahindra Bank Limited: 200,183 shares (ISIN INE237A01036, Scrip Code 500247, Face Value Rs.1)
- L&T Finance Limited: 153,999 shares (ISIN INE498L01015, Scrip Code 533519, Face Value Rs.10)
- Pine Labs Limited: 5,581,713 shares (ISIN INE15B701018, Scrip Code 544606, Face Value Rs.1)
- Solara Active Pharma Sciences Limited: 2,250 shares (ISIN INE624Z01016, Scrip Code 541540, Face Value Rs.10)
- Tata Communications Limited: 16,749 shares (ISIN INE151A01013, Scrip Code 500483, Face Value Rs.10)
- Varun Beverages Limited: 168,750 shares (ISIN INE200M01039, Scrip Code 540180, Face Value Rs.2)
- None of the listed shares are under lock-in

## Regulatory Changes

No regulatory changes; this is a routine listing action under existing ESOP/ESOS listing procedures.

## Compliance Requirements

Trading members are required to take note of the listing of these additional shares for trading purposes; no specific compliance action is required from investors.

## Important Dates

- Notice Date: July 7, 2026
- Effective listing/trading date: Wednesday, July 8, 2026

## Impact Assessment

This is a routine administrative listing notice reflecting incremental share issuances under employee stock option schemes across 10 companies. The volumes involved are generally small relative to total outstanding shares (except Pine Labs, which sees a comparatively larger addition of ~5.58 million shares), resulting in minor dilution. No significant market or trading impact is expected.
