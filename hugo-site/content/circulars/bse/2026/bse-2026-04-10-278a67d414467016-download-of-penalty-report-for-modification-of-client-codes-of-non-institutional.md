## Summary

BSE has released the penalty report for client code modifications of non-institutional trades across all segments for the month of March 2026. The report (file name: PM0326.Clg) has been uploaded to the EQUITY TRANSACTION FOLDER in Extranet and trading members are advised to download and retain it.

## Key Points

- Penalty report covers client code modifications of non-institutional trades for March 2026
- Applicable across all segments
- Report filename: PM0326.Clg (located in EQUITY TRANSACTION FOLDER in Extranet)
- Issued in accordance with Exchange Notice Nos. 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011)
- Members must save the file for future reference

## Regulatory Changes

No new regulatory changes. This notice is issued under the existing framework established by BSE Notice Nos. 20110729-24 and 20110826-4 governing penalties for client code modifications in non-institutional trades.

## Compliance Requirements

- Trading members must log in to Extranet and download the penalty report (PM0326.Clg) from the EQUITY TRANSACTION FOLDER
- Members are required to save the file for future reference
- Members with queries should contact BSE Trading Operations using the contact details provided

## Important Dates

- Notice Date: April 10, 2026
- Period Covered: March 2026 (client code modifications)
- File available: As of April 10, 2026 in Extranet

## Impact Assessment

This is a routine monthly operational notice with limited market-wide impact. It affects only those trading members who carried out client code modifications for non-institutional trades during March 2026. Members should review the penalty report to understand any financial penalties applicable to them and maintain records for compliance and audit purposes. Contact: 022-2272 8435 / 022-2272 5785 or bse.reports@bseindia.com.
