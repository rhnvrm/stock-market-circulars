## Summary

BSE has notified trading members that Visa Steel Limited (scrip code 532721) has formally changed its name to VISA Chrome Limited. The change takes effect on May 20, 2026, at which point the company's SCRIP ID and abbreviated name on the BOLT Plus System will be updated accordingly.

## Key Points

- Existing company name: Visa Steel Limited
- New company name: VISA Chrome Limited
- Scrip Code: 532721 (unchanged)
- New SCRIP ID on BOLT Plus System: VISACHROME
- New Abbreviated Name on BOLT Plus System: VISACHROME
- Effective date: May 20, 2026
- Segment: Equity
- Certificate of Incorporation attached as supporting document

## Regulatory Changes

No regulatory rule changes. This is a corporate action — a change in the legal name of the listed entity — which requires BSE to update system identifiers to reflect the new name.

## Compliance Requirements

Trading members should update their records and systems to recognize the new SCRIP ID (VISACHROME) and abbreviated name (VISACHROME) for scrip code 532721 effective May 20, 2026. No active compliance action is required from members beyond awareness of the identifier change.

## Important Dates

- **May 14, 2026**: Notice issued by BSE
- **May 20, 2026**: Effective date for name change and updated BOLT Plus System identifiers

## Impact Assessment

Minimal market impact. This is a routine administrative name change. The scrip code (532721) remains unchanged, so existing positions, orders, and records linked to the code are unaffected. Trading members using SCRIP ID or abbreviated name in their systems should update references from the old identifiers to VISACHROME ahead of May 20, 2026 to avoid lookup failures on BOLT Plus System.
