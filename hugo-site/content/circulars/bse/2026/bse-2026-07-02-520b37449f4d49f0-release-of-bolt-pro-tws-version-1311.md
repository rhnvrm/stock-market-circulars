## Summary

BSE has announced the release of a new version of BOLT Pro TWS (Trader Work Station) - version 13.11 - which will go live effective Monday, July 06, 2026. This version release is compulsory for the Equity segment and optional for Equity Derivatives, Currency Derivatives, Commodity Derivatives, and Electronic Gold Receipts (EGR) segments.

## Key Points

- New BOLT Pro TWS version 13.11 to be released and made live from July 06, 2026
- Compulsory version release in Equity segment
- Optional version release in Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments
- Key functionality: technical enhancements related to Trading System API changes in the Equity segment, as published via earlier circulars dated May 15, 2026 (20260515-36), May 29, 2026 (20260529-13), and June 24, 2026 (20260624-30)
- Users on Leased Line Connection can upgrade via the TWS Version tab of the Auto-Download Interface
- Version can also be downloaded from the BSE website via the help portal link

## Regulatory Changes

No new regulatory changes; this circular implements technical/API handling changes previously communicated via circulars 20260515-36, 20260529-13, and 20260624-30 related to Trading System APIs in the Equity segment.

## Compliance Requirements

- Trading members must upgrade to BOLT Pro TWS version 13.11 for the Equity segment (compulsory) by the effective date
- Members trading in Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments may optionally upgrade
- Members should test the new version during the mock trading session on Saturday, July 04, 2026

## Important Dates

- July 03, 2026 (7:00 PM onwards): New version available for download
- July 04, 2026 (Saturday): Mock trading session to test the new version
- July 06, 2026 (Monday): New version live and compulsory for Equity segment

## Impact Assessment

This is a routine technical/operational update affecting trading members' front-end trading software. It does not directly impact listed securities, market pricing, or investor actions. Trading members must ensure timely upgrade to maintain uninterrupted trading access in the Equity segment from July 06, 2026.
