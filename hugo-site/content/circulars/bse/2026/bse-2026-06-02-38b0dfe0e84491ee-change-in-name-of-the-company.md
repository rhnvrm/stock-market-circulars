## Summary

BSE has informed trading members that Ardi Investments & Trading Company (Scrip Code: 504370) has officially changed its name to Ardi Alliances Limited. The SCRIP ID and abbreviated name on the BOLT Plus system will be updated to ARDIALL, effective June 08, 2026.

## Key Points

- Company name changes from **Ardi Investments & Trading Company** to **Ardi Alliances Limited**
- Scrip Code remains unchanged: **504370**
- New SCRIP ID on BOLT Plus System: **ARDIALL**
- New Abbreviated Name on BOLT Plus System: **ARDIALL**
- Change is applicable to the **Debt segment**
- Effective date: **June 08, 2026**

## Regulatory Changes

The SCRIP ID and abbreviated name for the company on the BOLT Plus system are being updated to reflect the new legal name. No changes to the scrip code or other identifiers are indicated.

## Compliance Requirements

Trading members should update their systems and records to reflect the new company name, SCRIP ID (ARDIALL), and abbreviated name (ARDIALL) for Scrip Code 504370 effective June 08, 2026.

## Important Dates

- **June 02, 2026**: Notice issued by BSE
- **June 08, 2026**: New name, SCRIP ID, and abbreviated name become effective on BOLT Plus system

## Impact Assessment

This is a routine administrative name change with minimal market impact. Trading members dealing in Scrip Code 504370 (debt segment) should note the updated identifiers to avoid any operational confusion post June 08, 2026. No change in scrip code, ISIN, or other trading parameters is indicated.
