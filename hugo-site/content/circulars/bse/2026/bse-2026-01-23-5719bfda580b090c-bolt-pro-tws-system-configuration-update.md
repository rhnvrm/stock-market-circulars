## Summary

BSE has issued updated system configuration requirements for the BOLT PRO TWS (Trading WorkStation) application. Trading members are requested to review and update their systems to meet the recommended specifications to ensure optimal performance and stability. The circular emphasizes that BOLT PRO TWS should be used strictly on Windows 11, as other Windows OS versions may lead to runtime errors.

## Key Points

- Updated system specifications released for BOLT PRO TWS application
- Minimum RAM requirement: 8 GB (16 GB recommended)
- Processor: Intel i5 or equivalent
- Operating system: Windows 11 (mandatory)
- Free disk space requirement: 100 GB
- Network requirement: 1 Ethernet port
- Using non-recommended OS versions may cause runtime errors
- Helpdesk support available at 022-45720400/600, 022-69158500 and bsehelp@bseindia.com

## Regulatory Changes

No regulatory changes. This is a technical infrastructure update.

## Compliance Requirements

- Trading members must review current system configurations
- Systems should be updated to meet recommended specifications
- BOLT PRO TWS must be run on Windows 11 operating system only
- Members experiencing issues should contact BSE Helpdesk

## Important Dates

- Circular Date: January 23, 2026
- No specific implementation deadline mentioned

## Impact Assessment

**Operational Impact:** Medium - Trading members need to assess and potentially upgrade their hardware and software infrastructure to meet updated specifications. The mandatory Windows 11 requirement may necessitate OS upgrades for some members.

**Cost Impact:** Potential capital expenditure for members whose systems do not meet the recommended specifications, particularly RAM upgrades (16 GB recommended) and Windows 11 licensing.

**Business Continuity:** Critical for ensuring stable trading operations and preventing runtime errors that could disrupt trading activities.
