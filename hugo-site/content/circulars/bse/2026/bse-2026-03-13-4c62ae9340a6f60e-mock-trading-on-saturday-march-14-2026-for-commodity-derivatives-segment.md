## Summary

BSE has scheduled a mock trading session on Saturday, March 14, 2026 for the Commodity Derivatives segment. The circular provides technical connection parameters for trading members to connect to the Disaster Recovery (DR) site of the BOLT Plus trading system during the mock session.

## Key Points

- Mock trading session scheduled for Saturday, March 14, 2026 on the Commodity Derivatives segment
- Trading members must update their technical connection parameters to connect to the DR site of BOLT Plus
- BOLT TWS users must change configuration settings via the scrip profile icon or using 'Shift+F12' shortcut keys
- Third-party trading applications connecting via ETI APIs must use the specified DR IP and port details
- Primary DR: IP 10.255.241.26, Port 14910
- Secondary DR: IP 10.255.241.27, Port 14910
- Web-based systems (RTRMS, CLASS collateral, Extranet, etc.) accessible via provided URLs during the session

## Regulatory Changes

No regulatory changes. This is an operational circular for a scheduled mock trading session.

## Compliance Requirements

- Trading members participating in the mock session must update their BOLT TWS or third-party application configuration to use DR connection parameters
- Members using ETI APIs must configure Primary DR (IP: 10.255.241.26, Port: 14910) or Secondary DR (IP: 10.255.241.27, Port: 14910)
- Members using web-based systems should use the DR URLs and existing credentials as provided

## Important Dates

- **Mock Trading Session Date:** Saturday, March 14, 2026 (Commodity Derivatives segment)

## Impact Assessment

Low operational impact. This is a routine mock/disaster recovery drill for commodity derivatives trading members. Participation requires temporary reconfiguration of trading system connection parameters. No impact on live markets or regular trading operations. Only affects members who choose to participate in the mock session.
