## Summary

BSE has made available the penalty report for client code modifications of non-institutional trades executed during December 2025. The report (file-name PM1225.Clg) has been uploaded to the EQUITY TRANSACTION FOLDER on the BSE Extranet and is accessible to trading members. This penalty report is issued pursuant to Exchange Notices 20110729-24 (July 29, 2011) and 20110826-4 (August 26, 2011).

## Key Points

- Penalty report covers client code modifications for non-institutional trades in all segments during December 2025
- Report file name: PM1225.Clg
- Available in EQUITY TRANSACTION FOLDER on BSE Extranet
- Members must save the file for future reference
- Based on Exchange Notices dated July 29, 2011 and August 26, 2011

## Regulatory Changes

No new regulatory changes introduced. This is a routine report as per existing framework established in 2011.

## Compliance Requirements

- Trading members must download the penalty report from BSE Extranet
- Members are required to save the downloaded files for their future reference
- Members should review penalties applicable to their operations
- For queries or clarifications, contact BSE Trading Operations at:
  - Phone: 022-2272 8435 or 022-2272 5785
  - Email: bse.reports@bseindia.com

## Important Dates

- **Report Period**: December 2025
- **Notice Date**: January 13, 2026
- **Download Availability**: January 13, 2026

## Impact Assessment

**Market Impact**: Minimal. This is an administrative compliance notification affecting only trading members who made client code modifications during the reporting period.

**Operational Impact**: Low. Trading members need to download and review the penalty report to identify any penalties applicable to their operations. This is a routine monthly process for affected members.

**Financial Impact**: Limited to specific trading members who violated client code modification rules during December 2025. Penalties are assessed per existing regulations.
