## Summary

BSE has notified trading members that Hari Govind International Ltd (scrip code 531971) has officially changed its name to **Popees Baby Care India Limited**. The SCRIP ID and abbreviated name on the BOLT Plus System will be updated accordingly, effective June 05, 2026.

## Key Points

- Existing company name: Hari Govind International Ltd
- New company name: Popees Baby Care India Limited
- Scrip code remains unchanged: **531971**
- New SCRIP ID on BOLT Plus System: **POPEES**
- New abbreviated name on BOLT Plus System: **POPEES**
- Change effective from: **June 05, 2026**

## Regulatory Changes

No regulatory rule changes are involved. This is a corporate-level name change communicated to the Exchange under listing obligations. BSE is updating its system identifiers to reflect the new legal name of the company.

## Compliance Requirements

- Trading members should update their records and systems to reflect the new company name and updated SCRIP ID/abbreviated name on BOLT Plus System.
- No action required from investors beyond noting the identifier change; the scrip code (531971) remains the same.

## Important Dates

- **June 01, 2026**: Notice date
- **June 05, 2026**: Effective date for name and identifier change on BOLT Plus System

## Impact Assessment

Minimal market impact. The change is purely administrative — only the company name and BOLT Plus System identifiers are updated. The scrip code (531971) is unchanged, so existing orders, holdings, and trades are unaffected. Trading members should ensure front-end systems and client-facing displays reflect the new name to avoid confusion.
