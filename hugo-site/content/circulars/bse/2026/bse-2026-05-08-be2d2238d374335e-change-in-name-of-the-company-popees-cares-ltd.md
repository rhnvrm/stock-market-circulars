## Summary

BSE has notified trading members that Popees Cares Ltd (Scrip Code: 530565) has officially changed its name to Koiya International Limited. The Scrip ID and Abbreviated Name on the BOLT Plus System will be updated accordingly effective May 13, 2026.

## Key Points

- Scrip Code 530565 company name changes from **Popees Cares Ltd** to **Koiya International Limited**
- New Scrip ID on BOLT Plus System: **KOIYAIN**
- New Abbreviated Name on BOLT Plus System: **KOIYAIN**
- Previous Scrip ID was **KOIYA INTERNATIONAL LIMITED** (display name)
- Change is effective from **May 13, 2026**
- Segment: Equity

## Regulatory Changes

The SCRIP ID and ABBREVIATED NAME for Scrip Code 530565 on the BOLT Plus System will be updated to reflect the new company name, Koiya International Limited, with abbreviated identifier KOIYAIN.

## Compliance Requirements

Trading members are advised to update their systems and references to reflect the new scrip name and identifier (KOIYAIN) for Scrip Code 530565 effective May 13, 2026. No action is required from investors beyond being aware of the name change.

## Important Dates

- **Notice Date:** May 8, 2026
- **Effective Date of Name Change on BOLT Plus System:** May 13, 2026

## Impact Assessment

This is a routine administrative name change with low market impact. Trading members using BOLT Plus System must update their references from the old scrip identifiers to the new KOIYAIN identifier. The underlying scrip code (530565) remains unchanged, so existing holdings and orders tied to the scrip code are unaffected. Investors should note the new trading name when monitoring their portfolios.
