## Summary

BSE is enhancing the Abridged Contract Master File in the Equity Derivative Segment to include Index derivatives contracts introduced on the BSE FOCUSED IT (FOCIT) Index. This update takes effect from end of day Friday, May 8, 2026. The file format remains unchanged.

## Key Points

- The Abridged Contract Master File (`BSE_EQD_CONTRACT_abr_<ddmmyyyyy>.csv` and `EQD_CO_abr_<ddmmyy>.csv`) will now include Index derivatives on the BSE FOCUSED IT (FOCIT) Index
- Previously, the file covered Index derivatives on SENSEX and BANKEX, plus limited stock derivatives contracts
- No changes to the existing abridged contract master file format
- File will continue to be available on the BSE India website and member extranet portal
- Effective from end of day Friday, May 8, 2026

## Regulatory Changes

This is an operational enhancement referencing earlier circulars 20240729-4 (July 29, 2024) and 20240826-11 (August 26, 2024) that originally introduced the Abridged Contract Master File. No new regulatory requirements are introduced.

## Compliance Requirements

- Trading members using the Abridged Contract Master File should note the inclusion of FOCIT Index derivatives from May 8, 2026
- No changes required to existing file parsing or processing logic as the format remains the same
- Members should update internal systems to handle the additional FOCIT Index contract entries if relevant to their operations

## Important Dates

- **May 8, 2026 (end of day, Friday):** FOCIT Index derivatives contracts will be included in the Abridged Contract Master File

## Impact Assessment

Low operational impact. This is a straightforward data addition to an existing supplementary file used by trading members in the Equity Derivative Segment. The unchanged file format means no technical integration work is required. Members trading or referencing FOCIT Index derivatives will benefit from having these contracts available in the abridged file alongside existing SENSEX and BANKEX contracts.
