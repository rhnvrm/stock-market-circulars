## Summary

BSE has announced a mock trading session for the Electronic Gold Receipts (EGR) segment scheduled for Saturday, January 17, 2026. The circular provides technical connection parameters for trading members to connect to the Disaster Recovery (DR) site of the BOLT Plus trading system during the mock session. This includes configuration details for both BOLT TWS users and third-party trading applications using ETI APIs, as well as web-based systems connectivity information.

## Key Points

- Mock trading session scheduled for Electronic Gold Receipts segment on Saturday, January 17, 2026
- Trading members can connect via BOLT TWS or third-party trading applications through ETI APIs
- DR connection parameters provided for Commodity Derivatives segment
- Primary DR IP: 10.255.241.26, Port: 14910
- Secondary DR IP: 10.255.241.27, Port: 14910
- Web-based systems (RTRMS, EBOSS, CLASS, Extranet) accessible through provided URLs
- BOLT TWS users need to change technical connection parameters in configuration settings

## Regulatory Changes

No regulatory changes. This is a routine mock trading exercise for disaster recovery testing.

## Compliance Requirements

- Trading members must update technical connection parameters in their trading systems
- BOLT TWS users should modify configuration settings using scrip profile icon or "Shift+F12" shortcut
- Third-party application users must change connection parameters to DR site IPs and ports
- Members can access web-based systems using existing credentials through specified URLs

## Important Dates

- **January 17, 2026 (Saturday)**: Mock trading session for Electronic Gold Receipts segment

## Impact Assessment

This mock trading session has minimal operational impact as it is conducted on a non-trading day (Saturday) and is designed purely for testing disaster recovery capabilities. Trading members need to make temporary configuration changes only for the mock session. No impact on live trading, settlements, or market operations. The exercise ensures business continuity preparedness for the EGR segment.
