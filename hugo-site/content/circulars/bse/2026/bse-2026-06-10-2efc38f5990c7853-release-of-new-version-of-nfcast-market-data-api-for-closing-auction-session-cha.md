## Summary

BSE has released an updated version of the NFCAST Market Data API (Ver. 6.2) as part of its ongoing initiative to facilitate Closing Auction Session (CAS) related changes. This is a continuation of earlier circulars (20260515-25 dated May 15, 2026 and 20260330-39 dated March 30, 2026) covering ETI API and NFCAST Market Data API updates for the Closing Auction Session.

## Key Points

- NFCAST Market Data API Ver. 6.2 has been released with minor changes specific to the Closing Auction Session.
- The updated API documentation is available on the BSE website at http://www.bseindia.com/nta.aspx.
- Members are directed to consult the "Change Log" section in the updated document for detailed information on the modifications.
- This notice is part of a series of updates supporting the rollout of the Closing Auction Session in the Equity segment.

## Regulatory Changes

No new regulatory changes are introduced. This circular is an operational update to an existing API to support the previously announced Closing Auction Session functionality in the Equity segment.

## Compliance Requirements

- Trading members and third-party front-end trading application vendors must initiate necessary changes in their respective applications to integrate NFCAST Market Data API Ver. 6.2.
- Members must test the updated integration in the BSE test environment prior to the live rollout to ensure a smooth transition.

## Important Dates

- **June 10, 2026**: Circular issued; updated API documentation made available on BSE website.
- No specific go-live date mentioned in this circular; members are advised to prepare and test ahead of the rollout.

## Impact Assessment

This update primarily affects trading members and third-party front-end trading application vendors who rely on the NFCAST Market Data API. The impact on end investors is indirect. Failure to integrate and test the updated API version may result in disruptions for members during the Closing Auction Session rollout. The changes are described as minor, limiting the scope of integration effort required.
