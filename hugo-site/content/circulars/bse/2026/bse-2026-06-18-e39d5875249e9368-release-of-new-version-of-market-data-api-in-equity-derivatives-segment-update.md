## Summary

BSE is extending the Market Data Streams API structure changes — previously announced for the Equity segment — to the Equity Derivatives segment. The new API versions will be available in the test environment from July 14, 2026, with phased go-live dates in August and September 2026. Trading members and application vendors must complete testing before the live rollout.

## Key Points

- This notice extends earlier API change circulars (20260330-39, 20260515-25, 20260610-38) to the Equity Derivatives segment.
- NFCAST API will be upgraded from Version 5.0 to Version 6.2.
- MDI/EMDI API will be upgraded from Version 1.4.4 to Version 1.4.5.
- EOBI (Version 1.4) remains unchanged.
- Test environment for both APIs will be available from July 14, 2026.
- Trading members and front-end vendors are required to complete testing before go-live.

## Regulatory Changes

New versions of Market Data Stream APIs are being introduced in the Equity Derivatives segment to facilitate Closing Auction Session related changes, aligning with previously implemented changes in the Equity segment.

## Compliance Requirements

- Trading members must update their systems to support the new API versions before go-live.
- Front-end trading application vendors must test and certify compatibility in BSE's test environment.
- For queries or clarifications, members should contact: bse.tech@bseindia.com or 022-2272 8053.

## Important Dates

| API | Test Environment Availability | Mock Trading | Go-Live |
|---|---|---|---|
| NFCAST API (v5.0 → v6.2) | July 14, 2026 | September 05, 2026 | September 07, 2026 |
| MDI/EMDI API (v1.4.4 → v1.4.5) | July 14, 2026 | August 08, 2026 | August 10, 2026 |
| EOBI (v1.4) | No change | No change | No change |

## Impact Assessment

This update primarily impacts trading members and front-end trading application vendors operating in the Equity Derivatives segment. Systems using NFCAST or MDI/EMDI APIs must be updated and tested before the respective go-live dates. The phased approach (test → mock → live) provides adequate lead time for vendors to ensure compatibility. Failure to upgrade before go-live may result in disruptions to market data feeds in the Equity Derivatives segment.
