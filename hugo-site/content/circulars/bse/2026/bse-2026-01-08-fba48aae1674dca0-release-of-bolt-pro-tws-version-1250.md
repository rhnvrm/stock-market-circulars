## Summary

BSE will release BOLT Pro TWS version 12.50 on Monday, January 12, 2026. This is an optional version upgrade available for Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives, and EGR segments. The version introduces technical enhancements and will be available for download from January 9, 2026, with testing available during the mock trading session on January 10, 2026.

## Key Points

- New BOLT Pro TWS version 12.50 to be released on January 12, 2026
- Optional upgrade (not mandatory) for all trading segments
- Technical enhancements included in this release
- Available for Equity, Equity Derivatives, Currency Derivatives, Commodity Derivatives & EGR segments
- Leased Line Connection users can upgrade via Auto-Download Interface
- Alternative download available from BSE website

## Regulatory Changes

No regulatory changes. This is a technical software upgrade with optional adoption.

## Compliance Requirements

No mandatory compliance requirements. Trading members may choose to upgrade at their discretion. Members using Leased Line Connection can select version 12.50 in the TWS Version tab of the Auto-Download Interface.

## Important Dates

- **January 8, 2026**: Notice date
- **January 9, 2026, 7:00 PM onwards**: Version 12.50 available for download
- **January 10, 2026**: Testing available during mock trading session
- **January 12, 2026**: Version 12.50 goes live

## Impact Assessment

Minimal market impact as this is an optional software upgrade. Trading members can continue using existing versions or upgrade to access technical enhancements. No disruption to trading operations expected. Members requiring assistance can contact BSE Helpdesk at 022-45720400/600, 022-69158500, or email bsehelp@bseindia.com.
