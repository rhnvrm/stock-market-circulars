## Summary

BSE has issued a user manual for corporates, financial institutions, and banks to submit change in shareholding pattern applications through the IEFS (Integrated Exchange Filing System) portal. The manual provides step-by-step instructions for accessing the portal at https://iefs.bseindia.in and completing all required sections including present shareholding pattern, proposed shareholding pattern, director details, declarations, undertakings, politically exposed person details, networth details, and document uploads.

## Key Points

- IEFS portal URL: https://iefs.bseindia.in
- Login credentials: Member code and Login ID are the clearing code of the member
- First-time users must reset their password using the forget password feature
- Process involves seven main sections: Present Shareholding Pattern, Proposed Shareholding Pattern, Present Director Details, Declarations, Undertaking, Politically Exposed Person Details, and Networth Details
- System allows adding multiple promoter and non-promoter details
- All mandatory fields must be completed in each section
- Users must save information in each tab before proceeding to the next section

## Regulatory Changes

No regulatory changes announced. This is an operational user manual for existing IEFS platform functionality.

## Compliance Requirements

- Corporates, financial institutions, and banks must use the IEFS portal for submitting change in shareholding pattern requests
- All mandatory fields in each section must be completed
- Required declarations and undertakings must be checked and saved
- Political exposure status must be disclosed
- Networth details must be provided
- Supporting documents must be uploaded (section appears incomplete in source)
- Director details including any business interests or positions in other entities must be disclosed

## Important Dates

No specific dates mentioned. This is a procedural manual applicable immediately.

## Impact Assessment

Minimal market impact. This is an operational guide for members who need to submit shareholding pattern changes. The document streamlines the existing process by providing clear instructions for the online platform, improving efficiency for corporates, financial institutions, and banks undergoing ownership changes. No trading or market operations are affected.
