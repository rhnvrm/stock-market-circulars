## Summary

BSE has notified trading members that Morganite Crucible India Ltd (Scrip Code: 523160) has officially changed its corporate name to **Foseco Crucible (India) Limited**. Consequently, the Scrip ID and Abbreviated Name on the BOLT Plus System will be updated effective April 13, 2026.

## Key Points

- Company name changed from **Morganite Crucible India Ltd** to **Foseco Crucible (India) Limited**
- Scrip Code remains unchanged: **523160**
- New Scrip ID on BOLT Plus System: **FOSECOC**
- New Abbreviated Name on BOLT Plus System: **FOSECOC**
- Notice issued by Marian Dsouza, Assistant Vice President – Listing Compliance & Operations

## Regulatory Changes

The company has formally informed BSE of its name change. BSE is updating the scrip identification details on the BOLT Plus trading system to reflect the new corporate name. No change in scrip code or other trading parameters is indicated.

## Compliance Requirements

- Trading members must update their records and trading systems to reference the new Scrip ID **FOSECOC** and new company name **Foseco Crucible (India) Limited** from the effective date
- All order entry and reference systems using the old abbreviated name should be updated accordingly

## Important Dates

- **Notice Date:** April 8, 2026
- **Effective Date of Name/Scrip ID Change:** April 13, 2026

## Impact Assessment

This is a routine corporate name change with moderate operational impact for trading members who reference the scrip by its abbreviated name or Scrip ID on BOLT Plus. The scrip code (523160) remains the same, minimising disruption. Members using automated systems keyed to the old Scrip ID or abbreviated name should update their configurations before April 13, 2026 to avoid order entry errors.
