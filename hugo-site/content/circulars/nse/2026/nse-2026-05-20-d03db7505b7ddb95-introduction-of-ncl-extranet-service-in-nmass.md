## Summary

NSE Clearing Ltd (NCL) has released User Manual v1.0 (May 2026) for the new **NCL Extranet** module integrated into the NMASS (NSE Member Access Support System) portal. The module enables eligible clearing members to connect to NCL systems via an extranet channel and perform key operational tasks including API registration, file uploads/downloads, and dashboard monitoring.

## Key Points

- NCL has introduced an 'Extranet' module within the existing NMASS member portal.
- The user manual (Version 1.0, May 2026) governs usage of the new module.
- Members access the service through the existing NMASS Login and Landing pages.
- Role and Access Management controls are in place, with access levels varying by member type.
- Key functional areas include: API Registration, Upload, Download, and a dual-pane Dashboard (NCL Dashboard + Extranet Downloads).
- Document is classified Non Confidential and is subject to document control.

## Regulatory Changes

No change to trading rules or regulatory framework. This circular is a technology/operational enhancement — the Extranet module extends NMASS with a new connectivity channel for NCL member firms.

## Compliance Requirements

- Clearing members intending to use the Extranet module must follow role and access management procedures described in the user manual.
- API registration must be completed through the designated section of the NMASS portal before using programmatic access.
- Members should ensure internal staff are assigned appropriate roles as per the access matrix defined for their member type.

## Important Dates

- **Release Date:** May 2026 (exact date not specified in the circular; circular issued 2026-05-20).
- **Document Version:** v1.0 — initial release; members should monitor for subsequent version updates via the document distributor.

## Impact Assessment

The introduction of the NCL Extranet module in NMASS primarily impacts **clearing members and their operations/technology teams**. Firms that rely on manual file transfers or existing portal workflows may need to update internal processes to leverage API registration and extranet-based data exchange. The dashboard enhancement improves visibility into NCL-level data and extranet downloads. Overall market impact is low; this is an infrastructure and connectivity improvement that streamlines member-NCL interactions rather than altering market structure or compliance obligations.
