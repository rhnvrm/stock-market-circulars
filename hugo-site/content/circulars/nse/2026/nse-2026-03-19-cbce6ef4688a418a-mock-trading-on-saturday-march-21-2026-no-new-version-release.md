## Summary

NSE is conducting a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, March 21, 2026. This is a routine drill with no new NEAT version release. The session uses NEAT 1.3.3 and follows existing connectivity parameters. Members are encouraged to participate and re-login to the live environment to ensure smooth operations for Monday, March 23, 2026.

## Key Points

- Mock trading session scheduled for Saturday, March 21, 2026 in the SLBM segment
- No new NEAT version release; current version is NEAT 1.3.3
- Normal Market open time: 11:15 hrs; Normal Market close time: 14:00 hrs
- Live re-login window: 16:30 hrs to 17:00 hrs
- All outstanding orders will be purged before each session
- Trades during the mock session carry no financial obligation (no fund pay-in or pay-out)
- NOTIS application will not be accessible during the mock session
- Members should use all trading software and re-login to live environment post-session

## Regulatory Changes

No regulatory changes. This circular is operational in nature and references existing guidelines:
- Connectivity: NSE/MSD/67674 dated April 24, 2025
- Software testing compliance: NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234
- UCC/PAN validity: NSE/ISC/51754 dated March 24, 2022 and NSE/ISC/52722 dated June 23, 2022

## Compliance Requirements

- Members enabled for SLBM live trading are automatically eligible for mock participation
- Participation uses existing User IDs, IPs, and Box ID mappings from the live environment
- Members using Non-NEAT Frontend (NNF) must manually clear orders in their systems
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- Set appropriate branch and user limits from the Corporate Manager Terminal before placing orders
- Email requests (with User ID and segment details) required for: Pro Enablement, CTCL Conversion, Password Reset, Unlocking Corporate Manager User ID, and Change of Trading Rights
- Telnet connectivity from intended IP and Box ID setup must be verified prior to session

## Important Dates

- **Mock Trading Date:** Saturday, March 21, 2026
- **Normal Market Open:** 11:15 hrs
- **Normal Market Close:** 14:00 hrs
- **Live Re-login Start:** 16:30 hrs
- **Live Re-login Close:** 17:00 hrs
- **Live Trading Resumes:** Monday, March 23, 2026

## Impact Assessment

Minimal market impact. This is a routine mock trading drill for SLBM segment members only. No financial obligations arise from mock trades. The session is intended to verify connectivity and readiness ahead of the live trading week starting March 23, 2026. Members not participating in SLBM are unaffected. NNF users should take note of the manual order purge requirement.
