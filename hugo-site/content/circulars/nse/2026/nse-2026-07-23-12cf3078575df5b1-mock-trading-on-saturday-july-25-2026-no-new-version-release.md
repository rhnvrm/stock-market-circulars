## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) segment on Saturday, July 25, 2026. No new software version will be released for this mock session. All SLBM segment members enabled for live trading are automatically eligible to participate using their existing User IDs, IPs, and Box ID mappings.

## Key Points

- Mock trading session scheduled for Saturday, July 25, 2026 in the SLBM segment.
- No new NEAT version release; members will use NEAT Version 1.3.3.
- Trading Session-1 will run from the Primary Site, and Trading Session-2 from the Disaster Recovery (DR) Site.
- Live re-login window is provided after the mock session to test connectivity for the live environment.
- All outstanding orders will be cleared before each session; NNF software users must manually clear orders in their own systems.
- Mock trades will not result in any fund pay-in or pay-out.
- Members may use Mock Trading or a Simulated Environment to satisfy SEBI regulatory testing requirements under SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234.
- Only valid and compliant UCC/PAN records uploaded and approved before the cutoff will be permitted; UCC validation will be skipped during contingency order entry time.
- The NOTIS application will remain accessible during the mock session.
- Members should use all their trading software and re-login to the live environment to confirm connectivity ahead of live trading on Monday, July 27, 2026.

## Regulatory Changes

None. This is a routine operational mock trading circular; no new regulations are introduced. Reference is made to existing circulars NSE/MSD/73993 (interactive connectivity), NSE/MSD/46441, NSE/ISC/51754, and NSE/ISC/52722 (UCC/PAN validation), and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 (software testing requirements).

## Compliance Requirements

- Members must ensure telnet connectivity to the Exchange host from the intended IP address.
- Box ID must be correctly created and mapped with appropriate segment messages.
- NNF (Non-NEAT Frontend) users must ensure the User ID is of dealer type, converted for NNF, and mapped to the correct IP.
- Branch and user limits must be set appropriately via the Corporate Manager Terminal before placing mock orders.
- Requests for Pro Enablement, CTCL conversion, or password management must be sent to the Exchange via email with User ID and segment details.
- UCC/PAN records must be valid and approved before the cutoff time for participation.

## Important Dates

- **Mock Trading Date:** Saturday, July 25, 2026
- Trading Session-1 (Primary Site): Normal Market open 13:30, close 16:30
- Trading Session-2 (DR Site): Normal Market open 17:30, close 17:45
- Live Re-login window: 19:30 to 20:00
- **Live Market Resumption:** Monday, July 27, 2026

## Impact Assessment

This circular pertains to a routine operational mock trading exercise limited to the SLBM segment. It has no financial obligation (no fund pay-in/pay-out) and no impact on listed stocks or trading prices. Impact is confined to SLBM segment trading members, who must ensure system readiness and connectivity ahead of the live session resuming on Monday, July 27, 2026.
