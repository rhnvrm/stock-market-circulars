## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, January 03, 2026. This is a routine testing session with no new version release. The session includes primary site trading, disaster recovery site trading, and live re-login activities to ensure smooth connectivity for Monday, January 5, 2026.

## Key Points

- Mock trading session scheduled for January 03, 2026
- No new NEAT version release (current version: NEAT 1.3.3)
- Trading Session-1 from Primary Site: 11:15 AM to 2:00 PM
- Trading Session-2 from DR Site: 3:00 PM to 3:30 PM
- Live Re-login: 5:00 PM to 6:00 PM
- Contingency switchover timing: 12:00 PM to 1:30 PM
- All outstanding orders will be cleared before each session
- No financial obligations from mock trades
- NOTIS application accessible 11:15 AM to 12:30 PM and 3:00 PM till market close
- Connect2NSE and Extranet unavailable from 12:30 PM till end of session
- Extranet API unavailable from 12:30 PM to 2:00 PM

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Valid and compliant UCC/PAN must be uploaded and approved before cutoff
- Only approved UCC/PAN will be allowed for trading
- UCC validation will be skipped during contingency time for order entry
- Members using NNF software must manually clear orders in their systems
- Members should participate using all trading software and re-login into live environment
- For NNF Application Users: User ID must be dealer type, converted for NNF, and mapped with IP
- Branch/User limits must be set appropriately

## Important Dates

- **Mock Trading Date**: January 03, 2026
- **Primary Site Trading Session**: 11:15 AM - 2:00 PM
  - Normal Market Open: 11:15 AM
  - Contingency Start: 12:00 PM
  - Contingency End: 1:30 PM
  - Normal Market Close: 2:00 PM
- **DR Site Trading Session**: 3:00 PM - 3:30 PM
- **Live Re-login Window**: 5:00 PM - 6:00 PM
- **Next Trading Day**: Monday, January 5, 2026

## Impact Assessment

**Market Impact**: None. This is a mock trading session with no financial obligations. All trades executed during the session will not result in any fund pay-in or pay-out.

**Operational Impact**: Low. Members need to ensure connectivity and participate in the mock session to verify their systems are ready for regular trading on January 5, 2026. This is a routine operational test including disaster recovery site testing.

**Member Action Required**: Members should participate in all sessions using their trading software and complete the live re-login process to ensure smooth connectivity for Monday trading. Technical support available via toll-free number 1800-266-0050 (Option 1) or email msm@nse.co.in.
