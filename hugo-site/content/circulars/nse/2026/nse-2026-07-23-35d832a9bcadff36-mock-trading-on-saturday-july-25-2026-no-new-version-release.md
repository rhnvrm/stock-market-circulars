## Summary

NSE will conduct a mock trading session in the Currency Derivatives (CD) segment on Saturday, July 25, 2026. This is a routine operational drill with no new software version release, aimed at ensuring member systems and connectivity are ready ahead of live trading resuming on Monday, July 27, 2026.

## Key Points

- Mock trading date: July 25, 2026
- Trading Session-1 from Primary Site: Normal Market open 13:15 hrs, close 14:00 hrs
- Trading Session-2 from DR Site: Normal Market open 15:00 hrs, close 15:30 hrs; Trade Modification end time 15:40 hrs
- Live Re-login window: 19:30 hrs to 20:00 hrs
- NEAT Version to be used: 3.5.4
- All outstanding orders will be purged before each session; NNF software users must manually clear orders
- Trades in the mock session carry no financial obligation (no fund pay-in/pay-out)
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be permitted; UCC validation is skipped during contingency order entry time
- NOTIS application will remain accessible during the mock session
- Members must participate using all trading software and re-login to the live environment to ensure smooth connectivity for Monday, July 27, 2026

## Regulatory Changes

None. This is a routine mock trading exercise, not a regulatory change. It references prior circulars (NSE/MSD/73993, NSE/MSD/46441, NSE/ISC/51754, NSE/ISC/52722) and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 regarding simulated environment/mock trading as an option for meeting software testing requirements.

## Compliance Requirements

- Members trading in the CD segment should participate in the mock session using all trading software they use in the live market.
- Members must ensure telnet connectivity to the Exchange host and correct Box ID setup on the intended IP.
- NNF (Non-NEAT Frontend) users must ensure User ID is of dealer type, converted for NNF, and properly mapped to the IP.
- Appropriate branch and user limits must be set via the Corporate Manager Terminal before placing mock orders.
- Members using NNF software must manually clear outstanding orders as automatic purge does not apply to them.
- Only UCC/PAN records validated and approved before the cutoff will be permitted to trade in the mock session.
- Requests for mock participation setup (e.g., new mappings) must be sent to the Exchange via email with User ID and segment details.

## Important Dates

- Mock trading session: Saturday, July 25, 2026
- Trading Session-1 (Primary Site): 13:15–14:00 hrs
- Trading Session-2 (DR Site): 15:00–15:30 hrs (trade modification until 15:40 hrs)
- Live re-login window: 19:30–20:00 hrs
- Live market resumption: Monday, July 27, 2026

## Impact Assessment

This circular has minimal market impact as it pertains solely to a scheduled mock/test trading session in the Currency Derivatives segment with no financial settlement implications. Its primary relevance is operational, ensuring trading members' systems, connectivity, and re-login processes function correctly ahead of the live market session on July 27, 2026. Non-compliant or unprepared members risk connectivity issues at market open on Monday, but there is no impact on listed securities, stock prices, or broader market operations.
