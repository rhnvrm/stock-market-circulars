## Summary

NSE Clearing Ltd (NCL) has introduced a new Extranet module within the NMASS (NSE Member Access Support System) portal. The module (Version 1.0, released May 2026) provides member firms with extranet connectivity features including API registration, file upload/download functionality, and a dedicated NCL dashboard with extranet download capabilities.

## Key Points

- NCL has launched a new 'Extranet' module in the NMASS member portal (Version 1.0)
- Members can register APIs through the new API Registration section
- Upload and Download functionalities are available under the new module
- A dedicated NCL Dashboard with an Extranet Downloads sub-section has been introduced
- Role and Access Management has been updated: roles are configurable per member type for the new menus
- The circular is accompanied by a full User Manual published by NSE Clearing Ltd

## Regulatory Changes

No direct regulatory changes. This circular introduces a new operational capability within the existing NMASS member portal framework. Access controls and role management guidelines have been extended to cover the new Extranet menus.

## Compliance Requirements

- Member firms should review the User Manual for the NCL Extranet module (V1.0)
- Administrators should configure appropriate roles and access permissions for relevant staff under the updated Role and Access Management framework
- Members intending to use API-based connectivity should complete API registration via the new module

## Important Dates

- Release Date: May 2026 (Version 1.0 of the Extranet User Manual)
- Circular Date: 2026-05-20

## Impact Assessment

This update is operationally significant for member firms that rely on the NMASS portal for clearing-related activities. The introduction of the Extranet module expands connectivity options, enabling members to interact with NCL systems via API and manage data transfers more efficiently. Role-based access controls ensure that only authorised personnel can access the new features. No immediate market-wide or trading impact is expected; the primary impact is on back-office and technology teams responsible for member portal administration and integration.
