## Summary

NSE Clearing Ltd (NCL) has introduced the **Extranet module** in the NMASS (NSE Member Access and Support System) portal (Version 1.0, May 2026). This circular provides the accompanying user manual detailing how trading members can access and use the new Extranet features including API registration, file upload/download, and a dedicated NCL dashboard.

## Key Points

- NCL has launched the Extranet module within the NMASS Member Portal under version 1.0
- The module introduces new menu items governed by role-based access control depending on member type
- Key functional areas include: API Registration, Upload, Download, and an NCL Dashboard with Extranet Downloads
- Access is managed through a Role and Access Management framework with distinct roles per member category
- The user manual is classified as Non-Confidential and subject to document control

## Regulatory Changes

No regulatory or rule changes are introduced. This is a functional enhancement to the existing NMASS portal infrastructure operated by NSE Clearing Ltd.

## Compliance Requirements

- Trading members are expected to familiarise themselves with the new Extranet module in NMASS
- Members should ensure appropriate roles are assigned to their users for accessing the new menus
- API registration workflows within the portal should be completed as applicable to the member's operations

## Important Dates

- **Release Date:** May 2026 (Version 1.0)
- No specific deadline or go-live cutover date is mentioned in the available content; members should refer to the full circular for any effective date

## Impact Assessment

Low operational impact. The Extranet module is an additive enhancement to the NMASS portal and does not alter trading rules, margin requirements, or settlement procedures. Members interacting with NCL for data exchange or file-based workflows may benefit from the new Upload/Download and Dashboard features. IT and operations teams at member firms should review the user manual and configure user roles accordingly.
