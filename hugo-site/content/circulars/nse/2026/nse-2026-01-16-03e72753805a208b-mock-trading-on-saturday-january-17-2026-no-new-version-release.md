## Summary

NSE will conduct a mock trading session in the Currency Derivatives Segment on Saturday, January 17, 2026. The session will use existing NEAT version 3.5.4 with no new version release. Members are required to participate using all trading software and re-login into the live environment to ensure smooth connectivity for Monday, January 19, 2026.

## Key Points

- Mock trading session scheduled for January 17, 2026
- Trading session: 10:00 to 13:00 hrs from Primary Site
- Trade modification end time: 13:10 hrs
- Live re-login window: 16:00 to 16:30 hrs
- NEAT version 3.5.4 will be used (no new version release)
- All outstanding orders will be cleared before session
- No financial obligations from mock trades
- Only valid and compliant UCC/PAN will be allowed

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Members must use NEAT version 3.5.4 available at /cdsftp/cdscommon/NEATCDS354
- Members should refer to NEAT Adapter circulars NSE/MSD/70421 (September 25, 2025) and NSE/MSD/71601 (December 02, 2025)
- Compatible NEAT Adapter versions: 1.0.20, 1.0.22, or 1.0.23 for Windows/Linux
- Applicable GR ports: 10877 (versions 1.0.20) or 10879 (versions 1.0.22 and 1.0.23)
- Members using NNF software must manually clear orders in their systems
- Only UCC/PAN uploaded and approved before cutoff will be allowed
- All members enabled for Currency Derivatives trading in live market are automatically eligible

## Important Dates

- **January 17, 2026**: Mock trading session (10:00 - 13:00 hrs)
- **January 19, 2026**: Live trading resumes (Monday)

## Impact Assessment

This is a routine mock trading session with minimal market impact. It allows members to test their trading systems and connectivity without any financial obligations. The session ensures that all trading members can verify their systems are functioning properly before the next live trading day. No new software version is being introduced, reducing the complexity of the mock session. Members should use this opportunity to test all trading software and confirm connectivity for seamless operations on Monday.
