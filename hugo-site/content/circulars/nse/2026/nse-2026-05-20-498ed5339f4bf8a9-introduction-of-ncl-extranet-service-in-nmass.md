## Summary

NSE Clearing Ltd (NCL) has released a User Manual (Version 1.0, May 2026) for the newly introduced **Extranet module** within the NMASS (NSE Member Access and Support System) portal. This circular serves as formal notice of the module's availability and provides guidance for clearing members on accessing and using the new functionality.

## Key Points

- NCL has introduced an "Extranet" module in the NMASS member portal.
- A User Manual (V1.0) has been published to guide members through the new features.
- The Extranet module includes: API Registration, Upload, Download, and a Dashboard section.
- The Dashboard provides access to the NCL Dashboard and Extranet Downloads.
- Role and Access Management functionality is built into the module, with access levels determined by Member Type.
- The module is accessible via the existing NMASS Member Portal login page.

## Regulatory Changes

No regulatory changes. This is a technical/operational update introducing new portal functionality for NCL clearing members.

## Compliance Requirements

- Clearing members wishing to use the Extranet service should refer to the User Manual for login, role configuration, and feature usage.
- Members must manage user roles and access permissions within their organization through the Role Management section of the portal.

## Important Dates

- **Version Release Date:** May 2026
- **Circular Date:** 2026-05-20

## Impact Assessment

Low operational impact. This update adds a new self-service capability for clearing members via the NMASS portal, streamlining access to NCL extranet resources such as downloads, API registration, and data uploads. No disruption to existing trading or clearing workflows is expected. Members who adopt the Extranet module may benefit from more efficient access to NCL data and services.
