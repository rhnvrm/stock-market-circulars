## Summary

NSE is conducting a mock trading session in the Commodity Derivatives (CD) segment on Saturday, March 07, 2026. This is a continuation of the earlier circular (download no. 71999) dated December 24, 2025. No new NEAT version is being released; the current NEAT 1.5.4 remains in use.

## Key Points

- Mock trading session scheduled for Saturday, March 07, 2026 in the Commodity Derivatives segment
- No new version release; NEAT version 1.5.4 remains unchanged
- Download path on NSE Extranet: /comtftp/comtcommon/NEATCO154
- All outstanding orders will be purged before each session
- Trades during mock sessions carry no financial obligation (no fund pay-in or pay-out)
- UCC validation will be skipped during contingency time for order entry
- NOTIS application will be accessible during the mock session
- Members should re-login into live environment to ensure smooth connectivity for Monday, March 09, 2026

## Regulatory Changes

No regulatory changes. This is a routine operational mock session circular.

## Compliance Requirements

- Members must ensure Box IDs are created on the correct IP with appropriate segment messages
- NNF application users must ensure User ID is of dealer type, converted for NNF, and mapped with the IP
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- Members must set appropriate branch and user limits before the session
- Members should refer to circulars NSE/MSD/67674 (April 24, 2025) and NSE/COM/71599 (December 2, 2025) for interactive connectivity details
- Software testing should comply with circulars NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234

## Important Dates

- **Mock Session Date:** Saturday, March 07, 2026
- Normal Market Open: 10:00
- Contingency Start: 12:00
- Contingency End: 13:00
- Normal Market Close: 14:00
- Position Limit/Collateral Value Setup Cutoff: 14:10
- Trade Modification End: 14:10
- Live Re-login Start: 16:30
- Live Re-login Close: 17:00
- **Live Market Resumption:** Monday, March 09, 2026

## Impact Assessment

This circular has minimal market impact as it pertains to a routine mock/test trading session for Commodity Derivatives segment members. The primary purpose is to allow members to verify connectivity, test trading software, and ensure readiness for the live market on March 09, 2026. No financial settlements occur during the mock session. Only member firms trading in the CD segment are affected.
