## Summary

NSE Clearing Limited (NCL) has issued a circular (Download Ref No: NCL/CMPT/75369, Circular Ref. No: 0271/2026) announcing the implementation of OTP-based Two Factor Authentication (2FA) for user login on NCMS-CM (applicable to both Admin users and Sub users), effective from 25-July-2026. Alongside this security enhancement, a new NCMS EXE version 4.2.3 will be released on the same date, and members must download and install it to continue accessing the system.

## Key Points

- OTP-based 2FA will be mandatory for all NCMS-CM logins (Admin and Sub users) starting 25-Jul-2026.
- First-time login after rollout requires registration of a valid mobile number and email ID.
- OTP is sent to the registered mobile number and email ID upon each login attempt.
- Users can regenerate an OTP after 30 seconds if an invalid OTP is entered.
- After 4 consecutive invalid OTP entries, the user ID will be disabled, requiring the member to contact NSE Clearing Ltd to restore access.
- A new NCMS EXE version 4.2.3 will be released effective 25-Jul-2026, downloadable via https://www.connect2nse.com or the member portal path /cmftp/common/NCMSSetup_423_25Jul2026.

## Regulatory Changes

This is an operational/security enhancement mandated by NSE Clearing Ltd rather than a SEBI regulatory change. It introduces a mandatory 2FA layer for NCMS-CM access to strengthen login security for all member users.

## Compliance Requirements

- Members must register a valid mobile number and email ID for all Admin and Sub users prior to or during first login post-implementation.
- Members must download and install NCMS EXE version 4.2.3 from the specified sources before 25-Jul-2026 to ensure uninterrupted access.
- Members should be aware of the 4-invalid-OTP lockout policy and the process to contact NSE Clearing (cm_clearing_ops@nsccl.co.in or 18002660050, IVR Option 2) if a user ID is disabled.

## Important Dates

- **25-July-2026**: OTP-based 2FA becomes mandatory for NCMS-CM login; NCMS EXE version 4.2.3 becomes available and required.

## Impact Assessment

The change is purely operational, affecting the login workflow for clearing members' Admin and Sub users on NCMS-CM. There is no direct impact on trading, listed securities, or stock prices. However, members must act proactively (register contact details, upgrade software) before the effective date to avoid login disruptions or account lockouts, making this a medium-priority operational compliance item for all NSE Clearing members.
