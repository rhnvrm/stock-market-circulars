## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, March 7, 2026. There is no new version release associated with this mock session. Members are advised to participate using all trading software to ensure smooth connectivity ahead of the live trading session on Monday, March 9, 2026.

## Key Points

- Mock trading session scheduled for Saturday, March 07, 2026 in the SLBM segment
- No new version release; NEAT version remains 1.3.3
- Normal Market open time: 10:15 hrs
- Contingency Start Time: 12:00 hrs
- Contingency Close Time: 13:00 hrs
- Normal Market close time: 14:00 hrs
- Live Re-login window: 16:30 to 17:00 hrs
- All outstanding orders will be cleared before each session
- Trades during mock sessions carry no financial obligation (no fund pay-in or pay-out)
- UCC validation will be skipped during contingency time for order entry
- NOTIS application will be accessible during the mock session

## Regulatory Changes

No regulatory changes. Members should refer to existing circulars NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 for software testing requirements. Members may use either Mock Trading or Simulated Environment to meet regulatory requirements.

## Compliance Requirements

- All members enabled for SLBM trading in the live market are automatically eligible
- Members using NNF software must manually clear outstanding orders in their systems
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- NNF users must ensure: dealer-type User ID, NNF conversion, and IP mapping
- Set appropriate branch and user limits from Corporate Manager Terminal before placing orders
- For password resets, pro enablement, CTCL conversion, or unlocking Corporate Manager IDs, send email requests with User ID and segment details
- Connectivity parameters per circular NSE/MSD/67674 dated April 24, 2025

## Important Dates

- Mock Trading Date: Saturday, March 07, 2026
- Live Trading Resumes: Monday, March 09, 2026

## Impact Assessment

This is a routine operational mock session with no financial impact. It affects members trading in the SLBM segment and serves as a connectivity and system readiness check before the live trading week begins on March 9, 2026. No new software version is being released, minimizing preparation overhead for members.
