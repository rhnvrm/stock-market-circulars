## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) segment on Saturday, July 11, 2026. No new software version will be released for this mock session. The circular provides the mock session schedule and detailed instructions for members to ensure smooth connectivity and system readiness ahead of the resumption of live trading on Monday, July 13, 2026.

## Key Points

- Mock trading session scheduled for Saturday, July 11, 2026 in the SLBM segment.
- No new NEAT version release; mock session uses NEAT Version 1.3.3.
- Normal Market open time: 13:00 hrs; Normal Market close time: 15:00 hrs.
- Live Re-login window: start time 17:30 hrs, close time 18:00 hrs.
- All outstanding orders will be purged before each session; NNF software users must manually clear orders.
- Members may use Mock Trading or Simulated Environment to meet regulatory software testing requirements per NSE/MSD/46441 and SEBI/HO/MRD1/DSAP/CIR/P/2020/234.
- No financial obligation: trades during mock sessions will not result in fund pay-in or pay-out.
- UCC/PAN validation required; skipped only during contingency time for order entry.
- NOTIS application will be accessible during the mock session.
- Members must participate using all trading software and re-login to the live environment to ensure smooth connectivity for Monday, July 13, 2026.

## Regulatory Changes

No new regulatory changes; this is an operational circular referencing existing requirements under NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 regarding software testing via mock trading or simulated environments.

## Compliance Requirements

- Members enabled for SLBM trading in the live market are automatically eligible for mock trading using existing User IDs, IPs, and Box ID mappings.
- Members must verify Telnet connectivity to the Exchange host from the intended IP address.
- Box ID must be confirmed as created on the IP with appropriate segment messages.
- NNF application users must ensure User ID is dealer type, converted for NNF, and mapped with the IP.
- Branch and user limits must be set via the Corporate Manager Terminal before placing mock orders.
- Requests for Pro Enablement, CTCL Conversion, password reset/unlocking of Corporate Manager User ID, and trading rights changes must be sent via email to the Exchange.
- Only valid and compliant UCC/PAN records uploaded and approved before cutoff will be permitted; queries to uci@nse.co.in.

## Important Dates

- Mock trading session: Saturday, July 11, 2026.
- Market open: 13:00 hrs; Market close: 15:00 hrs.
- Live re-login window: 17:30 hrs to 18:00 hrs.
- Live trading resumes: Monday, July 13, 2026.

## Impact Assessment

This is a routine, low-impact operational circular affecting only trading members participating in the SLBM segment. There is no financial obligation, fund pay-in/pay-out, or impact on listed securities or live market trading. Members should ensure system and connectivity readiness during the mock session to avoid disruptions when live trading resumes on Monday, July 13, 2026.
