## Summary

NSE will conduct a mock trading session in the Currency Derivatives (CD) segment on Saturday, July 18, 2026, using the existing NEAT 3.5.4 version (no new version release). The circular provides the session schedule, connectivity guidance, and general instructions for members participating in the mock environment ahead of live trading resumption on Monday, July 20, 2026.

## Key Points

- Mock trading date: July 18, 2026 in the Currency Derivatives segment.
- Normal market open time: 16:45 hrs; normal market close time: 17:30 hrs; trade modification end time: 17:40 hrs.
- Live re-login window: 19:00 hrs to 19:30 hrs.
- No new NEAT version is being released; current version 3.5.4 remains in use.
- All outstanding orders will be purged before each session; NNF software users must manually clear orders.
- Trades executed during the mock session carry no financial obligation (no fund pay-in/pay-out).
- Only valid, compliant UCC/PAN records approved before cutoff will be permitted; UCC validation is skipped during contingency order entry time.
- NOTIS application will remain available during the mock session.
- Members should participate using all trading software and re-login to the live environment to ensure smooth connectivity for Monday, July 20, 2026.
- Eligibility for mock participation is automatic for all members enabled for live CD segment trading, using existing User IDs, IPs, and Box ID mappings.

## Regulatory Changes

No new regulatory changes; this is an operational circular referencing existing requirements under NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 regarding mock trading or simulated environment participation for regulatory compliance.

## Compliance Requirements

- Members must ensure valid and compliant UCC/PAN records are uploaded and approved before the specified cutoff.
- Members using NNF software must manually purge outstanding orders in their systems.
- Members must verify telnet connectivity and Box ID setup for the IPs they intend to use.
- NNF application users must ensure dealer-type User IDs are converted and mapped to the correct IP.
- Appropriate branch and user limits must be set via the Corporate Manager Terminal before placing mock orders.
- Requests for Pro enablement, CTCL conversion, password resets, unlocking of Corporate Manager IDs, and trading rights changes must be sent to the Exchange via email.

## Important Dates

- Mock trading session: Saturday, July 18, 2026.
- Market timings: Open 16:45 hrs, Close 17:30 hrs, Trade modification end 17:40 hrs.
- Live re-login window: 19:00 hrs to 19:30 hrs.
- Live market resumption: Monday, July 20, 2026.

## Impact Assessment

This is a routine, low-impact operational notice affecting only Currency Derivatives segment trading members' technical systems and connectivity testing. There is no financial obligation from mock trades and no impact on listed securities, market pricing, or investors. Members should ensure participation to confirm system readiness ahead of the next live trading session.
