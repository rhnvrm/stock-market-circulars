## Summary

NSE's Member Service Department (Circular Ref. No. 39/2026, Download Ref No. NSE/MSD/74261) announces the release of updated Linux and Windows UDP Server software for the Electronic Gold Receipts (EGR) segment. This release follows circular NSE/EGR/74203 dated May 13, 2026 on EGR trading parameters. Members must download and install the applicable version from the Extranet path starting May 16, 2026.

## Key Points

- Two new UDP server versions released for the EGR segment: Windows (v4.03.01) and Linux 64-bit (v4.03.00)
- Windows setup file: `UDPSERVER_V4.03.01.exe` available at Extranet path `egftp/egcommon/UDP403`
- Linux setup file: `LINUXUDPSERVER_v4.03.00_64.tar` available at Extranet path `egftp/egcommon/LinuxUDPSRVR`
- Software will be available from **May 16, 2026 at 09:00 hours** onwards on the Extranet
- User manuals for both Windows and Linux OS are attached to the circular for installation and configuration guidance
- Toll Free support available at **1800 266 0050 (Option 1)** for configuration assistance

## Regulatory Changes

No regulatory rule changes. This circular is a technical software release notification tied to the EGR segment trading parameters announced in NSE/EGR/74203 dated May 13, 2026.

## Compliance Requirements

- All members participating in the Electronic Gold Receipts (EGR) segment must download the appropriate UDP server version (Windows or Linux) from the Extranet
- Members should follow the compatibility, installation, and configuration instructions in the attached user manuals before deploying the new server
- Members experiencing configuration issues should contact NSE Member Support via Toll Free no: 1800 266 0050 (Option 1) or email msm@nse.co.in

## Important Dates

- **May 13, 2026**: Reference circular NSE/EGR/74203 issued on EGR trading parameters
- **May 15, 2026**: This circular issued (NSE/MSD/74261)
- **May 16, 2026, 09:00 hours**: UDP server software made available on Extranet for download

## Impact Assessment

This release has a medium operational impact for members actively trading in the EGR (Electronic Gold Receipts) segment. Members must update their UDP server software to the new versions to ensure continued compatibility with the EGR trading infrastructure. The release is routine in nature (a version increment) and is supported by detailed user manuals and a dedicated helpline, minimising disruption risk. No impact on equity, derivatives, or other non-EGR segments.
