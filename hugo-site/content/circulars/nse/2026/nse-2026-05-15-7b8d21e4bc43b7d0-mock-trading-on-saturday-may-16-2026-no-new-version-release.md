## Summary

NSE is conducting a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, May 16, 2026. No new software version will be released for this session. The session uses NEAT version 1.3.3 and is intended to help members validate connectivity and software readiness ahead of the live trading week beginning Monday, May 18, 2026.

## Key Points

- Mock trading in SLBM segment scheduled for Saturday, May 16, 2026
- No new version release; NEAT version 1.3.3 will be used
- Normal market open: 12:45 hrs; Normal market close: 15:15 hrs
- Live re-login window: 18:30 to 19:00 hrs
- All outstanding orders will be purged before each session
- Trades during mock sessions carry no financial obligation (no fund pay-in or pay-out)
- Connect2NSE and Extranet facility unavailable from 2:00 PM till end of mock session
- Extranet API unavailable from 2:00 PM to 3:00 PM
- NOTIS application will be accessible during the mock session
- UCC/PAN validation will be skipped during contingency time for order entry

## Regulatory Changes

No regulatory changes. This is a routine operational mock trading circular. Members may use Mock Trading or Simulated Environment to meet regulatory testing requirements as per SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 and NSE circular NSE/MSD/46441.

## Compliance Requirements

- Members must ensure only valid and compliant UCC/PAN uploaded and approved before cutoff are used
- NNF (Non-NEAT Frontend) users must manually clear orders before sessions; User IDs must be of dealer type, converted for NNF, and mapped with the relevant IP
- Members should set appropriate branch and user limits from the Corporate Manager Terminal before placing orders
- Members should participate using all trading software and re-login to the live environment post-session to ensure smooth connectivity for Monday, May 18, 2026
- For Pro Enablement or CTCL Conversion, send email requests with User ID and segment details
- Connectivity details per circular NSE/MSD/73993 dated April 30, 2026

## Important Dates

- **May 16, 2026**: Mock trading session in SLBM
- **May 18, 2026**: Live trading resumes; members must re-login to live environment after mock session
- **April 30, 2026**: Reference circular NSE/MSD/73993 for connectivity parameters

## Impact Assessment

This circular has low market impact as it pertains solely to an operational rehearsal for SLBM segment members. No financial settlements occur during mock sessions. The primary impact is on member firms' IT and operations teams who need to validate their trading systems, connectivity (Box ID, IP mappings, telnet), and software setup ahead of the live week. Members using NNF software should take note of the manual order purge requirement and the temporary unavailability of Connect2NSE and Extranet API during specified hours.
