## Summary

NSE has enabled members to submit User ID management requests for the Electronic Gold Receipts (EGR) segment via the ENIT platform. This is a continuation of circular NSE/MSD/74023 dated April 30, 2026, and extends existing ENIT functionality to cover EGR-specific NEAT, CTCL, and DMA user ID operations. The facility is effective from June 08, 2026.

## Key Points

- Members can now submit EGR segment User ID requests through the ENIT portal under the Trade > User ID Request and Trade > Member Reporting Pre Trade menus.
- Supported operations include: New NEAT User ID, Change NEAT User Name, NEAT User ID Validity Updation, Multi Lock NEAT User ID, NEAT to CTCL Conversion, New Non-NEAT User ID, and CTCL to Non-NEAT Conversion.
- Pre-trade reporting operations supported: CTCL ULC Mapping, CTCL ID Activation/Modification/Deactivation/Bulk Upload, and DMA ID Activation/Deactivation.
- Requests submitted before 1:30 PM on a working day will be processed the same day; in exigencies, within T+2 days; requests after 1:30 PM will be processed the next working day.
- Manual request processing via annexures (from circular NSE/MSD/74023) is available only if ENIT submission faces technical issues.

## Regulatory Changes

No new regulatory framework is introduced. This circular operationalizes the earlier circular NSE/MSD/74023 (April 30, 2026) by enabling ENIT-based self-service for EGR segment User ID management, reducing reliance on manual/paper-based requests.

## Compliance Requirements

- Members must use ENIT as the primary channel for EGR segment User ID requests.
- Members must comply with Exchange circular NSE/MSD/73956 dated April 29, 2026, and all applicable SEBI/Exchange/Regulatory Authority rules.
- Manual annexure-based requests are permitted only in case of ENIT submission issues, and require necessary approvals.

## Important Dates

- **Effective Date**: June 08, 2026
- **Same-day processing cutoff**: 1:30 PM on working days
- **Reference circular**: NSE/MSD/74023 dated April 30, 2026

## Impact Assessment

This is an administrative/operational enhancement with low market impact. It streamlines User ID management for members active in the EGR segment by enabling self-service through ENIT, reducing turnaround time and manual processing overhead. No impact on investors, securities prices, or broader market operations.
