## Summary

NSE will conduct a mock trading session in the Commodity Derivatives (CD) segment on Saturday, July 11, 2026. No new software version is being released for this mock session. The circular provides session timings, connectivity guidelines, and re-login instructions to ensure smooth participation and readiness for live trading on the following Monday.

## Key Points

- Mock trading session scheduled for Saturday, July 11, 2026 in the Commodity Derivatives segment.
- No new NEAT version release for this mock session (version remains 1.5.5).
- Normal Market open at 12:45 hrs and close at 15:00 hrs; Position Limit/Collateral value set-up cutoff and Trade Modification end time at 15:10 hrs.
- Live re-login window from 17:30 hrs to 18:00 hrs.
- All outstanding orders will be cleared before the session; NNF software users must manually clear orders.
- Trades in the mock session will not result in any fund pay-in or pay-out.
- Connect2NSE and Extranet facility unavailable from 1:00 pm to 6:00 pm; Extranet API unavailable from 2:30 pm to 3:30 pm.
- Members must participate and re-login to the live environment to ensure smooth connectivity for the next trading day, Monday, July 13, 2026.
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed; UCC validation skipped during contingency time.

## Regulatory Changes

None. This is an operational circular referencing existing SEBI and NSE guidelines (SEBI/HO/MRD1/DSAP/CIR/P/2020/234, NSE/MSD/73993, NSE/MSD/46441, NSE/ISC/51754, NSE/ISC/52722) that already permit Mock Trading or a Simulated Environment to satisfy regulatory testing requirements.

## Compliance Requirements

- Members enabled for live CD segment trading are automatically eligible; no separate registration needed.
- Ensure telnet connectivity to the Exchange host and correct Box ID/IP mapping before the session.
- NNF (Non-NEAT Frontend) users must ensure User ID is dealer type, NNF-converted, and IP-mapped.
- Set appropriate branch and user limits via the Corporate Manager Terminal before placing orders.
- Submit email requests with User ID and segment details for Pro Enablement, CTCL Conversion, or password management needs.
- Ensure UCC/PAN records are valid and approved before the cutoff time.

## Important Dates

- Mock session date: Saturday, July 11, 2026.
- Market open: 12:45 hrs; Market close: 15:00 hrs.
- Position limit/collateral cutoff and trade modification end: 15:10 hrs.
- Live re-login window: 17:30 hrs to 18:00 hrs.
- Connect2NSE/Extranet unavailable: 1:00 pm–6:00 pm; Extranet API unavailable: 2:30 pm–3:30 pm.
- Next live trading day requiring successful re-login: Monday, July 13, 2026.

## Impact Assessment

This is a routine, low-impact operational notice limited to members trading in the Commodity Derivatives segment. There is no effect on listed securities, live trading, or investor funds since mock trades do not settle. Members should ensure technical readiness (connectivity, re-login, UCC validation) to avoid disruption when live trading resumes on July 13, 2026.
