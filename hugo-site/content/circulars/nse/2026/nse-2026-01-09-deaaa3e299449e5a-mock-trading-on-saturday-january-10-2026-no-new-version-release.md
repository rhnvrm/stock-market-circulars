## Summary

NSE will conduct a mock trading session in the Currency Derivatives Segment on Saturday, January 10, 2026. The session will run from 11:00 to 12:30 hours with no new version release. Members are required to use NEAT version 3.5.4 and should participate using all trading software to ensure smooth connectivity for Monday, January 12, 2026.

## Key Points

- Mock trading date: January 10, 2026
- Trading session: 11:00 to 12:30 hours from primary site
- Trade modification end time: 12:40 hours
- Live re-login window: 19:00 to 19:30 hours
- NEAT version to be used: 3.5.4 (download path: /cdsftp/cdscommon/NEATCDS354)
- No new version release for this mock session
- All outstanding orders will be cleared before session
- No financial obligations - no fund pay-in or pay-out from mock trades
- NOTIS application will be accessible during mock session
- Connect2NSE and Extranet facility will not be available post 6:00 PM

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Members must participate in mock trading using all trading software
- Members must re-login into live environment to ensure connectivity for Monday, January 12, 2026
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed (as per circulars NSE/ISC/51754 dated March 24, 2022 and NSE/ISC/52722 dated June 23, 2022)
- Members using NNF software must manually clear orders in their systems
- Members must use appropriate NEAT Adapter version and port combinations:
  - Old NEAT Adapter 1.0.20 (Windows/Linux): Port 10877
  - Old NEAT Adapter 1.0.22 (Windows/Linux): Port 10879
  - New NEAT Adapter 1.0.23 (Windows/Linux): Port 10879
- Software testing requirements as per circular NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234

## Important Dates

- **January 10, 2026**: Mock trading session (11:00 to 12:30 hours)
- **January 10, 2026**: Trade modification end time (12:40 hours)
- **January 10, 2026**: Live re-login window (19:00 to 19:30 hours)
- **January 12, 2026**: Regular trading resumes (Monday)

## Impact Assessment

**Market Impact**: None. This is a routine mock trading session with no financial obligations or impact on actual trading operations.

**Operational Impact**: Low. Members need to allocate resources for mock session participation and testing. The session serves as connectivity verification ahead of Monday trading.

**Member Impact**: Members must ensure their trading systems, software, and connectivity are tested during the mock session. No financial settlement obligations arise from mock trades. Connect2NSE and Extranet will be unavailable after 6:00 PM on the mock day.

**Technology Requirements**: Members must use NEAT version 3.5.4 and appropriate NEAT Adapter versions with correct port configurations as specified in the circular.
