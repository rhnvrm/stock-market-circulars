## Summary

NSE will conduct a mock trading session for the Securities Lending and Borrowing Market (SLBM) segment on Saturday, January 10, 2026. This is a routine testing session with no new version release. The mock session aims to ensure system readiness and smooth connectivity for members before regular trading resumes on Monday, January 12, 2026.

## Key Points

- Mock trading date: January 10, 2026
- Trading session: 11:15 hrs to 12:30 hrs
- Live re-login window: 19:00 hrs to 19:30 hrs
- NEAT version: 1.3.3 (no new version)
- All outstanding orders will be cleared before each session
- No financial obligations - trades will not result in fund pay-in or pay-out
- Members should participate using all trading software to ensure connectivity
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet facility unavailable from 18:00 hrs onwards

## Regulatory Changes

No regulatory changes. This is a routine mock trading session.

## Compliance Requirements

- All members enabled for SLBM trading in live market are automatically eligible
- Valid and compliant UCC/PAN must be uploaded and approved before cutoff
- Members using Non-NEAT Frontend (NNF) software must manually clear orders in their systems
- Appropriate branch and user limits must be set from Corporate Manager Terminal before placing orders
- Members should ensure telnet connectivity and Box ID setup on intended IP addresses
- NNF users must have dealer-type User IDs that are NNF-converted and IP-mapped

## Important Dates

- **January 10, 2026**: Mock trading session (11:15 - 12:30 hrs)
- **January 10, 2026**: Live re-login window (19:00 - 19:30 hrs)
- **January 12, 2026**: Regular trading resumes (Monday)

## Impact Assessment

**Market Impact**: None - this is a non-financial mock session.

**Operational Impact**: Low - routine testing exercise. Members should participate to verify system connectivity and readiness. No new software version is being released, reducing implementation risk.

**Member Action Required**: Members should test their trading systems, connectivity, and order entry processes during the mock session to ensure smooth operations when live trading resumes on January 12, 2026.
