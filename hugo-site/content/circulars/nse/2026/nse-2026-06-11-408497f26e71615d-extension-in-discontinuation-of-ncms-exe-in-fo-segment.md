## Summary

NSE Clearing Limited (NCL) has extended the decommissioning date for the NCMS EXE application in the Futures & Options segment from an earlier deadline to December 14, 2026. This circular is a continuation of prior circulars NCL/CMPT/71877 (December 18, 2025) and NCL/CMPT/73546 (March 30, 2026).

## Key Points

- The NCMS EXE decommissioning deadline has been extended to **December 14, 2026**.
- Members must continue extensive testing of the NCMS API 2.0 facility.
- This is the second extension following the original circular in December 2025 and a prior extension in March 2026.
- Members are advised to ensure a smooth transition from NCMS EXE to NCMS API 2.0.

## Regulatory Changes

No new regulatory framework changes. This circular solely extends the previously communicated deadline for discontinuation of the NCMS EXE application in the F&O segment.

## Compliance Requirements

- All F&O segment members must actively test the NCMS API 2.0 interface.
- Members should plan and complete their transition from NCMS EXE to NCMS API 2.0 before December 14, 2026.
- Members should contact NSE Clearing at msd_clearing@nsccl.co.in or 1800 266 0050 (IVR Option 2) for support.

## Important Dates

- **December 14, 2026**: Revised deadline for NCMS EXE decommissioning in the F&O segment.
- **December 18, 2025**: Original circular NCL/CMPT/71877 announcing discontinuation.
- **March 30, 2026**: First extension circular NCL/CMPT/73546.

## Impact Assessment

Impact is limited to F&O segment members who are still using the legacy NCMS EXE application. The extension provides additional runway for technical migration and reduces near-term operational risk. There is no direct impact on trading, settlement, margins, or market participants who have already migrated to NCMS API 2.0.
