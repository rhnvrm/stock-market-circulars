## Summary

NSE will conduct a mock trading session in the Currency Derivatives (CD) Segment on Saturday, April 18, 2026. No new NEAT version is being released; members should use NEAT version 3.5.4. The session runs from 11:00 to 13:00 hrs, with live re-login available from 16:00 to 16:30 hrs.

## Key Points

- Mock trading date: April 18, 2026 (Saturday)
- Normal Market open: 11:00 hrs; Normal Market close: 13:00 hrs
- Trade Modification end time: 13:10 hrs
- Live Re-login window: 16:00–16:30 hrs
- NEAT version in use: 3.5.4 (no new version release)
- NEAT 3.5.4 download path on NSE Extranet: `/cdsftp/cdscommon/NEATCDS354`
- Old NEAT Adapter (v1.0.20) uses GR PORT 10877; New NEAT Adapter (v1.0.27) uses GR PORT 10879
- Refer to circular NSE/MSD/73674 dated April 09, 2026 for NEAT Adapter Exe details

## Regulatory Changes

No regulatory changes. This is a routine mock/test trading session.

## Compliance Requirements

- Members must use NEAT version 3.5.4 for the mock session.
- Members using NNF software must manually clear outstanding orders before each session.
- Software testing must comply with circulars NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234.
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be permitted for order entry.
- Members should re-login into the live environment after the mock session to ensure smooth connectivity for Monday, April 20, 2026.

## Important Dates

- Mock trading session: April 18, 2026 (Saturday)
- Live market resumes: April 20, 2026 (Monday)
- NEAT Adapter circular reference date: April 09, 2026 (NSE/MSD/73674)

## Impact Assessment

Minimal market impact. This is a standard mock trading exercise for Currency Derivatives members to test connectivity and software ahead of regular trading. No financial obligations arise from mock trades. Members using old or new NEAT Adapter versions must connect on the correct GR ports (10877 or 10879 respectively). Participation uses existing live-environment User IDs, IPs, and Box ID mappings.
