## Summary

NSE is conducting a mock trading session in the Currency Derivatives (CD) Segment on Saturday, March 07, 2026. There is no new version release; members continue to use NEAT version 3.5.4. Additionally, the new Order Snapshot Recovery via TCP/IP for MTBT broadcast in the CD Segment will be available for testing during this mock session and will go live on March 09, 2026.

## Key Points

- Mock trading date: March 07, 2026 (Saturday)
- Segment: Currency Derivatives (CD)
- No new NEAT version release; current version is NEAT 3.5.4
- Download path on NSE Extranet: `/cdsftp/cdscommon/NEATCDS354`
- Order Snapshot Recovery via TCP/IP for MTBT broadcast (CD Segment) available for mock testing; goes live March 09, 2026
- All outstanding orders will be cleared before each session
- No financial obligation: trades during mock sessions result in no fund pay-in or pay-out
- Members should re-login to live environment after mock to ensure smooth connectivity for Monday, March 09, 2026

## Regulatory Changes

No regulatory changes. This circular references NSE/MSD/73149 dated March 05, 2026 for the introduction of Order Snapshot Recovery via TCP/IP for MTBT broadcast in the CD Segment, which is being made available for mock testing.

## Compliance Requirements

- Members must use NEAT 3.5.4 for the mock session
- NNF application users must manually clear orders in their systems before each session
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed for order entry (UCC validation skipped during contingency time)
- Members should participate using all trading software and re-login to the live environment post-mock
- Software testing requirements per circular NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 apply

## Important Dates

- **March 07, 2026**: Mock trading session (Currency Derivatives Segment)
  - Normal Market open: 10:00
  - Contingency Start: 12:00
  - Contingency Close: 13:00
  - Normal Market close: 14:00
  - Trade Modification end: 14:10
  - Live Re-login start: 16:30
  - Live Re-login close: 17:00
- **March 09, 2026**: Order Snapshot Recovery via TCP/IP for MTBT broadcast goes live

## Impact Assessment

This is a routine operational mock session with low market impact. It affects only members active in the Currency Derivatives Segment. The introduction of Order Snapshot Recovery via TCP/IP for MTBT broadcast is a technical enhancement that can be tested during this session before its March 09, 2026 live deployment. Members should ensure connectivity and software readiness ahead of the live trading week.
