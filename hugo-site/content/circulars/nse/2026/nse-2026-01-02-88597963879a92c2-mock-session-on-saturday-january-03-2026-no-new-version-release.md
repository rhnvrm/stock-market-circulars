## Summary

NSE will conduct a mock trading session for the Commodity Derivatives segment on Saturday, January 03, 2026. This session follows previous circulars regarding new encryption mechanism for interactive messages. The mock session will use existing NEAT version 1.5.3 with no new version release required. Members can participate using either old NEAT Adapter version 1.0.20 or new version 1.0.23.

## Key Points

- Mock trading session scheduled for January 03, 2026
- NEAT version 1.5.3 to be used (no new version release)
- Two trading sessions planned: Session-1 from Primary Site (11:00-14:00) and Session-2 from DR Site (14:45-15:30)
- NEAT Adapter versions supported: 1.0.20 (old) using port 10857, and 1.0.23 (new) using port 10859
- All outstanding orders will be cleared before each session
- No financial obligations from mock trading activities
- UCC/PAN validation applies (skipped during contingency time)
- NOTIS application accessible from 11:15 AM to 12:30 PM and 2:45 PM till market close
- Connect2NSE and Extranet unavailable from 12:30 PM till end of mock session

## Regulatory Changes

No regulatory changes introduced. This circular implements previously announced encryption mechanism changes from circular NSE/COM/71599 dated December 02, 2025.

## Compliance Requirements

- Members must use valid and compliant UCC/PAN uploaded and approved before cutoff time
- Members using NNF software must manually clear orders in their systems
- Members may use either Mock Trading or Simulated Environment to meet SEBI regulatory requirements (per SEBI/HO/MRD1/DSAP/CIR/P/2020/234)
- Members should refer to connectivity parameters in circulars NSE/MSD/67674 (April 24, 2025) and NSE/COM/71599 (December 2, 2025)
- Installation guide available at NSE Extranet: /comtftp/comtcommon/Installation_Procedure

## Important Dates

- **January 03, 2026**: Mock trading session date
  - Trading Session-1 (Primary Site): 11:00-14:00
    - Normal market open: 11:00
    - Contingency start: 12:00
    - Contingency end: 13:30
    - Normal market close: 14:00
  - Trading Session-2 (DR Site): 14:45-15:30
    - Normal market open: 14:45
    - Normal market close: 15:30
  - Position limit/collateral setup cutoff: 15:40
  - Trade modification end: 15:40
  - Live re-login window: 17:00-18:00

## Impact Assessment

**Market Impact**: None - This is a routine mock trading session with no financial obligations or actual market transactions.

**Operational Impact**: Low - Members need to participate in testing using existing systems. No mandatory system upgrades required as both old and new NEAT Adapter versions are supported.

**Member Impact**: Minimal - Optional participation for testing purposes. Members can use this session to verify their systems and meet SEBI testing requirements. No disruption to live trading operations as session is scheduled for Saturday.
