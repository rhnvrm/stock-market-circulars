## Summary

NSE's Member Service Department is releasing new versions of the NEAT Adapter application across Capital Market (CM), Futures & Options (F&O), Currency Derivatives (CD), Securities Lending and Borrowing Market (SLBM), and Commodities Derivatives (COM) segments. This update follows circular NSE/MSD/74511 dated June 02, 2026, which announced a new CA certificate required for encryption of interactive messages.

## Key Points

- Version 1.0.28 of NEAT Adapter released for the SLBM segment (Windows and Linux).
- Version 1.0.29 of NEAT Adapter released for CM, F&O, CD, and COM segments (Windows and Linux).
- New versions incorporate the updated CA certificate required for encryption of interactive messages.
- Setup files are available on Extranet from June 05, 2026, 20:00 hours onwards.
- Coexistence guidelines provided for members running existing and new versions simultaneously.

## Regulatory Changes

No regulatory rule changes. This circular is a technical/operational update mandating adoption of a new CA certificate via updated NEAT Adapter versions to ensure continued secure connectivity.

## Compliance Requirements

- Members using NEAT Adapter must download and install the applicable new version for each segment they operate in.
- For SLBM: download `NA_1.0.28_Setup.exe` (Windows) or `NA_1.0.28_Setup.sh` / `NA_1.0.28_SetupFiles.zip` (Linux) from `/slbftp/slbcommon/NeatAdapter/` on Extranet.
- For CM, F&O, CD, COM: download `NA_1.0.29_Setup.exe` (Windows) or `NA_1.0.29_Setup.sh` / `NA_1.0.29_SetupFiles.zip` (Linux) from the respective segment paths on Extranet.
- Members must follow the provided coexistence instructions when transitioning from existing versions (e.g., v1.0.20) to new versions.

## Important Dates

- **June 05, 2026, 20:00 hours**: New NEAT Adapter versions become available on Extranet for download and installation.
- Continuation of circular NSE/MSD/74511 dated **June 02, 2026**.

## Impact Assessment

This is a targeted operational update affecting trading members who use the NEAT Adapter software to connect to NSE's trading systems. Members must proactively download and install the new versions to maintain compatibility with the updated CA certificate. Failure to upgrade may result in connectivity issues after the old certificate is retired. The update spans five segments (CM, F&O, CD, SLBM, COM), making it broadly applicable to most active members.
