## Summary

NSE will conduct a mock trading session in the Currency Derivatives (CD) Segment on Saturday, May 16, 2026. No new NEAT version is being released; members should continue using NEAT version 3.5.4. Trades during the session carry no financial obligation.

## Key Points

- Mock trading date: May 16, 2026 (Saturday)
- Segment: Currency Derivatives
- NEAT version in use: 3.5.4 (no new version release)
- Normal Market open: 12:30 hrs; close: 15:15 hrs
- Trade Modification end time: 15:25 hrs
- Live Re-login window: 18:30–19:00 hrs
- NEAT version download path on NSE Extranet: `/cdsftp/cdscommon/NEATCDS354`
- Applicable GR PORT for New NEAT Adapter (v1.0.27, Windows & Linux): 10879
- Connect2NSE and Extranet facility unavailable from 2:00 pm till end of mock session
- Extranet API unavailable from 2:00 pm to 3:00 pm
- All outstanding orders will be purged before each session

## Regulatory Changes

No regulatory changes. Members are reminded to refer to SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 and NSE circular NSE/MSD/46441 for software testing requirements. Mock Trading or Simulated Environment may be used to meet regulatory testing obligations.

## Compliance Requirements

- Members must use NEAT version 3.5.4 with NEAT Adapter v1.0.27 (Windows or Linux) on GR PORT 10879
- Refer to NSE/MSD/73993 dated April 30, 2026 for interactive connectivity details
- Refer to NSE/MSD/73674 dated April 09, 2026 for latest NEAT Adapter Exe application details
- Only UCC/PAN records uploaded and approved before the cutoff will be permitted; UCC validation skipped during contingency order entry
- Members using NNF software must manually clear outstanding orders in their systems
- Members should participate using all trading software and re-login to the live environment to ensure smooth connectivity for Monday, May 18, 2026

## Important Dates

- **May 16, 2026**: Mock trading session (Currency Derivatives Segment)
- **May 18, 2026**: Next live trading day; members should verify connectivity post-mock

## Impact Assessment

This is a routine operational mock trading session with no financial impact. All CD segment members are automatically eligible using existing live-market credentials. The session helps members validate connectivity and software readiness ahead of the live trading week. No version upgrade is required, minimising preparation overhead. Extranet and Connect2NSE services will be briefly unavailable during the session window.
