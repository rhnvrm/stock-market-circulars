## Summary

NSE is conducting a mock trading session in the Currency Derivatives (CD) Segment on Saturday, March 14, 2026. No new NEAT version is being released; members should use NEAT version 3.5.4. The session covers both Primary Site and BCP Site trading windows, followed by a live re-login period to ensure readiness for Monday, March 16, 2026.

## Key Points

- Mock trading date: Saturday, March 14, 2026
- No new NEAT version release; use NEAT 3.5.4 (download path: `/cdsftp/cdscommon/NEATCDS354`)
- Two trading sessions: Primary Site (09:00–12:15) and BCP Site (13:00–14:00)
- Trade Modification end time: 14:10; Live Re-login window: 16:30–17:00
- All outstanding orders will be purged before each session
- Trades during mock session carry no financial obligation (no fund pay-in or pay-out)
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- NOTIS application will be accessible during the mock session
- Members should re-login to the live environment after the session for smooth Monday connectivity

## Regulatory Changes

No regulatory changes. This is a routine operational mock trading exercise. Members may use Mock Trading or Simulated Environment to meet SEBI regulatory requirements as per circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234.

## Compliance Requirements

- Members must participate using all trading software and complete re-login into the live environment post-session
- NNF application users must manually clear orders in their systems (order purge is not automatic for NNF)
- Box ID must be configured on the correct IP with appropriate segment messages
- NNF user IDs must be of dealer type, converted for NNF, and mapped to the correct IP
- Branch and user limits must be set appropriately before participating
- For UCC/PAN queries, contact uci@nse.co.in

## Important Dates

- **March 14, 2026**: Mock trading session (Currency Derivatives Segment)
  - Trading Session 1 (Primary Site): 09:00–12:15
  - Trading Session 2 (BCP Site): 13:00–14:00
  - Trade Modification end: 14:10
  - Live Re-login: 16:30–17:00
- **March 16, 2026 (Monday)**: Live market resumes; mock session re-login ensures smooth connectivity

## Impact Assessment

Minimal market impact. This is a standard operational drill for Currency Derivatives Segment members to verify connectivity, software readiness, and switchover procedures. No financial settlements occur. Members who do not participate may face connectivity issues on the subsequent live trading day (March 16, 2026). The absence of a new NEAT version simplifies preparation as no software upgrade is required.
