## Summary

NSE will conduct a mock trading session for the Electronic Gold Receipts (EGR) segment on Saturday, July 18, 2026. This mock session uses the existing NEAT+ EGR version 1.0.0, with no new software version being released for this drill. The session is conducted from the Disaster Recovery (DR) site to test system readiness.

## Key Points

- Mock trading date: July 18, 2026 (Saturday)
- Conducted from Disaster Recovery (DR) Site
- No new NEAT+ version release; existing version 1.0.0 continues to be used (download path: /egftp/egcommon/NEATPlus100)
- Default password for mock session: "Neat@EG1"
- No financial obligation - trades during mock session do not result in fund pay-in or pay-out
- Members should refer to annexures for interactive login parameters, market data broadcast parameters, installation procedures, and DR switchover guidelines

## Regulatory Changes

None. This is a routine operational circular for a scheduled mock trading drill; no regulatory or policy changes are introduced.

## Compliance Requirements

- Members must ensure trading terminals are configured with the correct interactive parameters (Gateway Router IP 172.19.14.86, Port 11887) and broadcast parameters (Multicast IPs 239.50.50.63/239.50.50.68, Ports 10863/10868) for the EGR segment
- Systems should support the existing NEAT+ EGR version 1.0.0 with prerequisite VC++ redistributable packages installed
- Members must participate in live re-login testing between 19:00 and 19:30 hrs on the mock date
- Queries to be directed to NSE Toll-Free number 1800-266-0050 (Option 1) or msm@nse.co.in

## Important Dates

- Mock trading session: Saturday, July 18, 2026
- Pre-open session: closes at 16:53 (with random closure in last minute)
- Normal market open: 17:00; Normal market close: 17:30
- Trade modification end time: 17:40
- Live re-login window: 19:00 to 19:30 hrs

## Impact Assessment

This circular has minimal market impact as it pertains to a routine, no-financial-obligation mock trading drill in the EGR segment aimed at testing DR site readiness and system stability. It affects only trading members' technical preparation and does not impact listed securities, stock prices, or investor actions.
