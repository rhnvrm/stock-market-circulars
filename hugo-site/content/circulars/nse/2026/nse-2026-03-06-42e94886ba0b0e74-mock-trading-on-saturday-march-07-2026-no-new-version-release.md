## Summary

NSE will conduct a mock trading session in the Futures & Options (F&O) segment on Saturday, March 07, 2026. This is a continuation of the earlier circular (Download No. 71999) dated December 24, 2025. No new software version is being released. Members are required to participate using NEAT+ version 7.8.7 and re-login to the live environment post-session to ensure smooth connectivity for Monday, March 09, 2026.

## Key Points

- Mock trading session scheduled for Saturday, March 07, 2026 in the Futures & Options segment
- No new version release; NEAT+ version 7.8.7 is the applicable version
- Download path on NSE Extranet: `/faoftp/faocommon/NEATPlus787`
- Order Snapshot Recovery via TCP/IP for MTBT broadcast in FO Segment will be available for testing in this mock (effective in LIVE from March 09, 2026)
- All outstanding orders will be cleared before each session
- No financial obligations arise from mock trades (no fund pay-in or pay-out)
- NOTIS application will be accessible during the mock session
- Members should use all trading software and re-login to live environment post-mock

## Regulatory Changes

- Order Snapshot Recovery via TCP/IP for MTBT broadcast in FO Segment introduced (refer to NSE/MSD/73149 dated March 05, 2026). Available for testing in this mock; goes live from March 09, 2026.

## Compliance Requirements

- All members enabled for F&O trading in live market are automatically eligible and expected to participate
- Members must use existing User IDs, IPs, and Box ID mappings from the live environment
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed (UCC validation skipped during contingency time)
- Members using NNF software must manually clear orders in their systems
- Members should refer to circulars NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 for software testing regulatory requirements

## Important Dates

- Mock Session Date: Saturday, March 07, 2026
- Pre-Open Open Time: 10:00
- Pre-Open Close Time: 10:08 (random closure in last one minute)
- Normal Market Open Time: 10:15
- Contingency Start Time: 12:00
- Contingency Close Time: 13:00
- Normal Market Close Time: 14:00
- Trade Modification End Time: 14:10
- Live Re-login Start Time: 16:30
- Live Re-login Close Time: 17:00
- Order Snapshot Recovery via TCP/IP goes LIVE: March 09, 2026

## Impact Assessment

This is a routine mock trading session with low market impact. It serves as a connectivity and software testing exercise for F&O segment members ahead of the live introduction of Order Snapshot Recovery via TCP/IP for MTBT broadcast on March 09, 2026. Members who participate will be better prepared for the new feature going live. No financial settlements or obligations result from mock trades.
