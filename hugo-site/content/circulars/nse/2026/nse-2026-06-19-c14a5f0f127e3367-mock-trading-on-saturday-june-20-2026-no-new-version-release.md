## Summary

NSE is conducting a mock trading session for the Electronic Gold Receipts (EGR) segment on Saturday, June 20, 2026. There is no new software version release accompanying this mock session; members will continue using NEAT+ EGR Version 1.0.0. The circular provides session timings, login parameters, market data broadcast parameters, and installation instructions.

## Key Points

- Mock trading session scheduled for Saturday, June 20, 2026 in the EGR segment
- No new NEAT+ version release; existing version 1.0.0 remains in use
- Download path for NEAT+ Version 1.0.0: `/egftp/egcommon/NEATPlus100`
- Default login password for mock session: `Neat@EG1`
- Trades executed during the mock session carry no financial obligation (no fund pay-in or pay-out)
- Extranet/Member portal unavailable from 11:30 AM until end of mock session; Extranet API unavailable from 11:30 AM to 1:00 PM
- Live re-login window available from 18:30 to 19:00 hrs

## Regulatory Changes

No regulatory changes. This is an operational notification for a scheduled mock trading exercise.

## Compliance Requirements

- Members must use NEAT+ EGR Version 1.0.0 (no upgrade required)
- Members should download the required EXE and master files from the NSE Extranet path `/egftp/egcommon/NEATPlus100` if not already installed
- Members should review all four annexures: (1) mock trading instructions, (2) interactive login parameters, (3) market data broadcast parameters for EGR, and (4) installation procedure for NEATPlus EGR Version 1.0.0
- For queries, members should call Toll-Free: 1800-266-0050 (Option 1) or email msm@nse.co.in

## Impact Assessment

Impact is limited to EGR segment members who are required to participate in or test connectivity for the mock session. No financial settlements arise from mock trades. The absence of a new software version release simplifies preparation, as members need only ensure the existing NEAT+ EGR Version 1.0.0 is installed. Members should plan around the Extranet and API unavailability window between 11:30 AM and end of mock session for any live production activities.
