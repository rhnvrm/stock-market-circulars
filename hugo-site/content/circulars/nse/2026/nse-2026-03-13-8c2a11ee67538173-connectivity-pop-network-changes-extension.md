## Summary

NSE has issued an extension circular regarding POP (Point of Presence) network changes, accompanied by the updated Sify handbook containing the escalation matrix for members using leased line connectivity services.

## Key Points

- Members with leased line connectivity issues (Managed Service) should follow the Sify NOC escalation matrix or refer to the Aakash Portal.
- Members on Self Service leased line can use the escalation matrix only for new line configuration and link health at POP level.
- NOC escalation matrix covers issues and queries post-commissioning of member link connectivity (Managed Services).
- Onsite Team escalation matrix covers updates for new applications before commissioning (Managed Services).
- NSE Service Desk is available at 18004199963 / 044-61332900, email: nse.servicedesk@sifycorp.com.

## Regulatory Changes

No regulatory changes. This circular is an operational update extending the timeline for POP network changes and providing refreshed contact information.

## Compliance Requirements

- Members experiencing connectivity issues should escalate through the defined Sify NOC escalation matrix in the prescribed order.
- Members requesting new connections should use the Onsite Team escalation matrix before commissioning.

## Important Dates

- Circular Date: 2026-03-13
- The extension timeline for POP network changes is referenced but specific new dates are not detailed in the available content.

## Impact Assessment

Minimal market impact. This is an infrastructure and operational circular affecting members using Sify-managed leased line connectivity to NSE's POP network. It ensures members have updated contact details for timely resolution of connectivity issues, which indirectly supports uninterrupted trading access.
