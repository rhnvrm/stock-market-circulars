## Summary

NSE will conduct a mock trading session for the Securities Lending and Borrowing Market (SLBM) segment on Saturday, January 17, 2026. This is a routine testing session with no new version release. The trading session will run from 10:15 to 13:00 hrs, followed by a live re-login window from 16:00 to 16:30 hrs. Members are encouraged to participate to ensure smooth connectivity for Monday, January 19, 2026.

## Key Points

- Mock trading date: January 17, 2026
- Trading session: 10:15 - 13:00 hrs
- Live re-login window: 16:00 - 16:30 hrs
- NEAT version: 1.3.3 (no new release)
- All outstanding orders will be cleared before each session
- No financial obligations - trades will not result in fund pay-in or pay-out
- All members enabled for SLBM trading in live market are automatically eligible
- NOTIS application will not be accessible during mock session
- Only valid and compliant UCC/PAN uploaded before cutoff will be allowed
- UCC validation skipped during contingency time for order entry

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Members using NNF software must manually clear outstanding orders in their systems
- Members must ensure appropriate branch and user limits are set from Corporate Manager Terminal before placing orders
- For NNF application users: User ID must be dealer type, converted for NNF, and mapped with IP
- Members should ensure successful telnet connectivity to Exchange host from intended IP address
- Box ID must be created on IP with appropriate messages for the segment
- Members may use either mock trading or simulated environment to meet SEBI regulatory requirements (per SEBI/HO/MRD1/DSAP/CIR/P/2020/234)

## Important Dates

- **January 16, 2026**: Circular issue date
- **January 17, 2026**: Mock trading session (10:15 - 13:00 hrs)
- **January 17, 2026**: Live re-login window (16:00 - 16:30 hrs)
- **January 19, 2026**: Regular trading resumes (Monday)

## Impact Assessment

**Market Impact**: None - this is a testing session with no financial obligations or real market transactions.

**Operational Impact**: Low - routine mock trading session allowing members to test their systems, connectivity, and software. Members should participate using all trading software and re-login to live environment to ensure smooth operations for Monday trading. Members requiring password resets, pro enablement, CTCL conversion, or other administrative actions should send email requests with User ID and segment details. Support available via toll-free number 1800 266 0050 (Option 1) or email msm@nse.co.in.
