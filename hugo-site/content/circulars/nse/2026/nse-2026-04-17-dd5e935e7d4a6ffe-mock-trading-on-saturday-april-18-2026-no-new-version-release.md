## Summary

NSE is conducting a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, April 18, 2026. There is no new version release associated with this session. Members are encouraged to participate using all trading software and re-login to the live environment to ensure smooth connectivity for the live trading week beginning Monday, April 20, 2026.

## Key Points

- Mock trading session scheduled for April 18, 2026 in the SLBM segment
- No new NEAT version release; existing NEAT version 1.3.3 applies
- Normal Market open: 11:15 hrs; Normal Market close: 13:00 hrs
- Live re-login window: 16:00 hrs to 16:30 hrs
- All outstanding orders will be purged before each session
- NNF software users must manually clear orders in their own systems
- No fund pay-in or pay-out will result from mock trades
- Only valid UCC/PAN uploaded and approved before cutoff will be permitted
- UCC validation will be skipped during contingency time for order entry
- Members should re-login to the live environment post-session for Monday readiness

## Regulatory Changes

No regulatory changes are introduced by this circular. Members are referred to existing circulars:
- NSE/MSD/67674 dated April 24, 2025 for connectivity details
- NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 for software testing requirements
- NSE/ISC/51754 dated March 24, 2022 and NSE/ISC/52722 dated June 23, 2022 for UCC/PAN validity guidelines

## Compliance Requirements

- All SLBM-enabled members are automatically eligible and should participate
- Members using NNF must ensure User IDs are of dealer type, converted for NNF, and mapped with the correct IP
- Branch and user limits must be set from the Corporate Manager Terminal before placing mock orders
- Email requests with User ID and segment details required for: Pro Enablement, CTCL Conversion, Password Reset, Unlocking of Corporate Manager User ID, and Change of Trading Rights
- Connectivity via telnet to Exchange host must be verified from the intended IP address
- Box ID must be created on the IP with appropriate messages for the SLBM segment

## Important Dates

- **April 18, 2026**: Mock trading session date
- **11:15 hrs**: Normal Market open time
- **13:00 hrs**: Normal Market close time
- **16:00 hrs**: Live re-login start time
- **16:30 hrs**: Live re-login close time
- **April 20, 2026**: Live trading resumes; members must ensure connectivity is verified

## Impact Assessment

This is a routine operational exercise with no financial or regulatory impact on market participants. The session is limited to SLBM segment members and serves as a connectivity and system readiness check ahead of the live trading week. No new software version is being released, minimizing preparation overhead. Members who do not participate may face connectivity issues on Monday, April 20, 2026.
