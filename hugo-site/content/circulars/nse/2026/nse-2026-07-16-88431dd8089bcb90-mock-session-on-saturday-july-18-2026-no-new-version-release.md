## Summary

NSE will conduct a mock trading session in the Commodity Derivatives (CD) segment on Saturday, July 18, 2026. No new software version will be released for this mock session (NEAT version remains 1.5.5). All CD segment members enabled for live trading are automatically eligible to participate using their existing User IDs, IPs, and Box ID mappings.

## Key Points

- Mock session scheduled for Saturday, July 18, 2026, in the Commodity Derivatives segment.
- No new version release; existing NEAT version 1.5.5 will be used.
- Trading session timings: DR site session 16:45; Normal market open 17:30; Normal market close 17:40; Position Limit/Collateral value set up cut-off 17:40; Trade modification end time 19:00.
- Live re-login window: 19:00 to 19:30.
- All outstanding orders will be purged before each session; NNF software users must manually clear orders in their own systems.
- Trades in the mock session carry no financial obligation — no fund pay-in or pay-out will occur.
- Only valid and compliant UCC/PAN records uploaded and approved before the cutoff will be permitted to trade; UCC validation is skipped during contingency order entry time.
- Connect2NSE and Extranet facility will be unavailable from 3:00 pm until the end of the mock session; Extranet API will be unavailable from 3 pm to 4 pm.
- Members should participate using all their trading software and complete re-login to the live environment to ensure smooth connectivity ahead of Monday, July 20, 2026.
- Members may use either the Mock Trading session or a Simulated Environment to satisfy regulatory testing requirements under NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234.

## Regulatory Changes

No new regulatory requirement is introduced; this circular reiterates existing mock trading obligations referenced under NSE/MSD/46441 and SEBI/HO/MRD1/DSAP/CIR/P/2020/234, allowing members to fulfill software/system testing requirements via mock trading or a simulated environment.

## Compliance Requirements

- Members must ensure Telnet connectivity to the Exchange host from the intended IP address and confirm Box ID setup with appropriate segment messages.
- NNF (Non-NEAT Frontend) users must ensure their User ID is dealer type, NNF-converted, and IP-mapped.
- Branch and user limits must be set via the Corporate Manager Terminal before order placement in the mock session.
- Requests for Pro Enablement, CTCL conversion, and password management must be sent via email with User ID and segment details.
- Only valid/compliant UCC/PAN entries approved before the cutoff will be permitted; queries can be directed to uci@nse.co.in.

## Important Dates

- Mock session date: Saturday, July 18, 2026.
- Connect2NSE/Extranet unavailable: from 3:00 pm on the mock date until session end (Extranet API unavailable 3 pm–4 pm).
- Live re-login window: 19:00–19:30 on July 18, 2026.
- Members must re-login to live environment ahead of resumption of normal trading on Monday, July 20, 2026.

## Impact Assessment

This is a routine operational and technical readiness exercise limited to NSE's Commodity Derivatives segment members and has no direct impact on listed securities, stock prices, or investor trading activity. The primary risk is operational — members must complete pre-requisites (connectivity, Box ID, UCC/PAN validation) and re-login correctly to avoid connectivity issues when live trading resumes on July 20, 2026.
