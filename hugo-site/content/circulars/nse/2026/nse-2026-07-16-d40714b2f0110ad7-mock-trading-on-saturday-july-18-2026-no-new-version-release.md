## Summary

NSE will conduct a mock trading session for the Securities Lending and Borrowing Market (SLBM) segment on Saturday, July 18, 2026. No new software version will be released for this mock session. The circular specifies session timings and provides detailed annexures with instructions and pre-requisites for members to participate in the mock environment ahead of live trading resuming on Monday, July 20, 2026.

## Key Points

- Mock trading session scheduled for Saturday, July 18, 2026 in the SLBM segment.
- No new NEAT version release; members will use existing NEAT version 1.3.3.
- Normal market timings for the mock session: open at 17:00 hrs, close at 17:30 hrs.
- Live re-login window: 19:00 hrs to 19:30 hrs.
- All outstanding orders will be purged before each session; NNF software users must manually clear orders.
- Mock trading (or a simulated environment) can be used to satisfy regulatory software testing requirements per NSE/MSD/46441 and SEBI/HO/MRD1/DSAP/CIR/P/2020/234.
- No financial obligation arises from mock session trades (no fund pay-in/pay-out).
- Only valid and compliant UCC/PAN records uploaded and approved before the cutoff will be permitted to participate; UCC validation is skipped during contingency order entry time.
- NOTIS application will remain accessible during the mock session.
- All members enabled for live SLBM trading are automatically eligible for mock participation using existing User IDs, IPs, and Box ID mappings.
- Members must ensure telnet connectivity, correct Box ID setup, and (for NNF users) proper dealer-type User ID conversion and IP mapping.
- Appropriate branch and user limits must be set via the Corporate Manager Terminal before placing mock orders.
- Specific email requests are required for Pro enablement, CTCL conversion, password resets/unlocking of Corporate Manager User ID, and trading rights changes.

## Regulatory Changes

None. This is an operational circular for a scheduled mock trading exercise; it does not introduce new regulations but references existing circulars (NSE/MSD/73993, NSE/MSD/46441, NSE/ISC/51754, NSE/ISC/52722) and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 governing software testing and UCC/PAN validation requirements.

## Compliance Requirements

- Members trading in the SLBM segment must participate in mock trading or use an approved simulated environment to meet SEBI/NSE software testing obligations.
- Members must ensure UCC/PAN records are valid, uploaded, and approved before the specified cutoff to participate.
- NNF software users must manually purge outstanding orders in their own systems before each mock session.
- Members must verify telnet connectivity, Box ID mapping, and branch/user limits prior to the mock session.
- Any requests for Pro enablement, CTCL conversion, or Corporate Manager User ID changes must be submitted via email with User ID and segment details.

## Important Dates

- **July 18, 2026 (Saturday)**: Mock trading session in SLBM; market open 17:00 hrs, close 17:30 hrs; live re-login window 19:00–19:30 hrs.
- **July 20, 2026 (Monday)**: Members must ensure smooth connectivity for resumption of live trading following the mock session and re-login.

## Impact Assessment

This circular has minimal market impact as it pertains solely to a technical mock trading exercise in the SLBM segment, conducted outside regular trading days with no financial settlement implications. Its primary relevance is operational, ensuring that member trading systems, connectivity, and software remain compliant and functional ahead of live trading resumption on July 20, 2026. No listed companies or securities are directly affected.
