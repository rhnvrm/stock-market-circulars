## Summary

NSE's Currency Derivatives (CD) segment will conduct a mock trading session on Saturday, July 11, 2026. No new software version will be released for this session. The circular provides the mock session schedule and detailed instructions for members to prepare their systems and ensure smooth connectivity ahead of live trading on Monday, July 13, 2026.

## Key Points

- Mock trading session scheduled for July 11, 2026 in the Currency Derivatives segment.
- No new NEAT version release; existing NEAT 3.5.4 version continues to be used.
- Normal market close time: 15:00 hrs; trade modification end time: 15:10 hrs.
- Live re-login window: 17:30 hrs to 18:00 hrs.
- All outstanding orders will be purged before the session; NNF software users must manually clear orders.
- Mock trading fulfills regulatory testing requirements under NSE circular NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234; members may alternatively use the Simulated Environment.
- No financial obligation arises from mock session trades (no fund pay-in/pay-out).
- Only valid, pre-approved UCC/PAN records will be permitted; UCC validation is skipped during contingency order entry time.
- NOTIS application will remain accessible during the mock session.
- Members should re-login to the live environment post-session to ensure connectivity readiness for Monday, July 13, 2026.
- Annexures detail eligibility, telnet connectivity checks, Box ID setup, NNF application requirements, branch/user limit configuration, and procedures for requesting Pro enablement, CTCL conversion, password resets, and trading rights changes.

## Regulatory Changes

None. This is an operational/administrative circular facilitating a scheduled mock trading session; it references existing circulars (NSE/MSD/73993, NSE/MSD/46441, NSE/ISC/51754, NSE/ISC/52722) and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 without introducing new regulatory obligations.

## Compliance Requirements

- Members enabled for CD segment live trading are automatically eligible and must participate using existing User IDs, IPs, and Box ID mappings.
- Members must verify telnet connectivity to the Exchange host and confirm Box ID setup prior to the session.
- NNF application users must ensure dealer-type User IDs are properly converted and mapped to IPs.
- Members must set appropriate branch and user limits via the Corporate Manager Terminal before order placement.
- Requests for Pro enablement, CTCL conversion, password resets, unlocking of Corporate Manager User IDs, and trading rights changes must be sent via email with required details.
- Only valid and compliant UCC/PAN records approved before the cutoff will be permitted to trade.

## Important Dates

- Mock Trading Date: Saturday, July 11, 2026
- Normal Market Close: 15:00 hrs
- Trade Modification End Time: 15:10 hrs
- Live Re-login Start: 17:30 hrs
- Live Re-login Close: 18:00 hrs
- Next Live Trading Day: Monday, July 13, 2026

## Impact Assessment

This circular has minimal market impact as it pertains solely to a routine testing exercise in the Currency Derivatives segment. Trades executed during the mock session carry no financial obligation and do not affect live market pricing, listed securities, or investor positions. The primary operational impact is on trading members and their technology teams, who must ensure system readiness, complete required re-logins, and validate connectivity to avoid disruptions when live trading resumes on July 13, 2026.
