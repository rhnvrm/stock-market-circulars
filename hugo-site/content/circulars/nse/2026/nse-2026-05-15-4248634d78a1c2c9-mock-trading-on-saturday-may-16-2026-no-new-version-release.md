## Summary

NSE is conducting a mock trading session in the Capital Market segment on Saturday, May 16, 2026. There is no new version release for this session. The circular provides session timings, software version details, and important operational instructions for members.

## Key Points

- Mock trading session scheduled for Saturday, May 16, 2026 in the Capital Market segment
- No new NEAT+ version release for this mock session
- Current valid versions: NEAT+ 7.8.7 (active) and 7.8.8 (discontinuation date: June 06, 2026; download path: /common/NEATPlus788)
- NEAT Adapter version 1.0.27 (Windows and Linux) should be used with GR PORT 10823
- Members should refer to circular NSE/MSD/73674 dated April 09, 2026 for NEAT Adapter details
- All outstanding orders will be cleared before each session
- Trades during mock sessions carry no financial obligation (no fund pay-in or pay-out)

## Regulatory Changes

No regulatory changes. This is a routine mock trading session circular. Members may use Mock Trading or Simulated Environment to meet regulatory requirements per circular NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234.

## Compliance Requirements

- Members using NNF software must manually clear outstanding orders in their systems before each session
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be permitted; UCC validation will be skipped during contingency time for order entry
- Members should use NEAT Adapter version 1.0.27 with GR PORT 10823 for Capital Market segment
- Refer to circular NSE/MSD/73993 dated April 30, 2026 for interactive connectivity details
- Installation guide available on NSE Extranet at /common/InstallationProcedure
- Connect2NSE and Extranet facility will not be available from 2:00 PM till end of mock session
- Extranet API will not be available from 2:00 PM to 3:00 PM
- For UCC/PAN queries, contact uci@nse.co.in

## Important Dates

- **May 16, 2026**: Mock trading session in Capital Market segment
- **June 06, 2026**: Discontinuation date for NEAT+ version 7.8.8
- **April 09, 2026**: Reference circular NSE/MSD/73674 for NEAT Adapter details
- **April 30, 2026**: Reference circular NSE/MSD/73993 for connectivity details

## Impact Assessment

This is a routine operational circular affecting only NSE member firms that participate in mock trading sessions. There is no market-wide impact, no financial obligation from mock trades, and no new software version to deploy. Members need to ensure their NEAT+ and NEAT Adapter software is up to date and connectivity is verified before the session. NOTIS application will be accessible during the mock session for member reference.
