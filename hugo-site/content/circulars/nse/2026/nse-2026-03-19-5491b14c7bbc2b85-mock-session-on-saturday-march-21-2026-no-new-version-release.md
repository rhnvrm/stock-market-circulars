## Summary

NSE will conduct a mock trading session in the Commodity Derivatives (CD) segment on Saturday, March 21, 2026. There is no new NEAT version release associated with this mock session. Members are encouraged to participate to ensure smooth connectivity ahead of live trading on Monday, March 23, 2026.

## Key Points

- Mock trading session scheduled for Saturday, March 21, 2026 in the Commodity Derivatives segment
- No new NEAT version release with this mock session
- NEAT 1.5.5 is the recommended version; NEAT 1.5.4 will be discontinued on April 11, 2026 (mock)
- NEAT 1.5.5 download path on NSE Extranet: `/comtftp/comtcommon/NEATCO155`
- Trades during mock sessions carry no financial obligations (no fund pay-in or pay-out)
- NOTIS application will not be accessible during the mock session
- All outstanding orders will be cleared before each session
- Members using NNF software must manually clear orders in their systems

## Regulatory Changes

No regulatory changes. This is a routine operational mock trading circular.

## Compliance Requirements

- Members must ensure UCC/PAN is valid, uploaded, and approved before the cutoff to be eligible for order entry
- UCC validation will be skipped during contingency time for order entry
- NNF application users must ensure: dealer-type User ID, NNF conversion, and IP mapping
- Members should set appropriate branch and user limits from the Corporate Manager Terminal before placing orders
- Members should refer to circular NSE/MSD/67674 (April 24, 2025) and NSE/COM/71599 (December 2, 2025) for connectivity details
- Software testing requirements per circular NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234 can be met via this mock session

## Important Dates

- **March 21, 2026**: Mock trading session date
- **Normal Market Open**: 11:00
- **Normal Market Close**: 14:00
- **Position Limit/Collateral Value Setup Cutoff**: 14:10
- **Trade Modification End Time**: 14:10
- **Live Re-login Start Time**: 16:30
- **Live Re-login Close Time**: 17:00
- **April 11, 2026**: NEAT 1.5.4 discontinuation date (mock)

## Impact Assessment

Minimal market impact. This is a routine mock session limited to the Commodity Derivatives segment. Members who participate will be better prepared for live trading on March 23, 2026. Members still on NEAT 1.5.4 should plan to upgrade to NEAT 1.5.5 before April 11, 2026. No financial settlement or regulatory filing obligations arise from this session.
