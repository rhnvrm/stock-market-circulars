## Summary

NSE will conduct a mock trading session in the Commodity Derivatives (CD) segment on Saturday, July 25, 2026. No new software version will be released for this mock session (NEAT Version remains 1.5.5). The circular specifies session timings, re-login windows, and references annexures with pre-requisites and guidelines for participation, including instructions for switchover to the Disaster Recovery (DR) site.

## Key Points

- Mock trading session scheduled for Saturday, July 25, 2026, in the Commodity Derivatives segment.
- No new version release; NEAT Version remains 1.5.5.
- Trading Session-1 (Primary Site): Normal Market open/close from 13:15 to 15:30 hrs.
- Trading Session-2 (DR Site): Normal Market open/close from 16:15 to 16:30 hrs.
- Position Limit/Collateral value set up cut-off time and Trade Modification end time: 16:40 hrs.
- Live Re-login window: 19:30 hrs to 20:00 hrs.
- All outstanding orders will be purged before each session; NNF software users must manually clear orders.
- Mock trades result in no financial obligation (no fund pay-in or pay-out).
- Only valid and compliant UCC/PAN records uploaded and approved before cutoff will be permitted; UCC validation is skipped during contingency time.
- Members are advised to participate using all trading software and re-login to the live environment to ensure smooth connectivity for Monday, July 27, 2026.

## Regulatory Changes

None. This is an operational/technical notice; no new regulations are introduced. Reference is made to existing circulars including NSE/MSD/73993 (connectivity), NSE/MSD/46441 and SEBI/HO/MRD1/DSAP/CIR/P/2020/234 (software testing/simulated environment requirements), and NSE/ISC/51754 and NSE/ISC/52722 (UCC/PAN validation).

## Compliance Requirements

- Members must ensure Box ID setup and telnet connectivity from the intended IP address prior to the mock session.
- NNF application users must ensure User IDs are of dealer type, converted for NNF, and mapped with the correct IP.
- Appropriate branch and user limits must be set via the Corporate Manager Terminal before placing mock orders.
- Members must ensure UCC/PAN records are valid and approved before the cutoff time.
- Members must complete live re-login within the specified window (19:30–20:00 hrs) to confirm connectivity ahead of live trading on Monday, July 27, 2026.

## Important Dates

- Mock Session Date: Saturday, July 25, 2026
- Trading Session-1 (Primary Site): 13:15–15:30 hrs
- Trading Session-2 (DR Site): 16:15–16:30 hrs
- Position Limit/Collateral cut-off & Trade Modification end: 16:40 hrs
- Live Re-login window: 19:30–20:00 hrs
- Next live trading day: Monday, July 27, 2026

## Impact Assessment

This is a routine, low-impact operational notice for Commodity Derivatives segment members. It has no direct effect on listed securities, market pricing, or investor obligations, as mock trades carry no financial settlement. The primary impact is operational: members must verify system readiness, connectivity, and UCC/PAN compliance ahead of the mock session, and complete re-login to avoid connectivity issues at the start of live trading on the next business day.
