## Summary

NSE Clearing Ltd (NCL) has released a user manual (Version 1.0, May 2026) for the newly introduced **Extranet module** within the NMASS (NSE Member Access and Support System) portal. This module provides clearing members with structured access to NCL's extranet services including API registration, file uploads/downloads, and a dedicated dashboard.

## Key Points

- NCL has launched an Extranet module within the NMASS member portal
- The module is documented in a formal User Manual (V1.0, released May 2026)
- Role and access management is member-type specific — different menus are available based on member classification
- Key functional areas include: API Registration, Upload, Download, and a multi-section Dashboard
- The NCL Dashboard includes an "Extranet Downloads" sub-section for accessing relevant files
- Document is classified as Non-Confidential

## Regulatory Changes

No direct regulatory changes. This circular introduces an operational enhancement to the existing NMASS portal infrastructure, expanding the digital service interface available to NCL clearing members.

## Compliance Requirements

- Clearing members should review the User Manual to understand the new Extranet module functionality
- Members must manage roles and access permissions for their users within the portal according to their member type
- API registration through the new module may be required for members intending to use programmatic access to NCL extranet services

## Important Dates

- **Release Date:** May 2026 (exact date of circular: 2026-05-20)
- **Version:** 1.0 — initial release of the Extranet module user manual

## Impact Assessment

This circular has **medium operational impact** for NSE clearing members. The introduction of the NCL Extranet module in NMASS modernises how members interact with clearing-related data and services. Members will need to familiarise themselves with the new portal sections — particularly API Registration and the Dashboard — to take full advantage of the extranet capabilities. No trading activity or compliance filings are directly affected, but members relying on extranet data feeds or downloads should transition to using the new NMASS interface.
