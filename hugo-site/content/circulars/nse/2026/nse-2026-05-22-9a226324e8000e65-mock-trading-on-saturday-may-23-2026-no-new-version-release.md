## Summary

NSE is conducting a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, May 23, 2026. The session covers trading from both the Primary Site and the Disaster Recovery (DR) Site. No new version of the NEAT software will be released; members should use NEAT version 1.3.3.

## Key Points

- Mock trading scheduled for Saturday, May 23, 2026 in the SLBM segment
- No new NEAT version release; current version is NEAT 1.3.3
- Two trading sessions: Session 1 from Primary Site (09:15–10:05) and Session 2 from DR Site (11:00–11:30)
- Live re-login window: 13:00–13:30
- All outstanding orders will be purged before each session begins
- Trades during mock sessions carry no financial obligation (no fund pay-in or pay-out)
- Members using Non-NEAT Frontend (NNF) software must manually clear orders in their systems
- Only UCC/PAN uploaded and approved before the cutoff will be permitted; UCC validation is skipped during contingency time
- NOTIS application will be accessible during the mock session
- Live session updates available at www.nseindia.com

## Regulatory Changes

No regulatory changes. This is a routine operational mock trading circular.

## Compliance Requirements

- Members eligible for SLBM live trading are automatically eligible for mock participation
- Members must use existing live-environment User IDs, IPs, and Box ID mappings
- Confirm telnet connectivity from intended IP addresses before the session
- Verify Box ID is created on the IP with appropriate segment messages
- NNF users must ensure their User ID is of dealer type, converted for NNF, and mapped with the IP
- Set appropriate branch and user limits from the Corporate Manager Terminal before placing orders
- Members should re-login into the live environment after the mock session to ensure smooth connectivity for Monday, May 26, 2026
- For Pro Enablement or CTCL Conversion requests, send an email with User ID and segment details to msm@nse.co.in
- Refer to circular NSE/MSD/73993 dated April 30, 2026 for interactive connectivity details
- Refer to circulars NSE/MSD/46441 and SEBI/HO/MRD1/DSAP/CIR/P/2020/234 for software testing regulatory requirements

## Important Dates

- **May 23, 2026**: Mock trading session in SLBM
  - Session 1 (Primary Site): 09:15 – 10:05
  - Session 2 (DR Site): 11:00 – 11:30
  - Live Re-login: 13:00 – 13:30
- **May 26, 2026 (Monday)**: Live market resumes; members should ensure connectivity via mock re-login

## Impact Assessment

This circular has minimal market impact. It is a standard operational exercise for SLBM segment members to verify system readiness, connectivity, and disaster recovery capabilities. No financial transactions will be settled from mock trades. Members are encouraged to participate to ensure seamless trading on the following Monday.
