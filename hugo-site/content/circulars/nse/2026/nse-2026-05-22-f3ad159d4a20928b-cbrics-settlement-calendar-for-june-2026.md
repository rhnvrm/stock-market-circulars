## Summary

NSE Clearing Ltd. has issued the CBRICS (Corporate Bond) settlement calendar for June 2026. Participants and Custodians are directed to transfer securities for CBRICS settlement using Settlement mode 'On-market' as per the published schedule.

## Key Points

- Market Type: Corporate Bond
- Clearing Member: NSE Clearing Ltd. (CM-BP ID: IN568177)
- Settlement transfers to be made via NSDL/CDSL using 'On-market' settlement mode
- 21 settlement dates scheduled across June 2026
- Settlement numbers follow the format 2606XXX (e.g., 2606001 for 01-Jun-26)
- Previously used DP details (IN001002-10000617) have been deprecated since September 3, 2019 and must not be used

## Regulatory Changes

No new regulatory changes introduced. This is a routine monthly operational circular.

## Compliance Requirements

- Participants and Custodians must use Settlement mode 'On-market' for all CBRICS security transfers
- Transfers must align with the published settlement dates and numbers
- Deprecated DP details (IN001002-10000617) must not be used for any CBRICS settlement
