## Summary

NSE Clearing Ltd (NCL) has introduced an Extranet module within the NMASS (NSE Member Access System) portal. This User Manual (Version 1.0, released May 2026) guides members through the new Extranet service, covering login, role management, API registration, file upload/download, and a dedicated dashboard.

## Key Points

- NCL has launched a new "Extranet" module in the NMASS Member Portal.
- The module supports API registration for programmatic access.
- Members can upload and download files through the Extranet interface.
- A dedicated NCL Dashboard with an Extranet Downloads section is available.
- Role and Access Management controls are in place, with access levels varying by member type.

## Regulatory Changes

No regulatory changes. This is a purely operational/technical release introducing a new service module for NMASS portal users.

## Compliance Requirements

- Members intending to use the NCL Extranet service must register via the API Registration section in NMASS.
- Role assignments within the portal must be configured by the member's administrator per the Role Management guidelines in the user manual.
- Access permissions are determined by member type as defined by NCL.

## Important Dates

- User Manual Version 1.0 released: May 2026.
- No specific deadline or go-live date explicitly stated in the available content.

## Impact Assessment

This circular has a low market impact. It is an operational enhancement for NCL member firms who use the NMASS portal. Members gain new extranet connectivity options, including API-based integration and structured file exchange. No changes to trading rules, margins, or settlement procedures are involved.
