## Summary

NSE will conduct a mock trading session in the Electronic Gold Receipts (EGR) segment on Saturday, July 11, 2026. This is a non-live, test session and does not involve a new NEAT+ version release. Members are required to follow the specified session schedule, login parameters, and market data broadcast settings for the test.

## Key Points

- Mock trading session scheduled for Saturday, July 11, 2026, in the EGR segment.
- No new version release accompanies this mock session; existing NEAT+ EGR Version 1.0.0 remains applicable.
- Session timings include pre-open, normal market, block deal window, and auction market windows, plus a live re-login window in the evening.
- Default password for the session is "Neat@EG1".
- Trades executed during the mock session do not result in any fund pay-in or pay-out.
- Members should refer to Annexures 1-5 in the circular for detailed instructions on login parameters, market data broadcast, installation procedures, and DR site switchover guidelines.
- Updates related to the session will be posted on www.nseindia.com; support is available via toll-free number 1800-266-0050 (Option 1) or email msm@nse.co.in.

## Regulatory Changes

None. This is an operational/testing notice and does not introduce new regulations.

## Compliance Requirements

- Members must ensure their systems are configured with the correct NEAT+ EGR Version 1.0.0 and use the specified default password for login.
- Members should verify gateway router IP addresses, subnet ranges, and multicast broadcast parameters as per Annexures 2 and 3 prior to the session.
- Members should complete the live re-login process within the specified window (17:30-18:00 hrs) after the mock session.

## Important Dates

- **Mock Trading Date:** Saturday, July 11, 2026
- Pre-open session: 12:30-12:38 hrs (with random closure in the last minute)
- Normal market open: 12:45 hrs
- Auction market: 13:30-14:15 hrs
- Block deal window session-1: 14:00-14:15 hrs
- Normal market close: 15:00 hrs
- Trade modification end time: 15:10 hrs
- Live re-login window: 17:30-18:00 hrs

## Impact Assessment

This circular has minimal market impact as it pertains to a routine, non-live mock trading session in the EGR segment intended for system testing by exchange members. There is no financial obligation arising from trades executed during this session, and no changes to trading rules, listed securities, or investor-facing processes result from this notice. Relevance is limited to member firms' technical and operations teams responsible for trading system readiness.
