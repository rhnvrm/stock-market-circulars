## Summary

NSE has announced a change in ISIN code for Mos Utility Limited (MOS) equity shares effective August 08, 2025. The new ISIN INE0N7S01028 will replace the existing code due to sub-division of shares from Rs. 10/- to Rs. 2/-.

## Key Points

- Company: Mos Utility Limited (Symbol: MOS)
- New ISIN: INE0N7S01028
- Reason: Sub-division from Rs. 10/- to Rs. 2/-
- Effective date: August 08, 2025 (Ex-date)
- All trades from ex-date will use new ISIN

## Regulatory Changes

ISIN code change from existing code to INE0N7S01028 for MOS equity shares following corporate action of share sub-division.

## Compliance Requirements

- Exchange members must use new ISIN code for all trades from August 08, 2025
- System updates required to reflect new ISIN in trading platforms
- Proper identification of securities using updated ISIN code

## Important Dates

- **August 06, 2025**: Circular issued
- **August 08, 2025**: Ex-date - New ISIN becomes effective for all trades

## Impact Assessment

Routine administrative change with minimal market impact. Trading systems and participants need to update ISIN references. Share sub-division increases liquidity by reducing nominal share price from Rs. 10 to Rs. 2, making shares more accessible to retail investors.
