## Summary

NSE has enhanced the password policy for the NSE MF Invest platform, effective from January 3, 2026. All members accessing the platform via portal (including Admin and Sub-broker users) are mandatorily required to reset their passwords within 30 days from the effective date. Users will receive password expiry prompts upon login, and failure to reset within the 30-day period will result in automatic redirection to the password reset page with no further access until password is successfully reset.

## Key Points

- Password policy enhancement on NSE MF Invest platform effective January 3, 2026
- Login prompts will display password expiry period with option to reset before expiry
- Mandatory password reset required within 30 days for all portal users
- Applies to both Admin and Sub-broker users
- Automatic redirection to password reset page after 30-day period
- No platform access permitted until password is successfully reset
- Password expiry count commences from January 3, 2026
- Password guidelines available on change password screen

## Regulatory Changes

NSE has implemented an enhanced password security policy requiring periodic password resets. This represents a strengthening of authentication and access control measures for the mutual fund investment platform to improve security standards.

## Compliance Requirements

- **Mandatory Action**: All members must reset their NSE MF Invest platform passwords within 30 days from January 3, 2026
- **Applicable Users**: Admin users and Sub-broker users accessing via portal
- **Non-Compliance Consequence**: System will automatically redirect users to password reset page upon login; access denied until password reset is completed
- **Guidelines**: Members must follow password guidelines displayed on the change password screen
- **Support Contact**: Toll-free: 1800 2100 940, Email: support@nseinvest.com

## Important Dates

- **Effective Date**: January 3, 2026 - Enhanced password policy becomes effective
- **Password Expiry Count Start**: January 3, 2026 - 30-day countdown begins
- **Compliance Deadline**: February 2, 2026 - Last date to reset password (30 days from effective date)
- **Circular Date**: December 31, 2025

## Impact Assessment

**Operational Impact**: Medium - All NSE MF Invest platform users must take action within the specified timeframe. Failure to comply will result in immediate access restrictions, potentially disrupting mutual fund transaction processing capabilities.

**User Groups Affected**: Trading members, Admin users, and Sub-broker users who access NSE MF Invest platform via portal interface.

**Business Continuity**: Members should proactively reset passwords before the 30-day deadline to avoid service interruption. Organizations should communicate this requirement to all relevant users to ensure uninterrupted platform access.

**Security Enhancement**: This policy strengthens platform security through regular password rotation, reducing risks associated with credential compromise and unauthorized access to mutual fund investment operations.
