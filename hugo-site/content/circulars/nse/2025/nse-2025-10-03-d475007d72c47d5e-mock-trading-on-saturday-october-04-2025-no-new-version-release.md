## Summary

NSE will conduct a mock trading session for the Securities Lending and Borrowing Market (SLBM) on Saturday, October 04, 2025. The session will test trading from both Primary and DR Sites with no new version release. The software version remains NEAT 1.3.3. Members are encouraged to participate using all trading software and re-login to ensure smooth connectivity for Monday, October 06, 2025.

## Key Points

- Mock trading session scheduled for Saturday, October 04, 2025
- No new software version release - continues with NEAT 1.3.3
- Two trading sessions: Primary Site (11:15-14:00) and DR Site (15:00-16:00)
- Live re-login window: 18:00-18:30
- Contingency testing period: 12:00-13:30
- All outstanding orders will be cleared before each session
- No financial obligations from trades executed during mock session
- Connect2NSE and Extranet unavailable from 11:00 AM to 02:00 PM
- Extranet API unavailable from 11:00 AM to 11:30 AM
- NOTIS application will not be accessible during mock session
- UCC/PAN validation will be skipped during contingency time for order entry

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Members must ensure telnet connectivity to Exchange host from intended IP addresses
- Box ID must be created on IP with appropriate segment messages
- For Non-NEAT Frontend (NNF) users: User ID must be dealer type, converted for NNF, and mapped with IP
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed (as per circulars NSE/ISC/51754 and NSE/ISC/52722)
- Members should set appropriate branch and user limits
- Members using NNF software must manually clear orders in their systems
- All members enabled for SLBM trading in live market are automatically eligible for mock trading
- Members should participate using all trading software to ensure smooth connectivity for Monday, October 06, 2025

## Important Dates

- **October 03, 2025**: Circular date
- **October 04, 2025**: Mock trading session date
  - Trading Session-1 (Primary Site): 11:15-14:00
    - Normal Market open: 11:15
    - Contingency start: 12:00
    - Contingency end: 13:30
    - Normal Market close: 14:00
  - Trading Session-2 (DR Site): 15:00-16:00
  - Live Re-login: 18:00-18:30
- **October 06, 2025**: Regular trading resumes (Monday)

## Impact Assessment

**Market Impact**: None - this is a testing session with no financial obligations or impact on live trading.

**Operational Impact**: Low - routine mock trading session to test systems and connectivity. Members should allocate resources for participation to ensure smooth operations on Monday. The session includes disaster recovery site testing, which is essential for business continuity preparedness.

**Member Impact**: Members need to participate using their trading software and complete re-login procedures to ensure readiness for regular trading on Monday, October 06, 2025. Temporary unavailability of Connect2NSE, Extranet, and NOTIS during specified hours.
