## Summary

NSE will conduct mock trading sessions in the Commodity Derivatives Segment on Saturday, September 6, 2025. The session includes primary site trading (11:00-14:15) and BCP site trading (15:00-15:45), with live re-login period (18:00-18:30). No new software version will be released.

## Key Points

- Mock trading scheduled for September 6, 2025 in Commodity Derivatives Segment
- Two trading sessions: Primary site (11:00-14:15) and BCP site (15:00-15:45)
- Live re-login window from 18:00-18:30
- No financial obligations from mock trades
- All outstanding orders cleared before each session
- NEAT 1.5.1 software version to be used

## Trading Schedule

**Session 1 - Primary Site:**
- Normal Market Open: 11:00
- Contingency Start: 12:00
- Contingency End: 13:30
- Normal Market Close: 14:15

**Session 2 - BCP Site:**
- Normal Market Open: 15:00
- Normal Market Close: 15:45
- Position Limit/Collateral Setup Cutoff: 15:55
- Trade Modification End: 15:55

**Live Re-login:**
- Start Time: 18:00
- End Time: 18:30

## Compliance Requirements

- All commodity derivatives members automatically eligible
- Must use existing User IDs, IPs, and Box ID mappings from live environment
- Valid and compliant UCC/PAN required (uploaded and approved before cutoff)
- UCC validation skipped during contingency time for order entry
- Members should participate using all trading software and re-login to live environment

## System Specifications

- Software Version: NEAT 1.5.1
- NOTIS application accessible during session-2
- Connect2NSE and Extranet unavailable 04:00-17:00
- Extranet API unavailable 14:00-15:00
- Telnet connectivity must be confirmed from intended IP addresses
