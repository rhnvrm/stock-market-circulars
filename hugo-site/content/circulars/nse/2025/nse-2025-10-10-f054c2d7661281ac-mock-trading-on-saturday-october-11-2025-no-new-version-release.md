## Summary

NSE will conduct a mock trading session in the Futures & Options segment on Saturday, October 11, 2025. No new software version will be released. Members will use existing NEAT+ versions 7.8.3 and 7.8.4. The session includes mock trading from primary site (11:15-13:30), DR site (14:15-14:55), and live re-login testing (16:00-16:30). Version 7.8.3 will be discontinued on November 01, 2025 during mock sessions.

## Key Points

- Mock trading session scheduled for Saturday, October 11, 2025 in F&O segment
- No new NEAT+ version release; existing versions 7.8.3 and 7.8.4 to be used
- Primary site mock trading: 11:15 to 13:30 hours
- DR site mock trading: 14:15 to 14:45 hours (trade modification ends at 14:55)
- Live re-login session: 16:00 to 16:30 hours
- NEAT+ version 7.8.3 will be discontinued from November 01, 2025 (mock)
- NEAT+ version 7.8.4 is the recommended version
- All outstanding orders will be cleared before each session
- NOTIS application will not be accessible during mock session
- Only valid and compliant UCC/PAN uploaded before cutoff will be allowed

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Members must ensure valid and compliant UCC/PAN are uploaded and approved before cutoff
- Members using NNF software must manually clear orders in their systems
- Members should participate using all trading software and re-login into live environment
- Ensure successful telnet connectivity to Exchange host from intended IP address
- Confirm Box ID is created on the IP with appropriate messages
- Members may use either mock trading or simulated environment to meet regulatory requirements per SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234

## Important Dates

- **October 11, 2025**: Mock trading session date
  - 11:15-13:30: Primary site mock trading
  - 14:15-14:45: DR site mock trading
  - 14:55: Trade modification end time
  - 16:00-16:30: Live re-login session
- **October 13, 2025**: Expected normal trading day (Monday)
- **November 01, 2025**: NEAT+ version 7.8.3 discontinuation date (mock)

## Impact Assessment

**Operational Impact**: Low - This is a routine mock trading session with no new version release. Members continue using existing NEAT+ versions 7.8.3 and 7.8.4.

**Market Impact**: None - Mock trading sessions do not result in any financial obligations, fund pay-in, or pay-out.

**Member Action Required**: Members should participate to ensure smooth connectivity for Monday, October 13, 2025 trading. Members still using version 7.8.3 should plan migration to version 7.8.4 before November 01, 2025.

**Technical Testing**: The session includes disaster recovery site switchover testing, allowing members to verify their systems' capability to handle DR scenarios.
