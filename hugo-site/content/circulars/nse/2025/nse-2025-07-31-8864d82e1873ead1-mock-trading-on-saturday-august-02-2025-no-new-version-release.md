## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) on Saturday, August 02, 2025. The session includes testing of both primary site and disaster recovery (DR) site operations with no new software version release.

## Key Points

- Mock trading date: August 02, 2025
- Two trading sessions: Primary site (11:15-14:00) and DR site (15:00-15:30)
- Live re-login window: 17:30-18:00
- Software version: NEAT 1.3.3 (no new version)
- All outstanding orders will be cleared before each session
- No financial obligations from mock trades

## Trading Schedule

**Session 1 - Primary Site:**
- Normal Market open: 11:15
- Contingency start: 12:00
- Contingency end: 13:30
- Normal Market close: 14:00

**Session 2 - DR Site:**
- Normal Market open: 15:00
- Normal Market close: 15:30

**Re-login Timings:**
- Live re-login: 17:30-18:00

## Compliance Requirements

- Only valid and compliant UCC/PAN uploaded before cutoff allowed
- UCC validation skipped during contingency time
- Members must set appropriate branch and user limits
- NNF users require dealer-type User ID converted for NNF and mapped with IP
- Members should participate using all trading software

## Important Guidelines

- All members enabled for SLBM trading automatically eligible
- Use existing User IDs, IPs, and Box ID mappings
- NOTIS application not accessible during mock session
- Members using NNF software must manually clear orders
- Connectivity parameters available in circular NSE/MSD/67674

## Impact Assessment

Routine system testing to ensure operational readiness for regular trading on Monday, August 04, 2025. Critical for disaster recovery preparedness and system stability verification.
