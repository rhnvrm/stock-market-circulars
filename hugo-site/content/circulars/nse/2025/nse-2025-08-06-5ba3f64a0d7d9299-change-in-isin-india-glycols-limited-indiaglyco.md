## Summary

NSE has announced a change in ISIN code for India Glycols Limited (INDIAGLYCO) effective August 12, 2025, due to subdivision of equity shares from Rs. 10/- to Rs. 5/-.

## Key Points

- Company: India Glycols Limited (Symbol: INDIAGLYCO)
- New ISIN: INE560A01023
- Reason: Sub-division of shares from Rs. 10/- to Rs. 5/-
- Effective date: August 12, 2025 (Ex-date)

## Regulatory Changes

ISIN code change from existing code to INE560A01023 for all equity shares of India Glycols Limited.

## Compliance Requirements

- All NSE members must note the new ISIN code
- New ISIN applies to all trades from ex-date onwards
- Systems and records must be updated accordingly

## Important Dates

- **August 12, 2025**: Ex-date for new ISIN implementation
- All trades on and from this date will use the new ISIN

## Impact Assessment

Routine administrative change affecting trading systems and record-keeping. No impact on share value or trading operations, only identification code changes due to share subdivision.
