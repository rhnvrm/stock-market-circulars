## Summary

NSE has issued an update regarding the web file structure specifications for the NSE Mutual Funds Service System. The circular provides comprehensive technical documentation for uploading client information, including client master data in 183-column format, FATCA/CRS information, bank details, and various standardized codes for tax status, occupation, country codes, and other compliance-related data fields.

## Key Points

- Updated web file structure for NSE Mutual Funds Service System
- Client master common format with 183 columns specified
- Comprehensive FATCA/CRS upload requirements and field definitions
- Standardized code lists provided for multiple data categories
- Includes user master management and rights management specifications
- Client bank upload and FATCA upload formats detailed
- Various reference codes provided: country codes, wealth source codes, tax status, occupation codes, identification types, state codes, exemption codes, GIIN exempt categories, and Active NFE sub-categories

## Regulatory Changes

This circular updates the technical specifications and data format requirements for mutual fund participants uploading client information to NSE's system. It standardizes the data fields and codes required for FATCA (Foreign Account Tax Compliance Act) and CRS (Common Reporting Standard) compliance reporting.

## Compliance Requirements

- Mutual fund distributors and participants must use the updated 183-column client master format
- All FATCA/CRS information must be uploaded using the specified field formats and standardized codes
- Client bank details must be uploaded as per the prescribed format
- User master and rights management must be maintained according to specifications
- Participants must use only the approved country codes, tax status codes, occupation codes, and other reference codes provided in the documentation
- Client tax status, holding patterns, and nominee/guardian relationships must be recorded using standardized codes

## Important Dates

No specific implementation or compliance dates are mentioned in the available content. This appears to be a technical reference document for ongoing operations.

## Impact Assessment

This update impacts all mutual fund distributors, AMCs, and intermediaries using the NSE Mutual Funds Service System. The standardized format ensures consistency in client data reporting and facilitates compliance with international tax reporting obligations under FATCA and CRS. Participants will need to ensure their systems and processes align with the updated specifications to successfully upload and maintain client information. The comprehensive code lists help streamline data entry and reduce errors in tax compliance reporting.
