## Summary

NSE will conduct a mock trading session in the Currency Derivatives segment on Saturday, August 30, 2025, from 09:00 to 10:10 hours, with trade modification ending at 10:20. Live re-login will be available from 13:30 to 14:00 hours. No new software version will be released.

## Key Points

- Mock trading session on Saturday, August 30, 2025
- Trading hours: 09:00 to 10:10, trade modification until 10:20
- Live re-login: 13:30 to 14:00
- Using existing NEAT 3.5.3 software version
- All outstanding orders will be cleared before session
- No financial obligations from mock trades
- NOTIS application will not be accessible during session

## Regulatory Changes

No regulatory changes introduced. This is a routine mock trading session.

## Compliance Requirements

- Only valid and compliant UCC/PAN approved before cutoff allowed
- Members using NNF software must manually clear orders
- All CD segment enabled members automatically eligible
- Appropriate branch and user limits must be set before trading
- Pro Enablement and CTCL Conversion requests via email if needed

## Important Dates

- **August 30, 2025**: Mock trading session
- **09:00-10:10**: Trading session
- **10:20**: Trade modification end time
- **13:30-14:00**: Live re-login window
- **September 1, 2025**: Next trading day requiring smooth connectivity

## Impact Assessment

Minimal market impact as this is a routine testing session with no financial obligations. Members should participate to ensure smooth operations for Monday trading. Testing helps maintain system reliability and member preparedness.
