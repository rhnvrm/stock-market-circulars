## Summary

NSE will conduct a mock trading session in the Commodity Derivatives Segment on Saturday, October 11, 2025. This is a routine testing session with no new version release. Members are required to participate using all trading software and re-login into the live environment to ensure smooth connectivity for trading on Monday, October 13, 2025.

## Key Points

- Mock trading session scheduled for October 11, 2025 (Saturday)
- No new NEAT version release - existing versions 1.5.1 and 1.5.2 will be used
- Trading Session-1 from Primary Site: 11:00 to 13:30 hours
- Trading Session-2 from BCP site: 14:00 to 14:30 hours
- Live Re-login window: 16:00 to 16:30 hours
- NEAT version 1.5.1 will be discontinued on November 01, 2025 (mock)
- Version 1.5.2 is the recommended version
- All outstanding orders will be cleared before each session
- No financial obligations - trades will not result in fund pay-in or pay-out
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet facility unavailable from 21:00 to 00:00

## Regulatory Changes

No regulatory changes introduced in this circular.

## Compliance Requirements

- Members must participate in the mock session using all trading software
- Members must re-login into live environment after mock session
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- Members using NNF software must manually clear orders in their systems
- Ensure successful telnet connectivity before participation
- Participation enabled using existing User IDs, IPs, and Box ID mappings from live environment

## Important Dates

- **October 11, 2025**: Mock trading session
  - Session-1 (Primary Site): 11:00 - 13:30 hours
  - Session-2 (BCP site): 14:00 - 14:30 hours
  - Position Limit/Collateral setup cutoff: 14:40 hours
  - Trade modification end time: 14:40 hours
  - Live Re-login: 16:00 - 16:30 hours
- **October 13, 2025**: Regular trading resumes (Monday)
- **November 01, 2025**: Discontinuation date for NEAT version 1.5.1 (mock environment)

## Impact Assessment

**Market Impact**: None. This is a routine mock trading session with no impact on live trading or market operations.

**Operational Impact**: Low. Members need to allocate resources for participating in the mock session and ensure connectivity testing. The session helps ensure system readiness and smooth operations for the following trading week.

**Technical Impact**: Members should note the upcoming discontinuation of NEAT version 1.5.1 and migrate to version 1.5.2 (recommended version) before November 01, 2025. The mock session provides an opportunity to test connectivity and systems without financial risk.
