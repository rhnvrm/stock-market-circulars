## Summary

NSE is conducting a mock trading session for the pre-open session in the Futures & Options segment on Saturday, December 06, 2025, with live implementation scheduled for Monday, December 08, 2025. This circular updates members on file download requirements and timelines for participating in the pre-open session, following the earlier announcement in circular NSE/FAOP/71092 dated November 03, 2025.

## Key Points

- Mock trading session scheduled for December 06, 2025 (Saturday)
- Live pre-open session implementation on December 08, 2025 (Monday)
- Members must download specific contract files by 07:00 hrs on December 06, 2025
- Contract files will have Trading status (Normal mkt) field value of '6' for pre-open eligible contracts
- Files available via Extranet path /faoftp/faocommon/ and NSE website
- Regular contract files will be provided per normal timeline with different field values
- New files will replace regular files on the same path once disseminated

## Regulatory Changes

No regulatory changes introduced. This is an operational update for the implementation of pre-open session in F&O segment previously announced.

## Compliance Requirements

**For Members Participating in Pre-Open Session:**

- Download contract.gz file from Extranet path /faoftp/faocommon/ or NSE website (https://www.nseindia.com/all-reports-derivatives)
- Download MII Contract File (NSE_FO_contract_ddmmyyyy.csv.gz) from same path
- Complete downloads by 07:00 hrs on December 06, 2025
- Identify contracts eligible for pre-open session by checking Trading status (Normal mkt) field value = '6'
- Note that regular contract files will have different field values (not '6')

## Important Dates

- **December 06, 2025**: Mock trading session for pre-open in F&O segment
- **December 06, 2025 (by 07:00 hrs)**: Deadline for downloading contract files
- **December 08, 2025**: Live implementation of pre-open session in F&O segment

## Impact Assessment

**Market Impact:** Medium - Affects all F&O segment members who wish to participate in the new pre-open session feature.

**Operational Impact:** Members need to ensure technical readiness by downloading updated contract files and testing systems during the mock session. The pre-open session will add a new trading phase to the F&O segment, requiring system updates and operational adjustments.

**Timeline:** Tight implementation schedule with mock session on Saturday and live rollout on Monday, requiring weekend preparation by participating members.
