## Summary

NSE will conduct a mock trading session in the Futures & Options segment on Saturday, November 08, 2025. The session includes trading from the primary site (11:15-13:30 hrs), DR site (14:30-15:10 hrs), and live re-login practice (16:00-16:30 hrs). No new version release is being implemented. NEAT+ version 7.8.4 will be discontinued on December 06, 2025 (mock), while version 7.8.5 remains recommended.

## Key Points

- Mock trading session scheduled for Saturday, November 08, 2025
- Three sessions: Primary site trading (11:15-13:30), DR site trading (14:30-15:10), and live re-login (16:00-16:30)
- NEAT+ versions 7.8.4 and 7.8.5 will be available for testing
- Version 7.8.4 discontinuation date: December 06, 2025 (mock)
- Version 7.8.5 is the recommended version
- All outstanding orders will be cleared before each session
- Members using NNF software must manually clear orders in their systems
- No financial obligations - trades will not result in fund pay-in or pay-out
- NOTIS application will not be accessible during mock session
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed

## Regulatory Changes

No regulatory changes introduced. This circular provides operational instructions for a routine mock trading session.

## Compliance Requirements

- Members are required to participate using all trading software
- Members must re-login into live environment to ensure smooth connectivity for Monday, November 10, 2025
- Members must ensure telnet connectivity to Exchange host from their IP addresses
- Box ID must be created on the IP with appropriate setup
- UCC/PAN compliance: Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed (except during contingency time)
- Software testing requirements: Members may use either Mock Trading or Simulated Environment to meet regulatory requirements per SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234

## Important Dates

- **November 07, 2025**: Circular issue date
- **November 08, 2025**: Mock trading session date
  - 11:15-13:30 hrs: Trading Session-1 from Primary Site
  - 14:30-15:00 hrs: Trading Session-2 from DR Site
  - 15:10 hrs: Trade modification end time
  - 16:00-16:30 hrs: Live Re-login session
- **November 10, 2025**: Next live trading day (Monday)
- **December 06, 2025**: NEAT+ version 7.8.4 discontinuation date (mock)

## Impact Assessment

**Market Impact**: None. This is a mock trading session with no financial obligations or actual trades.

**Operational Impact**: Low. Members need to allocate resources for participation in the mock session to test systems and connectivity. The session provides an opportunity to practice disaster recovery procedures and ensure readiness for live trading.

**Member Impact**: Members should participate to verify their trading software, connectivity, and disaster recovery capabilities. This is particularly important for ensuring smooth operations when live trading resumes on Monday, November 10, 2025.

**Technical Impact**: Members using NEAT+ version 7.8.4 should plan migration to version 7.8.5 before the December 06, 2025 discontinuation date.
