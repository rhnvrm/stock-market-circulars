## Summary

NSE will conduct a mock trading session for the Securities Lending and Borrowing Market (SLBM) on Saturday, August 23, 2025. The session will run from 11:15 to 12:00 hours with re-login timings from 14:00 to 14:30 hours. No new software version will be released, continuing with NEAT 1.3.3.

## Key Points

- Mock trading session scheduled for Saturday, August 23, 2025
- Trading session: 11:15 AM to 12:00 PM
- Re-login window: 2:00 PM to 2:30 PM
- No new version release - continuing with NEAT 1.3.3
- All outstanding orders will be cleared before the session
- NOTIS application will not be accessible during mock session
- Connect2NSE and Extranet facility will remain available
- No financial obligations - no actual fund pay-in or pay-out

## Regulatory Changes

No regulatory changes announced. This is a routine mock trading session for system testing purposes.

## Compliance Requirements

- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed
- Members using NNF software must manually clear orders in their systems
- NNF users require dealer type User ID converted for NNF and mapped with IP
- Set appropriate branch and user limits from Corporate Manager Terminal before placing orders
- Members should participate using all trading software and re-login into live environment

## Important Dates

- **August 23, 2025**: Mock trading session date
- **11:15 - 12:00**: Trading session timing
- **14:00 - 14:30**: Re-login timing window
- **August 25, 2025**: Regular trading resumes (Monday)

## Impact Assessment

Minimal market impact as this is a routine system testing exercise. Members should ensure participation to verify connectivity and system readiness for Monday trading. The session helps maintain system reliability and allows members to test their trading infrastructure without financial risk.
