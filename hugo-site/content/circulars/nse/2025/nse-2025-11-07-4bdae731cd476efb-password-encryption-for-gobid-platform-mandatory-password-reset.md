## Summary

NSE is implementing enhanced security measures for the GOBID platform to reinforce security and improve cyber resilience. All users must perform a mandatory one-time password reset after November 14, 2025. Users will be prompted to reset their password upon successful login validation, and access will only be enabled after successful password reset.

## Key Points

- Enhanced security measures being implemented on GOBID platform
- One-time mandatory password reset required for all users
- Password reset must be completed on or after end of day November 14, 2025
- Users will be prompted to reset password after validating existing credentials
- Access to application will be blocked until password is successfully reset
- Market participants must take note to avoid login-related issues

## Regulatory Changes

No regulatory framework changes. This is an operational security enhancement measure.

## Compliance Requirements

- All GOBID platform users must mandatorily reset their passwords on or after November 14, 2025
- Existing credentials will be validated first upon login
- New password must be set following the system prompt
- Users cannot access the application without completing the password reset

## Important Dates

- **November 14, 2025**: End of day - deadline after which mandatory password reset becomes effective
- Users logging in after this date will be prompted for password reset

## Impact Assessment

**Operational Impact**: All GOBID platform users will experience a one-time disruption requiring password reset. Users who fail to reset passwords will be unable to access the platform.

**Security Impact**: Positive enhancement to cyber security and system resilience for the primary market GOBID platform.

**User Impact**: Market participants must ensure all users are aware of the requirement to avoid login issues and potential operational disruptions in IPO bidding activities.
