## Summary

NSE Clearing Limited is migrating collateral management services from the existing Collateral Interface for Members (CIM) to the new NMASS platform. The CIM platform will be discontinued, and all functionalities will be made available exclusively on NMASS from December 15, 2025. Members can create or modify roles starting December 1, 2025.

## Key Points

- CIM platform will be discontinued effective December 15, 2025
- All collateral-related functionalities migrating to NMASS platform
- No change in processes, only platform interface changes
- Super Admin users need to create or modify roles for new services
- Role creation/modification available from December 1, 2025
- User manual provided in annexure for NMASS functionalities
- Applies to all clearing members, custodians, and PCMs

## Regulatory Changes

No regulatory changes. This is an operational platform migration to consolidate member interfaces into a single platform (NMASS).

## Compliance Requirements

**For Clearing Members/Custodians:**
- Super Admin users must create or modify roles in NMASS before December 15, 2025
- Assign appropriate rights for migrated services to relevant users
- Complete role configuration to avoid operational disruptions
- Review user manual (Annexure) for NMASS functionalities

**Migrated Functionalities:**

1. **Collateral Release:**
   - New Request/Inquiry
   - Data Export
   - File Upload
   - Download Return Files
   - Instruction wise File upload

2. **EMI:**
   - Cash Deposit
   - G-sec Deposit

3. **Reports:**
   - Quantity Breach Report
   - Direct Pledge Securities Report
   - Data Export

## Important Dates

- **November 28, 2025:** Circular issued
- **December 1, 2025:** Role creation/modification available in NMASS for Super Admin users
- **December 15, 2025:** CIM discontinued; services exclusively available on NMASS

## Impact Assessment

**Operational Impact:**
- Medium impact requiring proactive action by clearing members
- Members must complete role configuration within two-week window (Dec 1-15)
- Failure to configure roles may cause operational challenges in accessing collateral management services
- No process changes minimize learning curve, only interface migration

**Affected Parties:**
- All clearing members in debt segment
- Custodians
- Professional Clearing Members (PCM)
- Super Admin users responsible for role management

**Benefits:**
- Single unified member interface (NMASS) for all services
- Streamlined collateral management operations
- Consolidated platform reduces complexity
