## Summary

NSE has announced a change in ISIN code for Dev Information Technology Limited (DEVIT) equity shares, effective August 21, 2025. The new ISIN INE060X01034 will replace the existing code due to subdivision of shares from Rs. 5/- to Rs. 2/-.

## Key Points

- Company: Dev Information Technology Limited (Symbol: DEVIT)
- New ISIN: INE060X01034
- Reason: Sub-division of equity shares from Rs. 5/- to Rs. 2/-
- Effective date: August 21, 2025 (Ex-date)
- Applies to all trades from the effective date

## Regulatory Changes

ISIN code update from existing code to INE060X01034 for DEVIT equity shares following share subdivision.

## Compliance Requirements

- All NSE members must update their systems with the new ISIN code
- New ISIN applies to all trades executed on and from August 21, 2025
- Trading systems and records must reflect the change from the ex-date

## Important Dates

- **August 21, 2025**: Ex-date for new ISIN effectiveness
- All trades from this date will use the new ISIN code

## Impact Assessment

Medium impact on trading operations requiring system updates for the new ISIN code. Routine administrative change following corporate action of share subdivision, affecting record-keeping and settlement processes.
