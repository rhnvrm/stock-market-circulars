## Summary

NSE will conduct a mock trading session in the Currency Derivatives Segment on Saturday, November 08, 2025. This session is a follow-up to circular NSE/CD/70422 regarding new encryption mechanisms for interactive messages and immediate order acknowledgement. Members are required to participate using all trading software and re-login to the live environment to ensure smooth connectivity for Monday, November 10, 2025.

## Key Points

- Mock trading session scheduled for Saturday, November 08, 2025
- Trading session: 11:00 to 13:30 hrs, Trade modification end: 13:40 hrs
- Live re-login window: 16:00 to 16:30 hrs
- NEAT Version 3.5.4 to be used (available at /cdsftp/cdscommon/NEATCDS354)
- Two NEAT Adapter versions supported: Version 1.0.20 (Old, Port 10877) and Version 1.0.22 (New, Port 10879)
- No financial obligations from trades executed during mock session
- NOTIS application will not be accessible during mock session
- All outstanding orders will be purged before each session

## Regulatory Changes

No regulatory changes introduced. This is a technical mock session related to previously announced encryption mechanism updates (circular NSE/CD/70422 dated September 25, 2025).

## Compliance Requirements

- Members must participate in mock trading using all their trading software
- Members must re-login to live environment during the specified window (16:00-16:30 hrs)
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed for trading
- Members using NNF software must manually clear orders in their systems
- Software testing requirements as per circular NSE/MSD/46441 and SEBI circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234
- Members should use appropriate NEAT Adapter version combination with corresponding ports

## Important Dates

- **November 07, 2025**: Circular issue date
- **November 08, 2025**: Mock trading session day
  - Trading: 11:00 - 13:30 hrs
  - Trade modification end: 13:40 hrs
  - Live re-login: 16:00 - 16:30 hrs
- **November 10, 2025**: Regular live trading resumes (Monday)

## Impact Assessment

**Market Impact**: None. This is a mock trading session with no financial obligations or settlement requirements.

**Operational Impact**: Low. Members need to allocate resources for mock session participation and testing. The session allows members to test their connectivity, trading software, and adapter applications before regular trading resumes on Monday. Members must ensure successful re-login to live environment to avoid connectivity issues on November 10, 2025.

**Technical Requirements**: Members can choose between old NEAT Adapter 1.0.20 (Port 10877) or new NEAT Adapter 1.0.22 (Port 10879) for connectivity. UCC validation will be enforced except during contingency periods.
