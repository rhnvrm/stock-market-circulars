## Summary

NSE will conduct a mock trading session in the Commodity Derivatives Segment on Saturday, November 08, 2025. The session will use existing NEAT version 1.5.3 with no new version release. Members are required to participate using all trading software and re-login into the live environment to ensure smooth connectivity for trading on Monday, November 10, 2025.

## Key Points

- Mock trading session scheduled for Saturday, November 08, 2025
- Trading session timing: 11:00 to 13:30 hours
- Position Limit/Collateral value setup cut off time: 13:40 hours
- Trade modification end time: 13:40 hours
- Live re-login window: 16:00 to 16:30 hours
- NEAT version 1.5.3 will be used (download path: /comtftp/comtcommon/NEATCO153)
- All outstanding orders will be cleared before each session
- NOTIS application will not be accessible during the mock session

## Regulatory Changes

No regulatory changes. This is a routine mock trading session.

## Compliance Requirements

- Members must participate using all trading software
- Members must re-login into live environment after mock session
- UCC/PAN validity requirements: Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed (per circulars NSE/ISC/51754 dated March 24, 2022 and NSE/ISC/52722 dated June 23, 2022)
- Members using NNF software must manually clear orders in their systems
- For NNF Application Users: User ID must be of dealer type, converted for NNF, and mapped with IP
- Members should set appropriate branch and user limits from the control menu

## Important Dates

- **November 07, 2025**: Circular issued
- **November 08, 2025**: Mock trading session
  - Trading: 11:00 - 13:30 hours
  - Position Limit/Collateral setup cutoff: 13:40 hours
  - Live re-login window: 16:00 - 16:30 hours
- **November 10, 2025**: Regular live trading resumes (Monday)

## Impact Assessment

**Market Impact**: None. This is a mock trading session with no financial obligations. Trades during mock sessions will not result in any fund pay-in or pay-out.

**Operational Impact**: Low. Members need to participate to ensure smooth connectivity for Monday trading. The session serves as a system readiness check for all trading members in the Commodity Derivatives Segment.

**Technical Requirements**: Members must ensure telnet connectivity, Box ID setup, and proper user ID configuration. All members enabled for trading in the Commodity Derivatives segment in the live market are automatically eligible for mock trading using existing User IDs, IPs, and Box ID mappings.
