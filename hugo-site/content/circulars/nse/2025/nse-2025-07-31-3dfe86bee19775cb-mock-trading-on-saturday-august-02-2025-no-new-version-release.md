## Summary

NSE will conduct a mock trading session for the Currency Derivatives segment on Saturday, August 02, 2025. The session includes testing from both primary and disaster recovery sites, with new pre-trade risk control features (Pre-emptive order cancellation on account of NNF ID-Algo ID validations) available for testing.

## Key Points

- Mock trading scheduled for Saturday, August 02, 2025
- No new software version release - using NEAT 3.5.3
- Testing includes both primary site and disaster recovery site sessions
- New pre-trade risk controls will be available for testing
- All outstanding orders will be cleared before each session
- No financial obligations from mock trades

## Trading Schedule

**Primary Site Session:**
- Normal Market Open: 11:00
- Contingency Start: 12:00
- Contingency Close: 13:30
- Normal Market Close: 14:00

**DR Site Session:**
- Trading Session: 14:45 - 15:30
- Trade Modification End: 15:40
- Live Re-login: 17:30 - 18:00

## New Features for Testing

- Pre-trade risk controls with pre-emptive order cancellation
- NNF ID-Algo ID validations (effective from August 04, 2025 in live environment)

## Compliance Requirements

- Members must participate using all trading software
- Re-login into live environment required for smooth Monday operations
- Only valid and compliant UCC/PAN will be allowed
- UCC validation skipped during contingency time
- Members using NNF software must manually clear orders

## Important Notes

- NOTIS application will not be accessible during mock session
- All CD segment members automatically eligible
- Support available at Toll-Free: 1800 266 0050 (Option 1)
- Live updates available at www.nseindia.com
