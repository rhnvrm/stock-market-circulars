## Summary

NSE will conduct a mock trading session in the Securities Lending and Borrowing Market (SLBM) segment on Saturday, November 08, 2025. This is a routine testing session with no new version release. The session will run from 11:15 to 13:30 hours for trading, followed by live re-login from 16:00 to 16:30 hours. All members enabled for SLBM trading are automatically eligible to participate.

## Key Points

- Mock trading date: November 08, 2025
- Trading session: 11:15 to 13:30 hours from primary site
- Live re-login window: 16:00 to 16:30 hours
- NEAT version: 1.3.3 (no new version release)
- No financial obligations arising from mock trades
- All outstanding orders will be cleared before each session
- NOTIS application will not be accessible during mock session
- Members should participate using all trading software and re-login to ensure smooth connectivity for Monday, November 10, 2025

## Regulatory Changes

No regulatory changes in this circular.

## Compliance Requirements

- Members using NNF software must manually clear orders in their systems
- Only valid and compliant UCC/PAN uploaded and approved before cutoff will be allowed (UCC validation will be skipped during contingency time)
- User IDs must be dealer type, converted for NNF, and mapped with IP for Non-NEAT Frontend users
- Set appropriate branch and user limits from Corporate Manager Terminal before placing orders
- Members should participate to meet SEBI testing requirements per circular SEBI/HO/MRD1/DSAP/CIR/P/2020/234

## Important Dates

- **November 08, 2025**: Mock trading session
  - 11:15 - Normal market open
  - 13:30 - Normal market close
  - 16:00 - Live re-login start time
  - 16:30 - Live re-login close time
- **November 10, 2025**: Regular live trading resumes on Monday

## Impact Assessment

Minimal impact as this is a routine mock trading session with no system changes. The session allows members to test their trading systems and connectivity in a controlled environment without financial obligations. Members are encouraged to participate to ensure smooth operations when live trading resumes on Monday. The session provides an opportunity to verify telnet connectivity, box ID setup, and user limits before the start of the trading week.
