- Recovery Certificate No. 4415 of 2021 released
- Party: Sanjay Jethalal Soni (HUF)
- Related entity: Timbor Home Limited
- Order type: Release Order for compliance

No regulatory framework changes. This is an individual enforcement action outcome.

This is a release order indicating compliance has been achieved or proceedings concluded. No new compliance requirements are imposed.
